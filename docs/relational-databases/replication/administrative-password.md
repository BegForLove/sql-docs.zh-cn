# 管理密码
  如果在 **分销商** 页为此发布服务器的此向导中，选择远程分发服务器，您必须输入的密码在连接复制进行发布服务器和分发服务器上使用之间 **distributor_admin** 登录名。 该密码必须与在配置分发向导的 **“分发服务器密码”** 页上或在 **“分发服务器属性”** 对话框的 **“发布服务器”** 页上指定的密码相匹配。  
  
## 选项  
 **密码**  
 为发布服务器和远程分发服务器之间的连接输入密码。  
  
 **确认密码**  
 重新输入密码以确认密码输入正确。
