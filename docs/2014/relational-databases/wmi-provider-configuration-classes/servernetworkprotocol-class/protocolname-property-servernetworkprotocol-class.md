# <a name="protocolname-property-servernetworkprotocol-class"></a>ProtocolName 属性（ServerNetworkProtocol 类）
  获取服务器网络协议的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个[ServerNetworkProtocol 类](servernetworkprotocol-class.md)对象，表示使用的实例的网络协议[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定服务器网络协议名称的字符串。
