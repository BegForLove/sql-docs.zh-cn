# <a name="instance-selection-upgrade"></a>实例选择(升级)
  使用“实例选择(升级)”页可以指定要升级的 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 实例。  
  
## <a name="options"></a>“常规”  
 必须同时升级实例中的所有组件。 不能在升级过程中添加或删除功能。
