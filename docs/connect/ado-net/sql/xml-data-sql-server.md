# <a name="xml-data-in-sql-server"></a>SQL Server 中的 XML 数据

[!INCLUDE[Driver_ADONET_Download](../../../includes/driver_adonet_download.md)]

SQL Server 展示 SQLXML 在 .NET 内的功能。 开发人员可以编写应用程序，以便从 SQL Server 实例访问 XML 数据、将数据置于 .NET 环境中、处理数据以及将更新发送回 SQL Server。 在 SQL Server 中，可以通过多种方式使用 XML 数据，包括数据存储，以及作为检索数据的参数值。 .NET 中的 SqlXml 类为在 SQL Server 中使用存储在 XML 列中的数据提供客户端支持  。 有关详细信息，请参阅 SQL Server 联机丛书中的“SQLXML 托管类”。
