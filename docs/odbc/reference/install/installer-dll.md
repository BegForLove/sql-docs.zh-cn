# <a name="installer-dll"></a>安装程序 DLL
> [!NOTE]  
>  从 Windows XP 和 Windows 服务器 2003 开始，ODBC 包含在 Windows 操作系统中。 您只应在早期版本的 Windows 上显式安装 ODBC。  
  
 安装程序 DLL 包含安装和删除 ODBC 组件、维护有关这些组件的注册表信息以及维护有关数据源的注册表信息的功能。 它由微软编写，可以重新分发。 有关安装程序 DLL 中函数的完整说明，请参阅安装程序[DLL API 参考](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)。
