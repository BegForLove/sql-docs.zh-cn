# <a name="syspdwhealthcomponentgroups-transact-sql"></a>sys.pdw_health_component_groups (Transact-SQL)

  存储信息的组件和设备的逻辑分组。  
  
|列名|数据类型|Description|范围|  
|-----------------|---------------|-----------------|-----------|  
|group_id|**int**|组件和设备的唯一标识符。<br /><br /> 此视图的键。|NOT NULL|
