# <a name="propertynumval-property-servernetworkprotocolproperty-class"></a>PropertyNumVal 属性（ServerNetworkProtocolProperty 类）
  获取所引用属性的数值。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.PropertyNumVal [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示实例[!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]上的网络协议属性的[ServerNetworkProtocolProperty 类](servernetworkprotocolproperty-class.md)对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定所引用属性的数值的 `uint32` 值。
