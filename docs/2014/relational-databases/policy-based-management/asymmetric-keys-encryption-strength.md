# <a name="asymmetric-keys-encryption-strength"></a>非对称密钥加密强度
  此规则检查非对称密钥是否是使用 1024 位或更强的加密创建的。  
  
## <a name="best-practices-recommendations"></a>最佳做法建议  
 使用 RSA 1024 位或更强的加密为数据加密创建非对称密钥。
