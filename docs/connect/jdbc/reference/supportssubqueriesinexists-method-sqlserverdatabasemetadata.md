# <a name="supportssubqueriesinexists-method-sqlserverdatabasemetadata"></a>supportsSubqueriesInExists 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库是否支持 EXISTS 表达式中的子查询。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean supportsSubqueriesInExists()  
```  
  
## <a name="return-value"></a>返回值  
 如果支持，则值为 true  。 否则为 **false**。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>备注  
 此 supportsSubqueriesInExists 方法是由 java.sql.DatabaseMetaData 接口中的 supportsSubqueriesInExists 方法指定的。
