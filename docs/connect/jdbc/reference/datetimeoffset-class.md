# <a name="datetimeoffset-class"></a>DateTimeOffset 类
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  此类型表示 microsoft.sql.datetimeoffset 类型的对象，并且是与相对应的 Java 类型[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] **datetimeoffset**数据类型。  
  
 **包：** microsoft.sql  
  
 **扩展：** java.lang.Object  
  
 **实现：** java.lang.Comparable\<DateTimeOffset>、java.io.Serializable  
  
## <a name="syntax"></a>语法  
  
```  
  
public final class DateTimeOffset  
```
