# <a name="getlogwriter-method-sqlserverdatasource"></a>getLogWriter 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  此方法仅供内部使用。 有关记录的详细信息，请参阅[跟踪驱动程序操作](../../../connect/jdbc/tracing-driver-operation.md)。  
  
## <a name="syntax"></a>语法  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>返回值  
 PrintWriter 对象。  
  
## <a name="remarks"></a>备注  
 此 getLogWriter 方法是由 javax.sql.DataSource 接口中的 getLogWriter 方法指定的。
