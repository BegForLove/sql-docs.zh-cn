# <a name="row-level-security"></a>行级安全性
  行级别安全性使客户可以基于执行查询的用户的特性（例如，组成员身份或执行上下文）来控制对数据库表进行的访问。 行级别安全性现已推出[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]2016年。 有关此功能的当前描述，请参阅当前文档中的 [行级别安全性](https://msdn.microsoft.com/library/dn765131.aspx) 。
