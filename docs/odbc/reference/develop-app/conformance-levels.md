# <a name="conformance-levels"></a>一致性级别
ODBC 驱动程序使应用程序能够访问不同的数据源。 每个驱动程序都允许应用程序在运行时确定哪些 ODBC 功能以及驱动程序和每个数据源支持的 SQL 语法。 这不是设计用于单个驱动程序或已知小型驱动程序集的应用程序的要求，因为这些应用程序可以简单地写入该驱动程序或驱动程序的功能。 为了帮助应用程序发现驱动程序和数据源功能，有两个一致性领域可用：ODBC 接口和 SQL 语法。

-   [SQL 一致性级别](../../../odbc/reference/develop-app/sql-conformance-levels.md)
