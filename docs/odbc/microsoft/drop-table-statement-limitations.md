# <a name="drop-table-statement-limitations"></a>DROP TABLE 语句限制
使用 Microsoft Excel 5.0、7.0 或 97 驱动程序时，DROP TABLE 语句将清除工作表，但不会删除工作表名称。 由于工作表名称仍然存在于工作簿中，因此无法使用相同的名称创建另一个工作表。
