# <a name="go-to-dialog-box-ssas"></a>“转到”对话框 (SSAS)
  通过 **“转到”** 对话框，您可以跳至表中的特定列。  
  
## <a name="uielement-list"></a>UIElement 列表  
  
|||  
|-|-|  
|术语|定义|  
|**转到(G)**|从 **“转到”** 列表中选择某列，然后单击 **“确定”**。|
