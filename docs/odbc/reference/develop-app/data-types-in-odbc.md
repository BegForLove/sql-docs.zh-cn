# <a name="data-types-in-odbc"></a>ODBC 中的数据类型
ODBC 使用数据类型的两个的集：SQL 数据类型以及 C 数据类型。 在数据源中，使用 SQL 数据类型和应用程序中的 C 代码中使用 C 数据类型。
