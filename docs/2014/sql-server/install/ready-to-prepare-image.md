# <a name="ready-to-prepare-image"></a>已可以准备映像
  **“已可以准备映像”** 页提供 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安装程序准备的功能的摘要。  
  
## <a name="options"></a>“常规”  
 此页上有下列选项：  
  
-   若要继续安装，请单击 **“安装”**。  
  
-   若要更改安装设置，请单击 **“上一步”**。  
  
-   若要退出安装程序，请单击 **“取消”**。
