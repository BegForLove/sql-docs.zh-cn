# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>创建和删除视图和过程使用 ExtendedAnsiSQL
启用 ExtendedAnsiSQL 标志后，Jet 4.0 支持创建和删除视图和通过 SQL 过程的功能。 这些视图和过程不会通过访问可见。
