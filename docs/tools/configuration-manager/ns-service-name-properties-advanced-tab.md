# NS$&lt;服务名称&gt; 属性（“高级”选项卡）
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] 以 **NS$**<user_configured name> 格式实现成服务。 如果定义了自定义属性，定义的属性及其属性值将显示在此选项卡上。 此选项卡有可能不列出任何属性。
