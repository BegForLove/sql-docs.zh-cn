# <a name="product-support"></a>产品支持
对 ODBC 的产品支持由 Microsoft 产品支持服务提供。 由于许多 Microsoft 产品使用 ODBC 作为核心组件，并且还重新分配了 ODBC 驱动程序，因此 Microsoft 产品支持服务认为 ODBC 是每个产品中集成 ODBC 的一部分。 根据该产品的支持协议，提供对 ODBC 的支持。 有关详细信息，请参阅各个产品文档。
