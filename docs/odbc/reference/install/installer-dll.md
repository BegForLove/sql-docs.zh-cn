# <a name="installer-dll"></a>安装程序 DLL
> [!NOTE]  
>  从 Windows XP 和 Windows Server 2003 开始，ODBC 包括在 Windows 操作系统中。 在早期版本的 Windows 上，应仅显式安装 ODBC。  
  
 安装程序 DLL 包含用于安装和删除 ODBC 组件、 维护有关这些组件的注册表信息和维护有关数据源的注册表信息函数。 它由 Microsoft 编写，并可以重新分发。 安装程序 DLL 中函数的完整说明，请参阅[安装程序 DLL API 参考](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)。
