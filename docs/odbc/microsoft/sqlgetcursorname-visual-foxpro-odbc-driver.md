 支持： 完整  
  
 ODBC API 一致性： 核心级别  
  
 返回的名称与关联的光标给定*hstmt*。 **SQLGetCursorName** Visual FoxPro ODBC 驱动程序 API 中包括，因为它是核心级别 API 功能的一部分; 它不能与使用其他 API 函数因为驱动程序不支持定位的更新。
