# <a name="analysis-services-query-execution-options-general-page"></a>Analysis Services 查询执行选项（“常规”页）
  使用此页可指定用于显示 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 查询结果的选项。  
  
 **执行超时值**  
 指示在取消查询之前等待的秒数。 值 0 指示无限期的等待或无超时。  
  
 重置为默认值  
 将此页上的所有值重置为原始默认值。
