# <a name="group-by-expression-list"></a>GROUP BY 表达式列表
GROUP BY 支持表达式列表和列名称。 具有 GROUP BY 子句的 SELECT 语句的选择列表只能包括来自 GROUP BY 子句或集函数的表达式。
