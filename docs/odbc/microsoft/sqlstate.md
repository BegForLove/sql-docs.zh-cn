# <a name="sqlstate"></a>SQLSTATE
对于数据源中发生的错误，ODBC 驱动程序将返回的本机错误映射到相应的 SQLSTATE。 对于驱动程序或驱动程序管理器检测到的错误，ODBC 驱动程序或驱动程序管理器将生成相应的 SQLSTATE。
