# <a name="localdberrorinstancestopfailed"></a>LOCALDB_ERROR_INSTANCE_STOP_FAILED
    
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|271|  
|事件源|SQL Server 本地数据库运行时 12.0|  
|组件|本地数据库运行时 API|  
|消息正文|无法在指定的时间内完成本地数据库实例的停止操作。|  
  
## <a name="explanation"></a>解释  
 停止操作在给定的时间内未能完成。  
  
## <a name="user-action"></a>用户操作  
 再次重试请求的操作。
