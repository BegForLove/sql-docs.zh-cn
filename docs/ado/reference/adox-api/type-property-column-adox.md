# <a name="type-property-column-adox"></a>Type 属性（列）(ADOX)
指示列的数据类型。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回可以是[DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md)常量之一的**Long**值。 默认值为**adVarWChar**。  
  
## <a name="remarks"></a>备注  
 此属性是可读/写的，直到将[列](../../../ado/reference/adox-api/column-object-adox.md)对象追加到集合或其他对象（该对象是只读的）。
