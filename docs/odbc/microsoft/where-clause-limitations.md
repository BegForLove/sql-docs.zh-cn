# <a name="where-clause-limitations"></a>WHERE 子句限制
WHERE 子句中的子句的最大数目为 40。  
  
 LONGVARBINARY 和 LONGVARCHAR 列可以与文本的长度，最多 255 个字符，但不能使用参数进行比较。
