# <a name="failover-cluster-report"></a>故障转移群集报告
  使用“故障转移群集报告”页可以查看安装系统针对此操作所扫描得到的报告结果。  
  
## <a name="options"></a>选项  
 若要查看扫描的详细报告，请单击报告链接。
