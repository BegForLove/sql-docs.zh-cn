# <a name="select-measure-group-tables-cube-wizard"></a>选择度量值组表（多维数据集向导）
  可以使用 **“选择度量值组表”** 页指定数据源视图以及为多维数据集选择度量值组表。  
  
## <a name="options"></a>“常规”  
 **数据源视图**  
 选择要用于多维数据集的数据源视图。  
  
 **度量值组表**  
 选择将用于多维数据集中的度量值组的表。
