# <a name="deploying-an-exception-message-box-application"></a>部署异常消息框应用程序
  异常消息框安装通过[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] ，支持使用在你自定义的 Windows 应用程序，以改进异常处理。 因为异常消息框安装的所有次级版本[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]除[!INCLUDE[ssEW](../../includes/ssew-md.md)]，可将其用于任何计算机上任何其他配置[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]客户端组件，包括在 SDK 中，已安装。
