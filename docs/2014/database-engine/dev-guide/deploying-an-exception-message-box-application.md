# <a name="deploying-an-exception-message-box-application"></a>部署异常消息框应用程序
  异常消息框由[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]安装，并支持在您的自定义 Windows 应用程序中使用以改进异常处理。 由于异常消息框由[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]除之外[!INCLUDE[ssEW](../../includes/ssew-md.md)]的所有版本安装，因此你可以在安装了[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]客户端组件（包括 SDK）的任何计算机上使用，而无需进行其他配置。
