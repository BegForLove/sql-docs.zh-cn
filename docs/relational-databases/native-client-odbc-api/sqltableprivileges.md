[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  **SQLTablePrivileges**可以对静态游标执行。 尝试执行**SQLTablePrivileges**上可更新 （由键集驱动或动态） 返回 sql_success_with_info 以指示游标类型已更改。
