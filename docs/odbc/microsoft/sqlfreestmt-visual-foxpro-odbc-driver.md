# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：核心级别  
  
 停止与特定*hstmt*关联的处理，关闭与*hstmt*关联的任何打开的游标，放弃挂起的结果，并选择性地释放与该语句句柄关联的所有资源。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) 。
