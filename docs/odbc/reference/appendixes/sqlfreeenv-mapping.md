# <a name="sqlfreeenv-mapping"></a>SQLFreeEnv 映射
当应用程序*通过 ODBC 1.x*驱动程序调用**SQLFreeEnv**时，调用  
  
```  
SQLFreeEnv(henv)   
```  
  
 映射到  
  
```  
SQLFreeHandle(SQL_HANDLE_ENV,Handle)  
```  
  
 如果将*Handle*参数设置为*henv*中的值，则为。
