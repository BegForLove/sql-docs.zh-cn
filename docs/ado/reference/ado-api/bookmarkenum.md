# <a name="bookmarkenum"></a>BookmarkEnum
指定，该值指示操作应开始的位置的书签。  
  
|常量|值|Description|  
|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|从当前记录开始。|  
|**adBookmarkFirst**|1|在第一条记录的启动。|  
|**adBookmarkLast**|2|起价最后一条记录。|
