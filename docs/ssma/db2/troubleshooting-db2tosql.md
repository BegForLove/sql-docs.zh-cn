# <a name="troubleshooting-db2tosql"></a>故障排除 (DB2ToSQL)
并不少见运行迁移问题中途执行控制台脚本或命令。  
  
可能与可修复的错误：  
  
1.  函数  
  
2.  过程  
  
3.  “查看”  
  
4.  触发器  
  
5.  修改数据类型  
  
6.  更改项目设置  
  
可以通过打开通过控制台创建的项目文件中使用的 SSMA 用户界面修复了问题。
