# <a name="drop-table-statement-limitations"></a>DROP TABLE 语句限制
使用 Microsoft Excel 5.0、 7.0、 或 97 驱动程序时，DROP TABLE 语句清除工作表，但不会删除工作表名称。 由于工作簿中仍存在的工作表名称，因此不能具有相同名称创建另一个工作表。
