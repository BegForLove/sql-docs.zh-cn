# <a name="configurationsetting-property---isinitialized"></a>ConfigurationSetting 属性 - IsInitialized
  指示报表服务器实例是否已初始化。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim IsInitialized As Boolean  
```  
  
```csharp  
public Boolean DatabaseLogonName;  
```  
  
## <a name="property-values"></a>属性值  
 **Boolean** 对象指示是否初始化报表服务器实例。
