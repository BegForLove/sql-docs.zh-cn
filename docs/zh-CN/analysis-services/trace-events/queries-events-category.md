# <a name="queries-events-category"></a>查询事件类别
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  “查询事件”事件类别具有下表中说明的事件类。  
  
|Event Class|事件 ID|Description|  
|-----------------|--------------|-----------------|  
|查询开始|9|收集从开始跟踪之后发生的所有查询开始事件。|  
|查询结束|10|收集从开始跟踪之后发生的所有查询结束事件。|  
  
 有关与每个“查询事件”事件类关联的列的信息，请参阅 [Queries Events Data Columns](../../analysis-services/trace-events/queries-events-data-columns.md)。
