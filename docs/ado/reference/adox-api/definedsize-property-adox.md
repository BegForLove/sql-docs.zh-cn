# <a name="definedsize-property-adox"></a>DefinedSize 属性 (ADOX)
指示列的规定的最大大小。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回**长**以字符为单位的数据值的最大长度的值。  
  
## <a name="remarks"></a>备注  
 默认值为零 (0)。  
  
 此属性是只读的[列](../../../ado/reference/adox-api/column-object-adox.md)已追加到集合的对象。
