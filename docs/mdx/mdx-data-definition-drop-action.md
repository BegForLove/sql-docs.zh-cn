# <a name="mdx-data-definition---drop-action"></a>MDX 数据定义 - DROP ACTION


  从指定多维数据集中删除指定的操作。  
  
## <a name="syntax"></a>语法  
  
```  
  
DROP ACTION CURRENTCUBE | Cube_Name  
   .Action_Name   
```  
  
## <a name="arguments"></a>参数  
 *Cube_Name*  
 提供多维数据集名称的有效字符串表达式。  
  
 *Action_Name*  
 提供要删除操作的名称的有效字符串表达式。
