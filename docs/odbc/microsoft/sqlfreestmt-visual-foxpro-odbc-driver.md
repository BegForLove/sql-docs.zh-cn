# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt （Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含 Visual FoxPro ODBC 驱动程序相关的信息。 有关此函数的常规信息，请参阅下的相应主题[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支持： 完整  
  
 ODBC API 一致性： 核心级别  
  
 停止与特定关联的处理*hstmt*，关闭任何打开的游标与关联*hstmt*、 放弃挂起的结果，并根据需要释放与该语句关联的所有资源句柄。  
  
 有关详细信息，请参阅[SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md)中*ODBC 程序员参考*。
