# <a name="configuring-data-sources"></a>配置数据源
有关数据源信息存储在系统注册表中。 用户修改通过管理计划的数据源信息。 这可以是 ODBC 管理器、 ODBC 控制面板设备或由应用程序或驱动程序的开发人员编写的管理程序。  
  
 可以使用 PowerShell 命令修改数据源。 有关这些 PowerShell 命令的详细信息，请参阅[Windows 数据访问组件 PowerShell 命令](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)。  
  
> [!NOTE]  
>  本节是特定于 Microsoft Windows® 平台。 数据源在其他平台上的配置方式是特定于平台的。
