# <a name="keyerrorlimit-element-assl"></a>KeyErrorLimit 元素 (ASSL)
  包含处理期间可接受的错误数。  
  
## <a name="syntax"></a>语法  
  
```xml  
  
<ErrorConfiguration>  
   ...  
      <KeyErrorLimit>...</KeyErrorLimit>  
   ...  
</ErrorConfiguration>  
```

|基数|0-1：可出现一次且仅出现一次的可选元素。|
