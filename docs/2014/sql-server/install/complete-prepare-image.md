# <a name="complete---prepare-image"></a>完成 - 准备映像
  使用**完成**页后，可以验证映像准备是否已完成。  
  
## <a name="options"></a>选项  
 此对话框包含指向此操作的安装日志文件的链接。 若要完成映像准备，请单击**关闭**。
