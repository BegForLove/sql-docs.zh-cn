# <a name="visual-foxpro-odbc-driver"></a>Visual FoxPro ODBC 驱动程序
微软Visual FoxPro是数据库构建和应用程序开发的强大面向对象的环境。 Microsoft Visual FoxPro ODBC 驱动程序使应用程序能够通过开放数据库连接 （ODBC） 界面打开、查询和更新 Visual FoxPro 和早期版本的 FoxPro 中的数据。  
  
 例如，使用 Microsoft 可视化 FoxPro ODBC 驱动程序，您可以执行以下操作：  
  
-   使用 Microsoft 查询从 Microsoft Excel 工作表查询和更新 Visual FoxPro 数据。  
  
-   使用 Microsoft Word 使用 Visual FoxPro 数据创建邮件合并字母。  
  
-   从 Microsoft Access 查询和更新 Visual FoxPro 视图和表。  
  
-   使用 Visual FoxPro 作为 Microsoft 可视化基础、微软视觉C++和 C 应用程序的数据存储。
