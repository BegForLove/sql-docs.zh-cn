# <a name="definedsize-property-adox"></a>DefinedSize 属性 (ADOX)
指示列的指定最大大小。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回一个**Long**值，该值为数据值的最大长度（以字符为单位）。  
  
## <a name="remarks"></a>备注  
 默认值为零 (0)。  
  
 对于已追加到集合的[列](../../../ado/reference/adox-api/column-object-adox.md)对象，此属性是只读的。
