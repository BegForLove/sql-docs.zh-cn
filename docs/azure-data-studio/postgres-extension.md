# <a name="postgresql-extension-preview"></a>ibanPostgreSQL 扩展（预览版）

PostgreSQL 扩展 （预览版），可连接到、 查询和开发适用于 Postgres 使用 Azure Data Studio 的功能。 

可用于 PostgreSQL 的 Azure Data Studio 功能包括：

- 连接管理器和查询编辑器
- 可自定义仪表板和见解小组件
- 代码片段
- [集成的终端](integrated-terminal.md)
- [键盘快捷键](keyboard-shortcuts.md)
- [源代码管理集成](source-control.md)
- [工作区和用户设置](settings.md)


## <a name="install-the-postgresql-extension-preview"></a>安装 PostgreSQL 扩展 （预览版）

如果还没有安装的 Azure Data Studio，请参阅其[安装说明](download.md)。

1. 从 Azure Data Studio 在侧栏中选择扩展图标。
   ![扩展图标](media/extensions/postgresql-extension/extensions-icon.png)

2. 在搜索栏中键入 postgresql。 选择的 PostgreSQL 扩展。

3. 选择“安装”  。 安装后，选择**重新加载**激活 Azure Data Studio 中的扩展。
