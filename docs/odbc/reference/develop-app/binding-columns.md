# <a name="binding-columns"></a>绑定列
从数据源提取数据返回到应用程序已为此目的分配的变量中的应用程序。 这可以完成之前，必须将关联应用程序，或*绑定*，这些变量和结果的列设置; 从概念上讲，此过程是相同应用程序变量绑定到语句参数。 当应用程序将绑定到变量的结果集列，它描述了该变量-地址、 数据类型等 — 向驱动程序。 该驱动程序结构，它维护适用于该语句，并使用的信息提取该行时返回值的列中存储此信息。
