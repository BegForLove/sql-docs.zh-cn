# <a name="deleterule-property-adox"></a>DeleteRule 属性 (ADOX)
指示删除主键时，将执行的操作。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回**长**值可以是之一[RuleEnum](../../../ado/reference/adox-api/ruleenum.md)常量。 默认值是**adRINone**。  
  
## <a name="remarks"></a>Remarks
