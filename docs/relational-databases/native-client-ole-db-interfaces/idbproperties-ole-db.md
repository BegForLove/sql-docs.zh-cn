# <a name="idbproperties-ole-db"></a>IDBProperties (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  OLE DB 标准规范允许提供程序为 DBPROPINFO::vValues 指定 VT_EMPTY****。 但是，[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]当您调用**IDB属性：：：获取属性信息**以**DBPROPSET_ROWSETALL**检索行集属性时，本机客户端 OLE DB 始终返回VT_EMPTY。
