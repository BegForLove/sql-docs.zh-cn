# <a name="sqlnumparams"></a>SQLNumParams
  数据库引擎中的改进，以[!INCLUDE[ssSQL11](../../includes/sssql11-md.md)] "允许 SQLNumParams" 为预期结果提供更准确的说明。 更准确的结果可能与早期版本的[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]SQLNumParams 返回的值不同。  
  
 有关详细信息，请参阅[元数据发现](../native-client/features/metadata-discovery.md)。
