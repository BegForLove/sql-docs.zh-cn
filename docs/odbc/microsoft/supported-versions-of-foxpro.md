# <a name="supported-versions-of-foxpro"></a>受支持的版本的 FoxPro
你可以使用 Microsoft® Visual FoxPro® ODBC 驱动程序，来访问 FoxPro 表中存储数据。 支持以下版本的 FoxPro 数据：  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   Visual FoxPro （所有版本）  
  
 Visual FoxPro 中存储的数据访问时，你可以选择连接到[数据库](../../odbc/microsoft/visual-foxpro-terminology.md)，其中包含零个或多[表](../../odbc/microsoft/visual-foxpro-terminology.md)或目录的[释放表](../../odbc/microsoft/visual-foxpro-terminology.md)。  
  
 有关如何连接到数据源的详细信息，请参阅[添加 Visual FoxPro 数据源](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md)。  
  
 若要下载驱动程序，请参阅[Visual FoxPro ODBC 驱动程序下载站点](http://go.microsoft.com/fwlink/?LinkId=121318)。
