# <a name="standard-data-stream-protocol"></a>标准数据流协议
标准数据流协议是访问异构 DBMS 中数据的一种方式。 事实上，标准数据流协议已经存在：

- ANSI/ISO 远程数据库访问 （RDA） 标准[：ISO/IEC 9579：2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615)。 尽管 ANSI/ISO 系统具有前景，但如今并未得到广泛实施。
