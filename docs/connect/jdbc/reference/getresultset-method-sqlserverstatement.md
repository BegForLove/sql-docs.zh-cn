# <a name="getresultset-method-sqlserverstatement"></a>getResultSet 方法 (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索作为 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 对象的当前结果。  
  
## <a name="syntax"></a>语法  
  
```  
  
public final java.sql.ResultSet getResultSet()  
```  
  
## <a name="return-value"></a>返回值  
 ResultSet 对象。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>备注  
 此 getResultSet 方法是由 java.sql.Statement 接口中的 getResultSet 方法指定的。
