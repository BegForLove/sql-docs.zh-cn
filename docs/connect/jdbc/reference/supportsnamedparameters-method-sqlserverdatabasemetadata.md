# <a name="supportsnamedparameters-method-sqlserverdatabasemetadata"></a>supportsNamedParameters 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库是否支持可调用语句中的命名参数。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean supportsNamedParameters()  
```  
  
## <a name="return-value"></a>返回值  
 **true**如果支持。 否则为 **false**。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.DatabaseMetaData 接口中的 supportsNamedParameters 方法指定此 supportsNamedParameters 方法。
