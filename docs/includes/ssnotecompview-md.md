  将此 SQL Server 2000 系统表作为一个视图包含进来是为了保持向后兼容性。 建议您改用当前的 SQL Server 系统视图。 若要查找一个或多个等效系统视图，请参阅[将系统表映射到系统视图 (Transact-SQL)](../relational-databases/system-tables/mapping-system-tables-to-system-views-transact-sql.md)。 后续版本的 Microsoft SQL Server 将删除该功能。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。
