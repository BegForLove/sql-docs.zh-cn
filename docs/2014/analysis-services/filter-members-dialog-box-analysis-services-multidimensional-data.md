# <a name="filter-members-dialog-box-analysis-services---multidimensional-data"></a>“筛选成员”对话框（Analysis Services - 多维数据）
  可以使用 **中的** “筛选成员” [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] 对话框，在 **“维度设计器”** 的 **“浏览器”** 选项卡中浏览维度的同时，按当前级别的成员标题、成员名称、成员唯一名称、键列值或值列值对维度成员进行筛选。  
  
## <a name="options"></a>选项  
  
|术语|定义|  
|----------|----------------|  
|**筛选器表达式**|显示一个网格，其中包含用于构造筛选表达式的属性、运算符和值。 请注意，添加一行之后，无法删除该行。 必须关闭并重新打开该对话框，才可指定新的筛选表达式。 该网格包含以下列：<br /><br /> **属性**：选择要用于筛选表达式的成员的属性。<br /><br /> **运算符**：选择要用于筛选表达式的运算符。<br /><br /> **值**：键入在**属性**中选择的属性的值，以使用**运算符**中指定的运算符进行计算。|  
|**测试窗格**|单击 **“测试”** 时，此窗格将显示筛选表达式所返回的成员。 如果使用在 **“筛选表达式”** 中指定的条件没有返回任何成员，则会显示警告。|  
|**考试**|单击此项可以测试在 **“筛选表达式”** 中指定的条件。|
