# 在跟踪暂停或停止之后运行跟踪 (SQL Server Profiler)
  本主题介绍如何使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]在跟踪暂停或停止之后重新启动跟踪。  
  
### 在跟踪暂停或停止之后运行跟踪  
  
1.  选择包含停止或暂停的跟踪的窗口。  
  
2.  在 **“文件”** 菜单上，单击 **“运行跟踪”**。
