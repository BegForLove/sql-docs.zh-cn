# <a name="close-method-sqlserverpooledconnection"></a>close 方法 (SQLServerPooledConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  关闭物理连接此[SQLServerPooledConnection](../../../connect/jdbc/reference/sqlserverpooledconnection-class.md)对象所表示。  
  
## <a name="syntax"></a>语法  
  
```  
  
public void close()  
```  
  
## <a name="exceptions"></a>异常  
 java.sql.SQLException  
  
## <a name="remarks"></a>注释  
 由 javax.sql.PooledConnection 接口中关闭的方法指定此关闭的方法。
