|||  
|-|-|  
|[轴](../../../ado/reference/ado-md-api/axes-collection-ado-md.md)|包含**轴**定义是单元集的对象。|  
|[CubeDefs](../../../ado/reference/ado-md-api/cubedef-object-ado-md.md)|包含**CubeDef**表示多维目录中的多维数据集对象。|
