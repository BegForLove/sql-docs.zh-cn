# <a name="alter-table-statement"></a>ALTER TABLE 语句
使用 Microsoft Access、dBASE 或 Paradox 驱动程序时，使用添加和删除表约束定义子句支持 ALTER TABLE 语句。 （Microsoft Excel 或文本驱动程序不支持 ALTER TABLE 语句。  
  
> [!NOTE]  
>  当您在不实现 Borland 数据库引擎的情况下使用 Paradox 驱动程序时，不支持 ALTER TABLE 语句;因此，如果不执行"不带悖论"，则不支持 ALTER TABLE 语句。仅允许读取和追加语句。
