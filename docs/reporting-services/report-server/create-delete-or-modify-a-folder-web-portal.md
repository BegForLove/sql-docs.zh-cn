# <a name="create-delete-or-modify-a-folder-web-portal"></a>创建、删除或修改文件夹（Web 门户）
  可以创建文件夹来组织和管理发布到报表服务器的项。 创建文件夹有助于用户查找他们关注的报表。 对于内容管理员来说，文件夹提供了应用权限的框架。 可以对特定的文件夹创建角色分配，来限制对处于开发阶段或者不应大范围分布的报表的访问。  
  
#### <a name="to-create-a-folder"></a>创建文件夹  
  
1.  启动[报表管理器 &#40;SSRS 本机模式 &#41;](http://msdn.microsoft.com/library/80949f9d-58f5-48e3-9342-9e9bf4e57896).  
  
2.  在 web 门户中，选择**文件夹**下**新建**菜单下拉列表。 或者，若要创建现有文件夹下的文件夹，请导航到该文件夹，然后选择**文件夹**下**新建**菜单下拉列表。  
  
     **中创建一个新文件夹**对话框随即打开。  
  
3.  键入文件夹名称。 文件夹名称可以包含空格，但不能包含以下用于 URL 编码的保留字符：; ? : @ & = + , $ / * < > |。 : @ & = + , $ / * < > |。 不能通过键入一系列文件夹名称来同时创建多个文件夹。  
  
4.  选择“创建”。  
  
#### <a name="to-modify-or-delete-a-folder"></a>修改或删除文件夹  
  
1.  在 web 门户中，导航到**主页**页，然后找到你想要修改的项。  
  
2.  选择**...**为所需的文件夹。  
  
3.  在上下文菜单中，选择**管理**。 “属性”页将打开。  
  
4.  通过以下方式修改文件夹属性：  
  
    -   若要修改关于该文件夹的显示文本，请键入名称或说明。  
  
    -   若要显示的默认视图中的文件夹上**主页**页上，清除**在磁贴视图中隐藏**。  
  
5.  或者，若要删除该文件夹及其内容，请单击 **“删除”**。  
  
6.  单击 **“应用”** 以保存更改。
