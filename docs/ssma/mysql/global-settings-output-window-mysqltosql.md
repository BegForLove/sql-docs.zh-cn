# <a name="global-settings-output-window-mysqltosql"></a>全局设置 （输出窗口） (MySQLToSQL)
使用**全局设置**对话框中指定的 SSMA 设置。  
  
若要访问输出窗口设置中，在**工具**菜单中，选择**全局设置**，单击**GUI**在左窗格中，并选择底部**输出窗口**。  
  
## <a name="options"></a>选项  
**显示错误列表窗格**  
选择 **，则返回 True**如果你想 SSMA 消息时显示错误列表窗格。  
  
**显示输出窗格**  
选择 **，则返回 True**如果你想 SSMA 消息时显示输出窗格。
