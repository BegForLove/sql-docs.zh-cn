# <a name="delete-group-dialog-box"></a>“删除组”对话框
  使用 **“删除组”** 对话框可以删除表、矩阵或列表数据区域中的组。 如果该组具有关联的行或列，则还可以删除这些结构。  
  
## <a name="options"></a>选项  
 **删除组以及相关的行和列**  
 选择该选项可以删除组定义和与该组关联的所有行和列。  
  
 **仅删除组**  
 选择此选项可以仅删除组定义。
