# <a name="attributes-property-adox"></a>Attributes 属性 (ADOX)
描述列特征。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回**长**值。 值指定的表的由表示特征[列](../../../ado/reference/adox-api/column-object-adox.md)对象。 值可以为组成[ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md)常量。 默认值为零 (**0**)，其中既不是**adColFixed**也不**adColNullable**。  
  
## <a name="applies-to"></a>适用范围  
  
- [列对象 (ADOX)](../../../ado/reference/adox-api/column-object-adox.md)
