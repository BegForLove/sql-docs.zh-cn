# <a name="odbc-component-files"></a>ODBC 组件文件
ODBC 驱动程序文件是 ODBC 桌面数据库驱动程序 4.0 的每个组件所必需的。  
  
 某些文件需要多个组件。 如果要将任何 ODBC 桌面数据库驱动程序与商业应用程序重新分发，则必须重新分发这些文件。  
  
 这些文件安装在 Windows 95 或更高版本的 [Windows_System] 目录中，或安装在 Windows NT 3.51 或更高版本和 Windows 2000 的 [Windows_System32 目录中。 如果 ODBC 文件以前安装在其他目录中，请确保使用 _Windows_System（或 System32） 目录中的较新文件。
