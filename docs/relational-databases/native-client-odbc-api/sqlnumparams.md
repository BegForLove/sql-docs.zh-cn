# <a name="sqlnumparams"></a>SQLNumParams
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  从 [!INCLUDE[ssSQL11](../../includes/sssql11-md.md)] 开始的数据库引擎改进允许 SQLNumParams 获取对预期结果的更准确的说明。 这些更准确的结果可能与 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]以前版本的 SQLNumParams 返回的值不同。  
  
 有关详细信息，请参阅[元数据发现](../../relational-databases/native-client/features/metadata-discovery.md)。
