# <a name="select-partitions-to-modify-usage-based-optimization-wizard"></a>选择要修改的分区（基于使用情况的优化向导）
  可以使用 **“选择要修改的分区”** 页选择要设计聚合的分区。  
  
## <a name="options"></a>“常规”  
 **（选择）**  
 选中列标题中的复选框，可以为度量值组中的所有分区设计聚合。  
  
 选中某个分区的复选框，可以仅为所选分区设计聚合。  
  
 **分区名称**  
 显示分区的名称。  
  
 **数据源**  
 显示用于为分区提供源数据的表或查询。  
  
 **聚合设计**  
 显示用于描述分区所使用的聚合设计的字符串。
