# <a name="instance-selection-uninstall"></a>实例选择（卸载）
  在“实例选择 – 卸载”页上，使用下拉框指定要卸载的 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 实例。  
  
## <a name="options"></a>“常规”  
 可以指定卸载 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]的整个实例，也可以指定仅卸载共享功能和管理工具。
