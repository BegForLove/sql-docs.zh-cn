# <a name="type-property-ado-md"></a>Type 属性 (ADO MD)
指示当前[成员](../../../ado/reference/ado-md-api/member-object-ado-md.md)的类型。  
  
## <a name="return-values"></a>返回值  
 返回[MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md)值并且是只读的。  
  
## <a name="remarks"></a>备注  
 此属性仅在属于某一[级别](../../../ado/reference/ado-md-api/level-object-ado-md.md)对象的[成员](../../../ado/reference/ado-md-api/member-object-ado-md.md)对象上受支持。 从属于某个[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)对象的**成员**对象引用此属性时，将发生错误。  
  
## <a name="applies-to"></a>应用于  
 [成员对象 (ADO MD)](../../../ado/reference/ado-md-api/member-object-ado-md.md)
