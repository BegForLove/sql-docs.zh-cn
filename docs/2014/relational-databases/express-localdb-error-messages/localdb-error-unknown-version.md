# <a name="localdberrorunknownversion"></a>LOCALDB_ERROR_UNKNOWN_VERSION
    
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|269|  
|事件源|SQL Server 本地数据库运行时 12.0|  
|组件|本地数据库运行时 API|  
|消息正文|指定的本地数据库版本在此计算机上不可用。|  
  
## <a name="explanation"></a>解释  
 指定的版本不可用。  
  
## <a name="user-action"></a>用户操作  
 安装请求的本地数据库运行时版本。
