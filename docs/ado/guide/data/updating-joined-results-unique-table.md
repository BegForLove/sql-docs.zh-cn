ADO 使您能够密切控制修改为使用唯一表动态属性的多个基础表的联接操作的正确的记录集在特定的基表。 有关使用唯一表的详细信息，请参阅上的唯一表和更新再同步的动态属性的 ADO 程序员参考主题。
