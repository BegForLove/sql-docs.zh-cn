# R Server (Standalone)
  **Microsoft R 服务器 （独立）** 是由 Microsoft R 服务提供企业级的分析平台的一部分。  
  
 它通过提供支持、 可扩展性和安全扩展 R。 [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] 通过添加并行和成块的处理的数据，从而使用户能够分析在企业规模较大的数据，并使用比在内存中可容纳大得多的数据还解决了开放源代码的 R 的内存中限制。  
  
  
## Microsoft R Server 入门  
  
 有关如何设置以支持 R 应用程序的 Windows 服务器的详细信息，请参阅 [创建独立 R 服务器](../../advanced-analytics/r-services/create-a-standalone-r-server.md)。  

您可以针对嵌入在应用程序内的 R 分析和与后端系统集成 R DeployR，以及设置 Azure 虚拟机，包括 R 服务器 （独立）。 有关详细信息，请参阅：

+ [设置 R Server 虚拟机](../../advanced-analytics/r-services/provision-the-r-server-only-sql-server-2016-enterprise-vm-on-azure.md)
+ [部署 R](https://msdn.microsoft.com/microsoft-r/deployr-about)
  
有关示例、 教程和有关 Microsoft R 的详细信息，请参阅 [Microsoft R 产品](https://msdn.microsoft.com/microsoft-r/microsoft-r-getting-started)。
