# <a name="specify-nested-join-dialog-box-mining-accuracy-chart-view"></a>“指定嵌套联接”对话框（“挖掘准确性图表”视图）
  可以使用 **“指定嵌套联接”** 对话框在嵌套表和事例表之间创建关系。 若要打开 **“指定嵌套联接”** 对话框，请选择嵌套表并单击 **“修改联接”**。  
  
## <a name="options"></a>“常规”  
 **添加关系**  
 单击此项可打开“创建关系”对话框。  
  
 **删除关系**  
 单击此项可删除“关系图”窗格中所选的关系。
