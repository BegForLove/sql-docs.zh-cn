# <a name="formattedvalue-property-ado-md"></a>FormattedValue 属性 (ADO MD)
指示格式化的显示的[单元格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)值。  
  
## <a name="return-values"></a>返回值  
 返回**字符串**和是只读的。  
  
## <a name="remarks"></a>备注  
 使用**FormattedValue**属性获取的格式化的显示的值[值](../../../ado/reference/ado-md-api/value-property-ado-md.md)属性[单元格](../../../ado/reference/ado-md-api/cell-object-ado-md.md)对象。 例如，如果单元格的值为 1056.87，并且此值表示一个美元金额， **FormattedValue**将是 $1,056.87。
