# <a name="gettransactiontimeout-method-sqlserverxaresource"></a>getTransactionTimeout 方法 (SQLServerXAResource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  获取为此参数设置的当前事务超时值[SQLServerXAResource](../../../connect/jdbc/reference/sqlserverxaresource-class.md)对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
public int getTransactionTimeout()  
```  
  
## <a name="exceptions"></a>异常  
 javax.transaction.xa.XAException  
  
## <a name="remarks"></a>注释  
 由 javax.transaction.xa.XAResource 接口中的 getTransactionTimeout 方法指定此 getTransactionTimeout 方法。
