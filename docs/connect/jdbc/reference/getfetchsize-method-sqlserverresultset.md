# <a name="getfetchsize-method-sqlserverresultset"></a>getFetchSize 方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此提取大小[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
public int getFetchSize()  
```  
  
## <a name="return-value"></a>返回值  
 **Int** ，该值指示当前的提取大小。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.ResultSet 接口中的 getFetchSize 方法指定此 getFetchSize 方法。
