# <a name="task-5-adding-data-flow-task"></a>任务 5：添加数据流任务
  在本任务中，您将向 SSIS 包的控制流添加一个数据流任务。  
  
1.  拖放式**数据流任务**从**SSIS 工具箱**到**控制流**在 SSIS 设计器的选项卡。 如果看不到**SSIS 工具箱**中, 单击任一处**控制流**选项卡上，单击**SSIS**菜单栏上，然后单击**SSIS 工具箱**。  
  
     ![拖放数据流任务从 SSIS 工具箱](../../2014/tutorials/media/et-addingdataflowtask.jpg "拖放数据流任务从 SSIS 工具箱")  
  
2.  右键单击**数据流任务**中**控制流**选项卡，单击**重命名**。  
  
3.  类型**接收、 清理、 匹配项，而 Curate 供应商数据**按**ENTER**。  
  
4.  双击**数据流任务**切换到**数据流**选项卡。
