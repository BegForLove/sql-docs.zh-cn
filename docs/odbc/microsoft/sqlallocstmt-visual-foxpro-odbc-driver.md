# <a name="sqlallocstmt-visual-foxpro-odbc-driver"></a>SQLAllocStmt（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：核心级别  
  
 为语句句柄分配内存，并将语句句柄与*hdbc*指定的连接关联。 驱动程序管理器将此调用传递给驱动程序，后者为*hstmt*结构分配内存。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLAllocStmt](../../odbc/reference/syntax/sqlallocstmt-function.md) 。
