# <a name="ruleenum"></a>RuleEnum
指定何时应遵循的规则[密钥](../../../ado/reference/adox-api/key-object-adox.md)被删除。  
  
|常量|ReplTest1|Description|  
|--------------|-----------|-----------------|  
|**adRICascade**|1|级联更改。|  
|**adRINone**|0|默认值。 不执行任何操作。|  
|**adRISetDefault**|3|外键的值设置为默认值。|  
|**adRISetNull**|2|外键的值设置为 null。|
