# MSSQL_REPL-2147201005
    
## 消息详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|-2147201005|  
|事件源|MSSQLServer|  
|符号名称||  
|消息正文|合并进程无法更新已发送到发布服务器的上次发送的生成。 如果此故障仍然存在，请重新初始化订阅。|  
  
## 解释  
 合并操作对订阅服务器调用一个存储过程，以找到它上次发送给发布服务器的最高生成；或对发布服务器调用一个存储过程，以找到它上次发送给订阅服务器的最高生成。 用于设置上次发送的生成的存储过程调用失败。  
  
## 用户操作  
 重新初始化订阅。
