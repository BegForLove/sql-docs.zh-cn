# <a name="sorting-limitations"></a>排序限制
GROUP BY 子句、 ORDER BY 子句、 SELECT DISTINCT 语句或外部联接中的排序键的最大长度为 255 个字节;排序行中的所有排序键的最大长度为 65,500 个字节。
