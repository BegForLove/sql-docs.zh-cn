# InstanceName 属性 (WMI MSReportServer_Instance)
  返回特定计算机上报表服务器实例的名称。 只读。  
  
## 语法  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## 属性值  
 一个表示报表服务器实例名称的 **String** 对象。
