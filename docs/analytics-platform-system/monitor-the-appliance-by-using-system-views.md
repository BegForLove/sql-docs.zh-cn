# <a name="monitor-the-appliance-with-system-views---analytics-platform-system"></a>监视设备，但系统视图的分析平台系统
本文列出了可用于监视 SQL Server PDW 系统视图。  
  
## <a name="to-monitor-the-appliance-by-using-system-views"></a>若要使用系统视图监视设备  
SQL Server PDW 包括使你能够获取有关设备运行状况、 状态和性能的详细的信息的综合系统视图。 此表提供了可用于每个监视功能的系统视图的链接。

|警报|[sys.pdw_health_alerts](../relational-databases/system-catalog-views/sys-pdw-health-alerts-transact-sql.md)<br /><br />[sys.dm_pdw_component_health_alerts](../relational-databases/system-dynamic-management-views/sys-dm-pdw-component-health-alerts-transact-sql.md)<br /><br />[sys.dm_pdw_component_health_active_alerts](../relational-databases/system-dynamic-management-views/sys-dm-pdw-component-health-active-alerts-transact-sql.md)<br /><br />[sys.dm_pdw_component_health_status](../relational-databases/system-dynamic-management-views/sys-dm-pdw-component-health-status-transact-sql.md)|
