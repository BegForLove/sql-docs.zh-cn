# <a name="source-property-ado-md"></a>源属性 (ADO MD)
指示中的数据的源[单元集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回**Variant**，以及是否是读/写的关闭[单元集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)对象和只读打开**单元集**对象。 **Variant**应包含一个有效**字符串**，例如，MDX 查询。
