# <a name="getresultset-method-sqlserverstatement"></a>getResultSet 方法 (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索作为 [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) 对象的当前结果。  
  
## <a name="syntax"></a>语法  
  
```  
  
public final java.sql.ResultSet getResultSet()  
```  
  
## <a name="return-value"></a>返回值  
 结果集对象。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 getResultSet 方法由 getResultSet 方法在 .sql 接口中指定。
