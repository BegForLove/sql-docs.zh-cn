# <a name="ordered-and-unordered-expressions-xquery"></a>已排序和未排序的表达式 (XQuery)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  默认情况下，中[!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]所有操作的排序模式均为**排序**模式。 因此，路径表达式和 FLWOR 表达式返回的节点序列（不包含**order by**子句）按文档顺序排列。  
  
 不支持 XQuery 规范中所述的其他**有序**和**无序**语法。
