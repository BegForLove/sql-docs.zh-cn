# <a name="streamtypeenum"></a>StreamTypeEnum
指定[流](../../../ado/reference/ado-api/stream-object-ado.md)对象中存储的数据的类型。  
  
|一直|值|说明|  
|--------------|-----------|-----------------|  
|**adTypeBinary**|1|指示二进制数据。|  
|**adTypeText**|2|默认值。 指示文本数据，该数据[位于字符集指定](../../../ado/reference/ado-api/charset-property-ado.md)的字符集中。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 等效项  
 这些常量没有 ADO/WFC 等效项。
