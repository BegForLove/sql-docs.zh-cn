# InstallationID 属性 (WMI MSReportServer_ConfigurationSetting)
  此属性返回特定报表服务器实例的唯一标识符。  
  
## 语法  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
