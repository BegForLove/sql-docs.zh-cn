# MSSQL_REPL-2147201021
    
## 消息详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|-2147201021|  
|事件源|MSSQLServer|  
|符号名称||  
|消息正文|发布“%1”处于非活动状态。 若要激活此发布，请启动快照代理以生成初始快照。|  
  
## 解释  
 指定的发布处于非活动状态。  
  
## 用户操作  
 若要激活发布，请运行发布的快照代理。
