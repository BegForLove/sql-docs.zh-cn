# <a name="internet-information-services-error-codes-for-remote-data-service"></a>适用于远程数据服务的 Internet 信息服务错误代码
下表列出了与远程数据服务使用情况相关的 Microsoft® Internet 信息服务 (IIS) 错误代码。 低两个字节的正十进制转换，显示完整的错误代码和十六进制值的负十进制转换。

|Internet Information Services 错误|Number|Description|
|------------------------------------------|------------|-----------------|
|**IDS_IIS_AccessDenied**|8208-2146820080 0x800A2010|Internet 服务器错误： 访问被拒绝。|
|**IDS_IIS_ObjectNotFound**|8209-2146820079 0x800A2011|Internet 服务器错误： 对象/模块找不到。|
|**IDS_IIS_RequestForbidden**|8210-2146820078 0x800A2012|Internet 服务器错误： 请求被禁止。|
|**IDS_IIS_UnexpectedError**|8447-2146819841 0x800A20FF|Internet 服务器错误。|
