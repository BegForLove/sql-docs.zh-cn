# <a name="clientnetworkprotocolproperty-class"></a>ClientNetworkProtocolProperty 类
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)][ClientNetworkProtocolProperty 类](../../../relational-databases/wmi-provider-configuration-classes/clientnetworkprotocolproperty-class/clientnetworkprotocolproperty-class.md)表示的客户端网络协议的属性。  
  
 与[ClientNetworkProtocolProperty 类](../../../relational-databases/wmi-provider-configuration-classes/clientnetworkprotocolproperty-class/clientnetworkprotocolproperty-class.md)，你可以执行以下任务：  
  
-   设置一个用于从属性列表选择客户端网络协议属性的索引值。  
  
-   检查当前属性所属的协议。  
  
-   检查当前属性的名称、类型和值。  
  
-   修改当前属性的属性字符串值和数值。  
  
-   设置当前属性的标志。
