# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>使用 ExtendedAnsiSQL 创建和删除视图和过程
启用扩展 AnsiSQL 标志后，Jet 4.0 支持通过 SQL 创建和删除视图和过程的能力。 这些视图和过程将无法通过 Access 可见。
