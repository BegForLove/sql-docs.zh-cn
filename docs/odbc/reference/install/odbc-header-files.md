# <a name="odbc-header-files"></a>ODBC 头文件
> [!NOTE]  
>  不需要再分发 ODBC 标头文件。 无视为 "核心"。  
  
 **标头文件**  
  
 Odbcinst.ini  
  
 Sql。h  
  
 Sqltypes。h  
  
 Sqltypes  
  
 Sqlucode。h  
  
 Msdasql  
  
 Msdadc  
  
 Sqlspi （用于驱动程序开发，请参阅[ODBC 服务提供程序接口（SPI）参考](../../../odbc/reference/syntax/odbc-service-provider-interface-spi-reference.md)，了解详细信息），从 WINDOWS 8 SDK 开始。
