# <a name="find-databases-wizard-select-locations-accesstosql"></a>查找数据库向导 （选择位置） (AccessToSQL)
在查找数据库向导的选择位置页上，查找访问数据库的输入的搜索参数。  
  
若要打开查找数据库向导中，在**文件**菜单上，单击**查找数据库**。 有关如何使用此对话框的详细信息，请参阅[添加和删除访问数据库文件](adding-and-removing-access-database-files-accesstosql.md)  
  
## <a name="options"></a>选项  
**“浏览”**  
单击此项可浏览的计算机或网络。 选择文件夹或位置搜索，然后单击**确定**。 单击**添加**若要将此位置添加到列表。  
  
**路径框**  
显示要添加到位置中的路径。 输入或浏览到你想要扫描，然后单击的路径**添加**按钮。  
  
**位置框**  
列出了 SSMA 将扫描的位置。 当您单击**添加**按钮，指定的路径添加到此框。  
  
**“添加”**  
将在路径文本框中的位置添加到的位置的列表。  
  
**替换**  
如果选择列表中的位置的项，则在路径文本框中的项将替换所选的项。  
  
**删除**  
要搜索的位置列表中移除选定的项。  
  
**全部或部分文件名**  
若要查找具有包含特定字符串名称的数据库，请输入的字符串。 不要使用通配符字符。  
  
**创建日期**  
若要查找特定日期范围内创建的数据库，请输入中的开始日期**从**中框和结束日期**到**框。  
  
**上次更新的日期**  
若要查找数据库的上次更新特定日期范围内，输入中的开始日期**从**中框和结束日期**到**框。  
  
**大小**  
若要查找与大小条件匹配的数据库，在第一个框中选择一个比较字符，输入一个数字的大小，然后从第三个框中选择字节、 千字节为单位或兆字节为单位。  
  
**“所有者”**  
输入的 Windows 用户全名的所有者。 这可能会将转为*域*\\*用户*。
