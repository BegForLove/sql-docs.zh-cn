# <a name="date-arithmetic"></a>日期算术
驱动程序支持从日期、时间或时间STAMP列中添加和减去整数。 对于 DATE 列，整数指定要添加或减去的天数。 对于 TIME 或 TIMESTAMP 列，整数指定要添加或减去的秒数。
