# <a name="nodejs-driver-for-sql-server"></a>适用于 SQL Server 的 Node.js 驱动程序

![向下键-带圆圈的](../../ssdt/media/download.png)[下载 NODE.JS SQL 驱动程序](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

繁琐的模块是 TDS 协议的 javascript 实现，由所有新式版本的 SQL Server 提供支持。 该驱动程序是一个开源项目, 在 Github 上提供。  
  
可以在 Windows、Linux 或 Mac 上使用 node.js 连接到 SQL 数据库。
