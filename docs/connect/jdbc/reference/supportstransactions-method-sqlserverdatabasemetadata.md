# <a name="supportstransactions-method-sqlserverdatabasemetadata"></a>supportsTransactions 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库是否支持事务。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean supportsTransactions()  
```  
  
## <a name="return-value"></a>返回值  
 如果支持,**则为 true** 。 否则为 **false**。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 supportsTransactions 方法由 supportsTransactions 方法在 Java.sql.databasemetadata 接口中指定。
