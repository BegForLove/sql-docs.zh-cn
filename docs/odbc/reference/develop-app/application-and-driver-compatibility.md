# <a name="application-and-driver-compatibility"></a>应用程序和驱动程序兼容性
ODBC 应用程序和驱动程序除了其版本外，还分为多个类别。 其中一些应用程序与某些驱动程序不兼容;在其他情况下，应用程序或驱动程序的类型可能与它们之间的向后兼容性问题有关。

-   [驱动程序类型](../../../odbc/reference/develop-app/types-of-drivers.md)
