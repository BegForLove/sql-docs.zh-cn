# <a name="source-property-ado-md"></a>Source 属性（ADO MD）
指示[单元](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)集中的数据的源。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回**变体**，并对已关闭的[单元格](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)对象进行读/写，并对打开的**单元格**对象进行只读。 **变体**应该包含有效的**字符串**，例如 MDX 查询。
