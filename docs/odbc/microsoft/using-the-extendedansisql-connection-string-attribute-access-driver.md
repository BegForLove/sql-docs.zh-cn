# <a name="using-the-extendedansisql-connection-string-attribute-access-driver"></a>使用 ExtendedAnsiSQL 连接字符串属性（Access 驱动程序）
Jet 4.0 具有多个新功能，默认情况下在 ODBC 桌面数据库驱动程序版本 4.0 中未启用。 要利用这些新功能，应用程序必须设置新的连接属性、向连接字符串添加新属性或在 DSN 中设置该属性。 该属性称为扩展 AnsiSQL，仅与 Microsoft ODBC 桌面数据库驱动程序 4.0 及更高版本相关。
