# <a name="numeric-literal-syntax"></a>数值语法
以下语法用于 ODBC 中的数字文本：  
  
 *数字文本*：：**签名-数字文本&#124;无符号-数字-文本*  
  
 *签名数字文本*：：* =*符号*=*无符号数字文本*  
  
 *无符号数字文本*：：**精确-数字-文本&#124;近似-数字-文本*  
  
 *精确数字文本*：：**无符号整数*=*期间*=*无符号整数** *&#124;期间无符号整数*  
  
 *符号*：：**加号&#124;减号*  
  
 *近似-数字-文本*：：=*曼蒂萨 E 指数*  
  
 *曼蒂萨*：：**精确数字-文字*  
  
 *指数*：：**签名整数*  
  
 *符号整数*：：* =*符号*=*无符号整数*  
  
 *无符号整数*：：**数字...*  
  
 *加号*：：**+*  
  
 *减号*：：* -  
  
 *数字*：：* 1 &#124; 2 &#124; 3 &#124; 4 &#124; 5 &#124; 6 &#124; 7 &#124; 8 &#124; 9 &#124; 0  
  
 *期间*：* 。
