# <a name="messages-returned-by-the-odbc-driver-for-oracle"></a>对于 Oracle ODBC 驱动程序返回的消息
> [!IMPORTANT]  
>  将 Windows 的未来版本中删除该功能。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 如果可用的 Oracle 错误消息，它将返回前有一个通过 [Microsoft] [适用于 Oracle ODBC 驱动程序，] 和 [Oracle] 标记;否则，如以下示例所示的 [Oracle] 标记没有返回的消息：  
  
## <a name="oracle-error-message"></a>Oracle 错误消息：  
  
```  
[Microsoft][ODBC driver for Oracle][Oracle]ORA-nnnnn message-text  
```  
  
## <a name="odbc-driver-for-oracle-error-message"></a>Oracle 错误消息的 ODBC 驱动程序：  
  
```  
[Microsoft][ODBC driver for Oracle]  
```
