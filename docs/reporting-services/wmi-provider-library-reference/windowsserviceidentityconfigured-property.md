# WindowsServiceIdentityConfigured 属性 (WMI MSReportServer_ConfigurationSetting)
  返回在上次配置报表服务器 Windows 服务时所指定的运行时要使用的标识。 只读。  
  
## 语法  
  
```vb  
Public Dim WindowsServiceIdentityConfigured As String  
```  
  
```csharp  
public string WindowsServiceIdentityConfigured;  
```  
  
## 属性值  
 为 **String** 值，包含在上次配置报表服务器 Windows 服务时所指定的运行时要使用的标识。
