# <a name="publisher-properties---publisher-general"></a>发布服务器属性 - 发布服务器，常规
  **“发布服务器属性”** 对话框的 **“常规”** 页显示有关发布服务器使用的分发服务器和分发数据库的只读信息。 若要更改发布服务器的分发服务器或分发数据库，请执行以下操作：  
  
1.  禁用发布服务器上的发布。 有关详细信息，请参阅[禁用发布和分发](disable-publishing-and-distribution.md)。  
  
2.  重新配置发布和分发。 有关详细信息，请参阅[配置发布和分发](configure-publishing-and-distribution.md)。
