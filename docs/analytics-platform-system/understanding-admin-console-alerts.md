# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>了解分析平台系统中的管理控制台警报
警报显示在装置**管理控制台**并在**System Center Operations Manager** (SCOM)。 使用此列表中的警报帮助识别哪些警报需要进一步研究。  
  
了解如何连接到**管理控制台**通过使用 Internet Explorer，请参阅[通过使用管理控制台监视设备&#40;分析平台系统&#41;](monitor-the-appliance-by-using-the-admin-console.md)。 璝惠**SCOM**，请参阅[监视使用 System Center Operations Manager 通过设备&#40;分析平台系统&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
有关获取警报信息通过使用 Transact\-SQL，请参阅[监视通过使用系统视图设备&#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-system-views.md)。  
  
## <a name="types-of-alerts"></a>类型的警报  
指示正常状态的警报名称通常不需要调查。 警报名称中包含的单词重大有时需要执行操作。 调查是必需的所有其他类型的警报。
