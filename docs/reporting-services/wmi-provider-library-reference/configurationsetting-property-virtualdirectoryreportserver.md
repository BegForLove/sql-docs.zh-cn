# <a name="configurationsetting-property---virtualdirectoryreportserver"></a>ConfigurationSetting 属性 - VirtualDirectoryReportServer
  返回报表服务器应用程序的虚拟目录。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## <a name="property-values"></a>属性值  
 包含虚拟目录的 **string** 。
