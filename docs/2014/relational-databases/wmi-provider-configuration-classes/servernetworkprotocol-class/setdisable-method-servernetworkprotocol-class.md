# <a name="setdisable-method-servernetworkprotocol-class"></a>SetDisable 方法（ServerNetworkProtocol 类）
  禁用服务器网络协议。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.SetDisable()  
  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 [ServerNetworkProtocol 类] servernetworkprotocol-class.md) 表示的实例所使用的网络协议的对象[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个 uint32 值，如果服务已成功修改，则为 0；如果不支持请求，则为 1；其他任何数字表示出现错误。
