# <a name="getlong-method-sqlservercallablestatement"></a>getLong 方法 (SQLServerCallableStatement)

|----------|-----------------|  
|[getLong (int)](../../../connect/jdbc/reference/getlong-method-int.md)|在给定参数索引的情况下，检索指定参数的值作为 Java 编程语言中的 long  。|
