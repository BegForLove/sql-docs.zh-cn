# <a name="ltaliasgt-properties-alias-tab"></a>&lt;别名&gt;属性 （别名选项卡）
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]别名是可以用于建立连接的备用名称。 别名封装了连接字符串所必需的元素，并使用用户所选择的名称显示这些元素。 使用“\<别名>属性”对话框中的“别名”页查看或指定别名连接字符串的元素。  
  
## <a name="options"></a>选项  
 **别名**  
 用于引用此连接的名称（别名）。  
  
 **管道名称** / **端口号**  
 连接字符串的其他元素。 此框的名称随所选协议的不同而变化。 请参阅下列主题中的示例。  
  
 **协议**  
 连接所用的协议。  
  
 **Server**  
 所连接的 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 实例的名称。
