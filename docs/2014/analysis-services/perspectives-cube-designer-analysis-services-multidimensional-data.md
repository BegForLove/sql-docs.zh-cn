# <a name="perspectives-cube-designer-analysis-services---multidimensional-data"></a>透视（多维数据集设计器）（Analysis Services - 多维数据）
  可以使用多维数据集设计器中的 **“透视”** 选项卡，创建和修改所选多维数据集的透视。  
  
 **“透视”** 选项卡包含以下窗格：  
  
## <a name="panes"></a>窗格  
 **工具栏**  
 使用工具栏可以执行该选项卡上的常见操作。  
  
 **透视详细信息**  
 使用“透视详细信息”窗格可以创建和修改所选多维数据集的透视。
