# <a name="schema-text-file-driver"></a>架构（文本文件驱动程序）
架构包括有关数据源中每个表（文本文件）的信息，包括表的格式、要扫描以确定列类型的行数、表的第一行是否包含列名称、源文件是否使用 OEM 或 ANSI 代码页编写，以及每列的名称、数据类型和宽度。
