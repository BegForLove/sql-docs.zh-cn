# <a name="visual-database-tools-f1-help"></a>Visual Database Tools F1 帮助
  这些主题介绍 Visual Database Tools 元素的对话框和属性窗口（如“表属性”窗口和“列属性”选项卡）中的命令和选项。  
  
 若要访问对话框主题，请在对话框打开并具有焦点时按 F1。 若要访问属性窗口主题，请在“属性”窗口中单击并按 F1。
