> [!NOTE]  
>  本主题提供文本文件特定于驱动程序信息。 有关此函数的常规信息，请参阅下的相应主题[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
|列|注释|  
|------------|--------------|  
|TABLE_QUALIFIER|返回到目录的路径。|  
|TABLE_OWNER|因为不支持所有者名称，将在此列中返回 NULL。|  
|NULLABLE|SQL_NO_NULLS 针对参与的列返回中的主键或唯一索引。|
