# <a name="python-sql-driver"></a>Python SQL 驱动程序

![向下键-带圆圈的](../../ssdt/media/download.png)[用于安装适用于 Python 的 SQL 驱动程序](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

可以在 Windows、Linux 或 Mac 上使用 Python 连接到 SQL 数据库。   
  
## <a name="getting-started"></a>入门  
提供了几个 python SQL 驱动程序。 但是, Microsoft 在**pyodbc**驱动程序中放置了测试工作量及其置信度。 选择驱动程序, 并相应地配置开发环境:
* [Python SQL 驱动程序 - pyodbc](pyodbc/python-sql-driver-pyodbc.md)
* [Python SQL 驱动程序 - pymssql](pymssql/python-sql-driver-pymssql.md)
