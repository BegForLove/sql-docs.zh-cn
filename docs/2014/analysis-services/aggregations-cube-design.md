# <a name="aggregations-cube-design"></a>聚合（多维数据集设计）
  可以使用多维数据集设计器中的 **“聚合”** 选项卡创建和修改聚合设计。  
  
 **“聚合”** 选项卡包含以下窗格：  
  
## <a name="panes"></a>窗格  
 **工具栏**  
 使用工具栏可以执行该选项卡上的常见操作。  
  
 **标准视图**  
 使用“标准视图”窗格可以创建和修改聚合设计。  
  
 **高级的视图**  
 使用“高级视图”窗格可以创建和修改聚合。 此视图应仅允许高级用户使用。
