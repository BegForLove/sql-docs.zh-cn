# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>什么 &#39; s SSMA for DB2 中的新增功能 (DB2ToSQL)
本主题列出每个版本中的 DB2 更改 SSMA。  

## <a name="may-2016"></a>2016 年 5 月  
SSMA for DB2 的 2016 年 5 月版本包含以下更改：  

1.  添加了对 SQL Server 2016 支持
2.  添加的转换为 SQL Server 内存中和 hekaton 功能 DB2 内存中和常规表
3.  添加的转换为 SQL Server 策略对象 （for DB2 的行级别安全性） DB2 访问控制
4.  添加的转换 DB2 版本由系统控制表为 SQL Server 临时表
5.  改进的 DB2 分析器和冲突解决程序
6.  删除安装程序检查 for.Net 2.0
7.  从 Db2 安装程序中删除不必要的 *.dll
8.  "保存项目"固定和 SSMA 控制台的"打开项目"命令
9.  SSMA 控制台的固定"securepassword"命令
10. 固定的初始加载的对象计数
11. 全局设置中的已修复的 bug
  
## <a name="march-2016"></a>2016 年 3 月  
DB2 SSMA 的 2016 年 3 月预览版本包含以下更改：  
  
1.  支持迁移到 SQL Server 2016  
  
## <a name="january-2016"></a>2016 年 1 月  
DB2 SSMA 的 2016 年 1 月维护版本包含以下更改：  
  
1.  添加了对许多标准功能支持  
  
2.  固定的 DB2 分析器错误  
  
3.  固定的 DB2 v9 zOS 支持 (RFC 5690920)  
  
4.  固定的 DB2 未在转换过程中解析标识符错误  
  
5.  添加的视图日志菜单项的 SSMA (文档 RFC 5706203)  
  
6.  添加的遥测  
  
## <a name="november-2014"></a>2014 年 11 月  
2014 年 11 月发行版的 SSMA for DB2 是其类型的第一个。
