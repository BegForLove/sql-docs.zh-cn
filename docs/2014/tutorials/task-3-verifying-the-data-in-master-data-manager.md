# <a name="task-3-verifying-the-data-in-master-data-manager"></a>任务 3：在主数据管理器中验证数据
  此任务中，在你验证**供应商**上创建实体**MDS**使用**Master Data Manager Web Application**。  
  
1.  如果**主数据管理器**已打开，则单击**SQL Server 2012 Master Data Services**顶部导航到主页。 否则，请导航到[ http://localhost/mds ](http://localhost/mds)以启动**主数据管理器**。  
  
2.  选择**供应商**为**模型**，然后单击**资源管理器**。  
  
     ![主数据管理器的资源管理器按钮](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "主数据管理器的资源管理器按钮")  
  
3.  查看存储在 MDS 上的数据。 如果看不到数据，确认是否选择**供应商**为**模型**在主页上，在启动前**资源管理器**。 你可以将添加到或从供应商列表中删除通过使用**添加成员**和**删除成员**工具栏上的按钮。
