# 打开跟踪文件 (SQL Server Profiler)
  本主题说明如何使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]打开跟踪文件。  
  
### 打开跟踪文件  
  
1.  在 **“文件”** 菜单上，单击 **“打开”**，再单击 **“跟踪文件”**。  
  
2.  在 **“打开文件”** 对话框中，选择要打开的跟踪数据文件。
