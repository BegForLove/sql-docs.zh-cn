# <a name="view-join-properties-visual-database-tools"></a>查看联接属性 (Visual Database Tools)
  您可以在“属性”窗口中查看视图的属性集合。  
  
### <a name="to-view-join-properties"></a>查看联接属性  
  
1.  在“关系图”窗格中右键单击相应的联接。  
  
2.  在快捷菜单上单击“属性”。  
  
     此时，将在“属性”窗口中显示该联接的属性。
