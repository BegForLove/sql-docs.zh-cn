# <a name="attribute-properties---set-usage-property"></a>特性属性-设置用法属性
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]你可以通过使用设置属性的用法**维度结构**在维度设计器，从访问视图[!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]。  
  
 当设置属性的用法时，所做的更改直到处理维度时才会生效。 有关详细信息，请参阅[处理多维模型 (Analysis Services)](../../analysis-services/multidimensional-models/processing-a-multidimensional-model-analysis-services.md)。  
  
### <a name="to-set-usage-for-an-attribute"></a>设置属性的用法  
  
1.  在解决方案资源管理器中，右键单击某个维度，再单击“打开”。  
  
     默认情况下，将打开 **“维度结构”** 视图。  
  
2.  在“属性”中，右键单击要为其设置用法的属性，指向“设置属性用法”，再单击下列选项之一：  
  
    -   **Regular**  
  
    -   **Key**  
  
    -   **Parent**
