# <a name="download-the-microsoft-drivers-for-php-for-sql-server"></a>下载 Microsoft Drivers for PHP for SQL Server

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

## <a name="windows"></a>Windows

以下版本的 Windows 上的 PHP 的驱动程序是可供下载：

- [Microsoft Drivers 5.2 for PHP for SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=56729)
- [Microsoft Drivers 4.3 for PHP for SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=55642)
- [4.0、 3.2、 3.1 和 3.0 for PHP for SQL Server 的 Microsoft 驱动程序](https://www.microsoft.com/en-us/download/details.aspx?id=20098)

## <a name="linux-and-macos"></a>Linux 和 macOS

用于 PHP 的驱动程序轻松地下载，并且使用 PECL 在 Linux 和 macOS 上安装。 请参阅[Linux 和 macOS 安装教程](installation-tutorial-linux-mac.md)有关详细信息。 如果你需要手动为 PHP 在 Linux 和 macOS 上安装的驱动程序，则以下版本是可供下载：

- [Microsoft Drivers 5.2 for PHP for SQL Server](https://github.com/Microsoft/msphpsql/releases/tag/v5.2.0)
- [Microsoft Drivers 4.3 for PHP for SQL Server](https://github.com/Microsoft/msphpsql/releases/tag/v4.3.0)
