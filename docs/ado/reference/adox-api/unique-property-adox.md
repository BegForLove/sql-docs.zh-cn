# <a name="unique-property-adox"></a>Unique 属性 (ADOX)
指示是否必须是唯一索引键。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回**布尔**值。  
  
## <a name="remarks"></a>备注  
 默认值为 **False**。  
  
 此属性是只读[索引](../../../ado/reference/adox-api/index-object-adox.md)已追加到集合的对象。
