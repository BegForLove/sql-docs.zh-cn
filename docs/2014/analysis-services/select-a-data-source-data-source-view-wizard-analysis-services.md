# <a name="select-a-data-source-data-source-view-wizard-analysis-services"></a>选择数据源（数据源视图向导）(Analysis Services)
  可以使用 **“选择数据源”** 页为数据源视图选择数据源。  
  
## <a name="options"></a>选项  
 **关系数据源**  
 选择要创建数据源视图的数据源。  
  
 此列表显示了已添加到项目中的数据源。 如果所需数据源不在此列表中，请单击 **“新建数据源”** ，以便向项目中添加新数据源。  
  
> [!NOTE]  
>  尽管可能会列出项目的联机分析处理 (OLAP) 数据源，但只能为关系数据源添加数据源视图。  
  
 **数据源属性**  
 列出当前所选数据源的连接字符串中的访问接口信息。  
  
 **新的数据源**  
 打开数据源向导，在其中可以创建新数据源以便添加到项目中。
