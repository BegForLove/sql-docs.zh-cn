# <a name="primarykey-property-adox"></a>PrimaryKey 属性 (ADOX)
指示索引是否表示表的主键。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回**布尔**值。  
  
## <a name="remarks"></a>Remarks  
 默认值为 **False**。  
  
 此属性为只读上[索引](../../../ado/reference/adox-api/index-object-adox.md)已追加到集合的对象。
