# <a name="task-3-verifying-the-data-in-master-data-manager"></a>任务 3：在主数据管理器中验证数据
  在此任务中，您将验证**供应商**实体是否使用**主数据管理器 Web 应用程序**在**MDS**上创建。

1.  如果**主数据管理器**已打开，请单击顶部的**SQL Server 2012 主数据服务**以导航到主页。 否则，导航到`http://localhost/MDS`启动**主数据管理器**。

2.  为**模型**选择**供应商**，然后单击**资源管理器**。

     ![“主数据管理器 - 资源管理器”按钮](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "“主数据管理器 - 资源管理器”按钮")

3.  查看存储在 MDS 上的数据。 如果看不到数据，请确认在启动**资源管理器**之前，在主页上为**模型**选择了**供应商**。 您可以使用工具栏上的 **"添加成员"** 和 **"删除成员"** 按钮从供应商列表中添加或删除。
