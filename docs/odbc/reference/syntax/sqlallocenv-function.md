**一致性**  
 版本引入了： ODBC 1.0 标准合规性： 不推荐使用  
  
 **摘要**  
 在 ODBC 3。*x*，ODBC 2。*x*函数**SQLAllocEnv**已被取代**SQLAllocHandle**。 有关详细信息，请参阅[SQLAllocHandle 函数](../../../odbc/reference/syntax/sqlallochandle-function.md)。
