# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：级别1  
  
 将缓冲区与 SQL 语句中的参数标记相关联。 Visual FoxPro ODBC 驱动程序支持由*fParamType*参数指定的输入参数。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) 。
