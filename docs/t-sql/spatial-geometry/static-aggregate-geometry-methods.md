# <a name="static-aggregate-geometry-methods"></a>静态聚合几何图形方法
[!INCLUDE[tsql-appliesto-ss2012-asdb-xxxx-xxx_md](../../includes/tsql-appliesto-ss2012-asdb-xxxx-xxx-md.md)]
