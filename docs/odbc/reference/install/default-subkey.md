# <a name="default-subkey"></a>默认子项
注册表可能指定具有默认子键的默认数据源。 此子键是数据源规范子键的特殊情况，具有与任何其他数据源规范子键相同的值。 唯一的区别是，它不被列为 ODBC 数据源子键下的值。
