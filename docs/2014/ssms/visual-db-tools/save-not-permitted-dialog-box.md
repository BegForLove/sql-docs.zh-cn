# <a name="save-not-permitted-dialog-box"></a>“保存”（不允许）对话框
  “保存”  （不允许）对话框警告不允许保存更改，因为所做的更改要求删除并重新创建列出的表。  
  
 以下操作可能要求重新创建表：  
  
-   在表中间添加一个新列  
  
-   删除列  
  
-   更改列为 Null 性  
  
-   更改列的顺序  
  
-   更改列的数据类型  
  
 若要更改此选项，请在“工具”  菜单中单击“选项”  ，展开“设计器”  ，然后单击“表设计器和数据库设计器”  。 选中或清除“阻止保存要求重新创建表的更改”  复选框。
