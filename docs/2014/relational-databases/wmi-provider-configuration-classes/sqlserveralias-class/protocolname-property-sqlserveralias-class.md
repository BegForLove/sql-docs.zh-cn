# <a name="protocolname-property-sqlserveralias-class"></a>ProtocolName 属性（SqlServerAlias 类）
  获取服务器连接别名使用的协议的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个[SqlServerAlias 类](sqlserveralias-class.md)对象，表示[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]别名。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定服务器连接别名使用的协议的名称的字符串值。
