# <a name="supportsstatementpooling-method-sqlserverdatabasemetadata"></a>supportsStatementPooling 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库是否支持语句池。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean supportsStatementPooling()  
```  
  
## <a name="return-value"></a>返回值  
 **true**如果支持。 否则为 **false**。  
  
## <a name="exceptions"></a>异常  
 java.sql.SQLException  
  
## <a name="remarks"></a>注释  
 由 java.sql.DatabaseMetaData 接口中的 supportsStatementPooling 方法指定此 supportsStatementPooling 方法。
