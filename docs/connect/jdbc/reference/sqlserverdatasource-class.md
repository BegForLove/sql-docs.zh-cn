# <a name="sqlserverdatasource-class"></a>SQLServerDataSource 类
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  表示连接到的特定属性的列表[!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)]通过使用数据库[SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md)对象。  
  
 **包：** com.microsoft.sqlserver.jdbc  
  
 **实现：** [ISQLServerDataSource](../../../connect/jdbc/reference/isqlserverdatasource-interface.md)，DataSource、 java.io.Serializable、 javax.naming.Referenceable  
  
## <a name="syntax"></a>语法  
  
```  
  
public class SQLServerDataSource  
```  
  
## <a name="remarks"></a>注释  
 此类支持到 SQLServerDataSource 类、 ISQLServerDataSource 接口和数据源接口解包。 有关详细信息，请参阅[包装和接口](../../../connect/jdbc/wrappers-and-interfaces.md)。
