# <a name="mssqlserver3260"></a>MSSQLSERVER_3260
    
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|3260|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|DMPIO_BUFFER_OVERFLOW|  
|消息正文|内部缓冲区已满。|  
  
## <a name="explanation"></a>解释
