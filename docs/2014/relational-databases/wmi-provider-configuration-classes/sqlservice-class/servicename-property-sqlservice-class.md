# <a name="servicename-property-sqlservice-class"></a>ServiceName 属性（SqlService 类）
  获取服务的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.ServiceName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示服务的 [SqlService 类](sqlservice-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定服务名称的字符串值。
