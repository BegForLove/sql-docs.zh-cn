# <a name="protocols-for-ltservergt-sql-server-configuration-manager"></a>适用于 &lt;服务器&gt; 的协议（SQL Server 配置管理器）
  若要配置 [!INCLUDE[ssDE](../../includes/ssde-md.md)]使用的协议，可以在控制台窗格中为所需服务器实例选择 **“协议”** ，在详细信息窗格中右键单击相应协议，然后单击 **“属性”**。
