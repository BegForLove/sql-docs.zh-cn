# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  从 Windows 8 开始删除了对可视化工作室分析器的支持（视觉工作室分析器仅包含在旧版本的 Visual Studio 中）。 对于其他故障排除机制，请使用 BID 跟踪。  
  
 Microsoft®可视化工作室™分析器提供了 ODBC 应用程序的高级视图。 开发人员可以从应用程序的角度而不是从组件或代码的角度评估、分析和调试结构、性能和交互。 Visual Studio 分析器收集的信息以事件的形式表示应用程序的两个组件之间的某种交互。 事件可以由函数调用和函数（如连接、数据库查询和事务）返回组成。
