# <a name="machineaccountidentity-property-wmi"></a>MachineAccountIdentity 属性 (WMI)
  获取安装了报表服务器的计算机的计算机帐户标识。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>属性值  
 一个`String`值，该值包含报表服务器安装的计算机的计算机帐户标识。
