# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations 和游标库
> [!IMPORTANT]  
>  此功能将在将来版本的 Windows 中删除。 避免在新的开发工作中使用此功能，并计划修改当前使用此功能的应用程序。 Microsoft 建议使用驱动程序的光标功能。  
  
 游标库不支持调用**SQLBulk 操作**。 有关**SQLBulk 操作**的一般信息，请参阅[SQLBulk 操作函数](../../../odbc/reference/syntax/sqlbulkoperations-function.md)。
