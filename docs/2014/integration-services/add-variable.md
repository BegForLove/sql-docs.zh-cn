# <a name="add-variable"></a>添加变量
  可以使用 **“添加变量”** 对话框指定新变量的属性。  
  
## <a name="options"></a>“常规”  
 **容器**  
 在列表中选择容器。 容器定义了变量的作用域。 容器可以是包，也可以是包中的可执行文件。  
  
 **名称**  
 键入变量名称。  
  
 **Namespace**  
 指定变量的命名空间。 默认情况下，用户定义变量位于 **User** 命名空间中。  
  
 **值类型**  
 选择数据类型。  
  
 **ReplTest1**  
 键入值。 该值必须与 **“值类型”** 选项中指定的数据类型相符。  
  
 **只读**  
 如果选择此项，则变量将是只读的。
