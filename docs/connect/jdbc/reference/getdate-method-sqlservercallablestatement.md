# <a name="getdate-method-sqlservercallablestatement"></a>getDate 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索指定参数的值作为 Java 编程语言中的 java.sql.Date 对象。

|Name|Description|  
|----------|-----------------|  
|[getDate (int)](../../../connect/jdbc/reference/getdate-method-int.md)|根据给定的参数索引，检索指定参数的值作为 Java 编程语言中的 java.sql.Date 对象。|  
|[getDate （int、 java.util.Calendar）](../../../connect/jdbc/reference/getdate-method-int-java-util-calendar.md)|与给定的参数索引和日历对象的 Java 编程语言中某个 java.sql.Date 对象中检索指定参数的值。|  
|[getDate (java.lang.String)](../../../connect/jdbc/reference/getdate-method-java-lang-string.md)|根据给定的参数名称，检索指定参数的值作为 Java 编程语言中的 java.sql.Date 对象。|  
|[getDate （java.lang.String，java.util.Calendar）](../../../connect/jdbc/reference/getdate-method-java-lang-string-java-util-calendar.md)|与给定的参数名称和日历对象的 Java 编程语言中某个 java.sql.Date 对象中检索指定参数的值。|
