# <a name="types-of-data-sources"></a>类型的数据源
有两种类型的数据源： 计算机数据源和文件数据源。 尽管它们都包含类似信息有关的数据源，但它们的区别在于存储此信息的方式。 由于这些差异，它们使用稍有不同的方式。  
  
 本部分包含以下主题。  
  
-   [计算机数据源](../../odbc/reference/machine-data-sources.md)  
  
-   [文件数据源](../../odbc/reference/file-data-sources.md)
