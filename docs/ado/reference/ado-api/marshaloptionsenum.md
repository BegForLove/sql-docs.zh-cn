# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
指定应返回哪些记录到服务器。  
  
|常量|ReplTest1|Description|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|默认值。 返回到服务器的所有行。|  
|**adMarshalModifiedOnly**|1|仅返回已修改的到服务器的行。|
