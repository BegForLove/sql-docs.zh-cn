# <a name="add-objects"></a>添加对象
  根据您希望查看的对象类型限制显示的对象选择对话框。  
  
## <a name="options"></a>选项  
 **特定对象**  
 打开“选择对象”对话框，可以进一步定义对象搜索。  
  
 **特定类型的所有对象**  
 打开“选择对象类型”对话框，可以指定应包含在基础列表中的对象类型。  
  
 **在服务器**_\<服务器名 >_  
 在指定服务器上添加所有适合的对象。 只出现在“登录名”-“新建”等服务器范围对话框中。  
  
 **属于该架构的所有对象**  
 添加由“架构名称”框中指定的架构拥有的所有对象。 只出现在“数据库用户”-“新建”等数据库作用域对话框中。
