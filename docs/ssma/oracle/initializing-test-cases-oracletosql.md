# <a name="initializing-test-cases-oracletosql"></a>初始化测试用例 (OracleToSQL)
在此步骤中，你将指定为新的测试用例的初始信息。  
  
## <a name="parameters"></a>Parameters  
**测试用例的名称**  
输入用于标识该测试用例的名称。  
  
**创建日期**  
今天的当前日期，自动定义。  
  
**上次修改的日期**  
填充自动保存功能。不应更改。  
  
**测试用例说明**  
输入任何其他信息以标识测试用例的用途。
