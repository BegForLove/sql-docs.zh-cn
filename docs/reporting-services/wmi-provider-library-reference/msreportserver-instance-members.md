# MSReportServer_Instance 成员
  MSReportServer_Instance 类为客户端提供连接到已安装的报表服务器所需的基本信息。  
  
## 公共属性  
  
|||  
|-|-|  
|[EditionID](../../reporting-services/wmi-provider-library-reference/editionid-property-wmi-msreportserver-instance.md)|返回报表服务器的版本类别。|  
|[EditionName](../../reporting-services/wmi-provider-library-reference/editionname-property-wmi-msreportserver-instance.md)|返回报表服务器版本类别的可读名称。|  
|[InstanceID](../../reporting-services/wmi-provider-library-reference/instanceid-property-wmi-msreportserver-instance.md)|此对象支持 [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] 基础结构，但不能通过代码直接使用。|  
|[InstanceName](../../reporting-services/wmi-provider-library-reference/instancename-property-wmi-msreportserver-instance.md)|返回特定计算机上报表服务器实例的名称。 只读。|  
|[IsSharePointIntegrated](../../reporting-services/wmi-provider-library-reference/issharepointintegrated-property-wmi-msreportserver-instance.md)|指示是否将报表服务器配置为 SharePoint 集成模式。|  
|[版本](../../reporting-services/wmi-provider-library-reference/version-property-wmi-msreportserver-instance.md)|返回报表服务器的版本。|
