# <a name="reserved-database-names-in-parallel-data-warehouse"></a>并行数据仓库中保留的数据库名称
保留的数据库名称是不能分配给 SQL Server PDW 中的用户数据库的名称。  
  
保留的数据库名称的列表：  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
