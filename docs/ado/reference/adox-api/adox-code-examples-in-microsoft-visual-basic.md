# <a name="adox-code-examples-in-microsoft-visual-basic"></a>Microsoft Visual Basic 中的 ADOX 代码示例

> [!NOTE]
>  将整个代码示例中的，从到 End Sub，子粘贴到您的代码编辑器。 如果使用完整的示例或段落格式设置都将丢失，该示例可能无法正确运行。
