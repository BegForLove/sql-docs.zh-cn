# <a name="issuedby-property-securitycertificate-class"></a>IssuedBy 属性（SecurityCertificate 类）
  获取颁发安全证书的对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.IssuedBy [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示安全证书的 [SecurityCertificate 类](securitycertificate-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定发出安全证书的对象的字符串值。
