# <a name="dependencies-property-sqlservice-class"></a>Dependencies 属性（SqlService 类）
  获取依赖于所引用服务的服务列表。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.Dependencies [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示服务的 [SqlService 类](sqlservice-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个包含依赖于所引用服务的服务列表的字符串数组。
