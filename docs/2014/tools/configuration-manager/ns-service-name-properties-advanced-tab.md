# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;服务名称&gt; 属性（“高级”选项卡）
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] 采用以下格式命名的服务作为实现**NS$**_< 用户配置的名称 >_。 如果定义了自定义属性，定义的属性及其属性值将显示在此选项卡上。 此选项卡有可能不列出任何属性。
