# <a name="sqlfreeenv-visual-foxpro-odbc-driver"></a>SQLFreeEnv（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：核心级别  
  
 关闭 Visual FoxPro ODBC 驱动程序并释放与该驱动程序关联的所有内存。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLFreeEnv](../../odbc/reference/syntax/sqlfreeenv-function.md) 。
