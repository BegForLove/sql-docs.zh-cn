# <a name="report-properties-dialog-box-code"></a>“报表属性”对话框 -&gt;“代码”
  选择**代码**上**报表属性**对话框可以定义由报表的表达式中使用自定义代码。  
  
## <a name="options"></a>选项  
 **自定义代码**  
 键入要在报表中使用的方法。 代码必须是用 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[vbprvb](../includes/vbprvb-md.md)]编写的。
