# 关闭跟踪窗口 (SQL Server Profiler)
  本主题介绍了如何关闭正在运行的跟踪或正在使用 [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]查看的已保存跟踪的跟踪窗口。  
  
### 关闭跟踪窗口  
  
1.  选择要关闭的跟踪窗口。  
  
2.  在 **“文件”** 菜单上，单击 **“关闭”**。  
  
> [!IMPORTANT]  
>  如果跟踪正在运行，则关闭跟踪窗口将停止相关跟踪。
