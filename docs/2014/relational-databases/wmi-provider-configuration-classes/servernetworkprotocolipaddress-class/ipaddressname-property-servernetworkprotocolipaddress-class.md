# <a name="ipaddressname-property-servernetworkprotocolipaddress-class"></a>IpAddressName 属性（ServerNetworkProtocolIPAddress 类）
  获取 IP 地址的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.IpAddressName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个[ServerNetworkProtocolIPAdress 类](servernetworkprotocolipaddress-class.md)对象，表示的实例上的网络协议 IP 地址[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定 IP 地址名称的字符串值。
