# <a name="object-already-exists"></a>对象已存在
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
您试图保存的对象与数据库中已有的某个对象同名。 请另选一个名称，然后继续。  
  
也可指示列名已存在。
