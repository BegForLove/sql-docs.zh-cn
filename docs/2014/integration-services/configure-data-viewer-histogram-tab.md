# <a name="configure-data-viewer-histogram-tab"></a>配置数据查看器（“直方图”选项卡）
  可以使用 **“配置数据查看器”** 对话框的 **“直方图”** 选项卡指定要显示的列。 直方图可以按值的分布显示数值列中的数据值。 该图表只反应一个缓冲区的内容。  
  
## <a name="options"></a>选项  
 **可视化的列**  
 选择要以直方图形式显示的列。
