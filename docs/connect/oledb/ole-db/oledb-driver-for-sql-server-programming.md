# <a name="ole-db-driver-for-sql-server-programming"></a>用于 SQL Server 编程的 OLE DB 驱动程序
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-asdbmi-md](../../../includes/appliesto-ss-asdb-asdw-pdw-asdbmi-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  SQL Server 的 OLE DB 驱动程序是一种低级别的 COM API，用于访问数据。 SQL Server 的 OLE DB 驱动程序开发工具、 实用工具，或者需要高性能的低级别组件的建议。 SQL Server 的 OLE DB 驱动程序是本机、 高性能提供程序访问[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]表格数据流 (TDS) 协议直接。  
  
 OLE DB 驱动程序的 SQL Server 提供对应用程序连接到 OLE DB 支持[!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]。  
  
 SQL Server 的 OLE DB 驱动程序是 OLE DB 版本 2.0 – 符合访问接口。
