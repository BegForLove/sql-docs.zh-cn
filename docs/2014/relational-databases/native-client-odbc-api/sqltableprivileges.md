# <a name="sqltableprivileges"></a>SQLTablePrivileges
  **SQLTablePrivileges**可以执行对静态游标。 尝试执行**SQLTablePrivileges**上 SQL_SUCCESS_WITH_INFO，该值指示游标类型已更改可更新 （键集驱动或动态） 返回。
