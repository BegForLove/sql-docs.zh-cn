# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;服务名称&gt; 属性（“高级”选项卡）
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] 以名称格式为 **NS$** _<user_configured name>_ 的服务的形式实现。 如果定义了自定义属性，定义的属性及其属性值将显示在此选项卡上。 此选项卡有可能不列出任何属性。
