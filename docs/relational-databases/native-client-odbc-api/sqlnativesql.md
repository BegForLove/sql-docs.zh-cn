[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client ODBC 驱动程序满足**SQLNativeSql**而无需访问服务器的请求。 该函数有效地测试 SQL 语句的语法。 语法检查不能决定是否有效，标识符或 SQL 语句中的表达式的结果和[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]本机 SQL 返回**SQLNativeSql**可能无法运行。
