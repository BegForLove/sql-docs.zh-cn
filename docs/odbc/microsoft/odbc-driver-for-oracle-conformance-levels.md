# <a name="odbc-driver-for-oracle-conformance-levels"></a>ODBC Driver for Oracle 一致性级别
> [!IMPORTANT]  
>  将 Windows 的未来版本中删除该功能。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 ODBC 定义两种类型的驱动程序的一致性标准： 标准 API 一致性和标准 SQL 语法合规性。 API 一致性驱动程序支持的函数的引用。 SQL 一致性是指的驱动程序支持的 SQL 语法。 每个标准的一致性级别组成。  
  
 本部分提供一致性适用于 Oracle 的 ODBC 驱动程序信息。  
  
 本部分包含以下主题。  
  
-   [API 一致性级别 （适用于 Oracle ODCB 驱动程序）](../../odbc/microsoft/api-conformance-level-odbc-driver-for-oracle.md)  
  
-   [支持的选项 （适用于 Oracle ODCB 驱动程序）](../../odbc/microsoft/supported-options-odbc-driver-for-oracle.md)  
  
-   [SQL 一致性级别 （适用于 Oracle ODCB 驱动程序）](../../odbc/microsoft/sql-conformance-levels-odbc-driver-for-oracle.md)
