# <a name="programming-with-ado-md"></a>使用 ADO MD 进行编程
若要将 ADO MD 与开发工具一起使用，应建立对 ADO MD 类型库的引用。 ADO MD 库的说明是 Microsoft ActiveX 数据对象（多维）库。 ADO MD 库文件名是 msadomd，程序 ID （ProgID）是 "ADOMD"。 有关如何建立对库的引用的详细信息，请参阅开发工具的文档。
