# <a name="raw-file-source-editor-columns-page"></a>原始文件源编辑器（“列”页）
  原始文件源从文件中读取原始数据。 因为数据的表示方式是源所固有的，所以数据无需转换，并且几乎不需要分析。 有关详细信息，请参阅 [Raw File Source](data-flow/raw-file-source.md)。
