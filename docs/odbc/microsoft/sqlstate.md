# <a name="sqlstate"></a>SQLSTATE
对于数据源中出现的错误，ODBC 驱动程序将返回本机错误映射到相应的 SQLSTATE。 有错误，如果检测到由驱动程序或驱动程序管理器，ODBC 驱动程序或驱动程序管理器生成相应的 SQLSTATE。
