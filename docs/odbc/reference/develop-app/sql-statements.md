# <a name="sql-statements"></a>SQL 语句
ODBC 应用程序通过执行 SQL 语句来执行几乎所有数据库访问。 这些语句的形式（在运行时硬编码或构造、互操作或特定于数据源等）取决于应用程序的需求。
