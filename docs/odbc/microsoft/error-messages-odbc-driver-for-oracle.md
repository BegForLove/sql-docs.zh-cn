# <a name="error-messages-odbc-driver-for-oracle"></a>错误消息 （适用于 Oracle 的 ODBC 驱动程序）
> [!IMPORTANT]  
>  将 Windows 的未来版本中删除该功能。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 发生错误时，Microsoft ODBC Driver for Oracle 将返回 SQLSTATE （ODBC 错误代码） 和一条错误消息。 从驱动程序检测到的错误和 Oracle Server 返回的错误，该驱动程序将派生此信息。
