# <a name="post-download-instructions"></a>发布下载指令
  使用此页可以指定目标服务器的下载指令。  
  
## <a name="options"></a>“常规”  
 **指令类型**  
 指定要发布的下载指令的类型。  
  
 **Description**  
 查看下载指令的说明。  
  
 **轮询间隔**  
 设置目标服务器的轮询间隔。 仅适用于“设置轮询间隔”指令。  
  
 **所有目标服务器**  
 选择此选项可以将下载指令发布到所有目标服务器。  
  
 **以下目标服务器**  
 选择此选项可以将下载指令发布到所选目标服务器。  
  
 **选择**  
 指定目标服务器应接收下载指令。  
  
 **目标服务器**  
 查看目标服务器的名称。  
  
 **本地时间**  
 按照目标服务器所在的时区来查看其日期和时间。  
  
 **轮询间隔**  
 查看目标服务器的当前轮询间隔。
