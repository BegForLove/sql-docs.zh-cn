# <a name="default-subkey"></a>默认子项
注册表可能会使用默认子项指定默认数据源。 此子项是一种特殊情况的数据源规范子项，并具有与任何其他数据源规范子项相同的值。 唯一的区别是在 ODBC 数据源的子项下的值作为未列出。
