# <a name="nodejs-driver-for-sql-server"></a>适用于 SQL Server 的 Node.js 驱动程序

![下载向下箭头线圈出](../../ssdt/media/download.png)[下载 Node.js SQL 驱动程序](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

繁琐的模块是 TDS 协议的 javascript 实现，由所有新式版本的 SQL Server 提供支持。 该驱动程序是一个开放源代码项目，Github 上提供。  
  
可以连接到 SQL 数据库使用 Node.js 在 Windows、 Linux 或 mac。
