# <a name="step-1-configure-development-environment-for-php-development"></a>步骤 1：配置用于 PHP 开发的开发环境

* 标识要使用哪个版本的 PHP 驱动程序，根据你的环境，此处所述： [PHP SQL 驱动程序的系统要求](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* 下载并安装适用的 PHP 驱动此处：[下载 Microsoft PHP 驱动程序](https://www.microsoft.com/download/details.aspx?id=20098)

* 将 IIS 配置托管 PHP 应用程序，此处所述：[配置 IIS 的 PHP SQL 驱动程序](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux 和 Mac


*   配置加载 PHP 驱动程序和配置 web 服务器托管 PHP 应用程序，此处所述： [PHP Linux 和 Mac 驱动程序安装教程](../../connect/php/installation-tutorial-linux-mac.md)
