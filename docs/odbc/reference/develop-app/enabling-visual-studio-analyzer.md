# <a name="enabling-visual-studio-analyzer"></a>启用 Visual Studio Analyzer
> [!IMPORTANT]  
>  从 Windows 8 开始删除了对可视化工作室分析器的支持（视觉工作室分析器仅包含在旧版本的 Visual Studio 中）。 对于其他故障排除机制，请使用 BID 跟踪。  
  
 Microsoft Visual Studio 分析器是一种应用程序分析工具，旨在提供应用程序在所有层和系统中性能的高级观察。 它侧重于组件之间的交互。 可以从 ODBC 管理员的"跟踪"选项卡启动和停止可视化工作室分析器。  
  
 要查看 ODBC 生成的任何事件，请按照以下步骤操作：  
  
1.  访问 ODBC 数据源管理员。  
  
2.  单击“跟踪”**** 选项卡。  
  
3.  单击 **"开始可视化工作室分析器跟踪**"。  
  
4.  单击“确定”。   
  
 可视化工作室分析器事件生成将继续，直到选择**停止视觉工作室分析器跟踪**。
