# <a name="internet-explorer-error-codes-for-remote-data-service"></a>远程数据服务的 Internet 资源管理器错误代码
下表列出了 Microsoft® Internet Explorer 错误代码与远程数据服务使用情况相关。 将显示低两个字节的正小数转、完整误差代码的负十进制转换和十六进制值。

|互联网浏览器（维诺）错误|Number|说明|
|------------------------------------------|------------|-----------------|
|**IDS_WinInet_CantConnect**|8195 -2146820093 0x800A2003|Internet 客户端错误：无法连接到服务器。|
|**IDS_WinInet_连接重置**|12031 -2146816257 0x800A2EFF|互联网客户端错误：连接重置。|
|**IDS_WinInet_Error**|8193 -2146820095 0x800A2001|互联网客户端错误。|
|**IDS_WinInet_无效服务器响应**|8430 -2146819858 0x800A20EE|Internet 客户端错误：服务器响应无效。|
|**IDS_WinInet_ SSLPost 限制**|8196 -2146820092 0x800A2004|互联网客户端错误：TLS/SSL 错误（可能为 32K 数据上传限制）。|
|**IDS_WinInet_Timeout**|8194 -2146820094 0x800A2002|互联网客户端错误：请求超时。|
