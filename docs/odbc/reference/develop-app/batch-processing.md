# <a name="batch-processing"></a>批处理
对 SQL 语句的批处理支持不是被广泛使用，因此可互操作应用程序有条件地或根本不应使用它们。 有关详细信息，请参阅[执行批处理](../../../odbc/reference/develop-app/executing-batches.md)。
