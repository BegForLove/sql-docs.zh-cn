# <a name="ado-fundamentals"></a>ADO 基础知识
ADO 有关以编程方式访问、 编辑和更新数据的各种数据源通过 OLE DB 系统接口，使开发人员功能强大、 逻辑的对象模型。 ADO 的最常见的用法是查询表或关系数据库中的表、 检索和应用程序中显示的结果以及可能允许用户更改和保存更改的数据。 其他任务包括：  
  
-   查询数据库使用 SQL 和显示结果。  
  
-   通过 Internet 访问文件存储中的信息。  
  
-   操作消息和电子邮件系统中的文件夹。  
  
-   正在从数据库的数据保存到 XML 文件。  
  
-   执行与 XML 中所述的命令和检索的 XML 流。  
  
-   将数据保存到二进制或 XML 流。  
  
-   允许用户查看和更改数据库表中的数据。  
  
-   创建和重用参数化数据库命令。  
  
-   执行存储的过程。  
  
-   动态创建灵活的结构，它名为**记录集**，以保存、 导航和操作数据。  
  
-   执行事务性数据库操作。  
  
-   筛选和排序的运行时条件的数据库信息的本地副本。  
  
-   创建和操作数据库中的分层结果。  
  
-   绑定到数据识别组件数据库字段。  
  
-   创建远程，断开**记录集**。  
  
 ADO 公开各种选项和设置来提供这种灵活性。 因此，务必了解如何使用 ADO 应用程序，将每个目标都分成可管理的部分中需要有序的方法。
