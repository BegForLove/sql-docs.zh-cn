# <a name="sha-property-securitycertificate-class"></a>SHA 属性（SecurityCertificate 类）
  获取安全证书的 SHA 指纹属性。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.SHA [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 [SecurityCertificate 类] securitycertificate-class.md) 代表安全证书对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值
