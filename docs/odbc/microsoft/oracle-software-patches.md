# <a name="oracle-software-patches"></a>Oracle 软件修补程序
> [!IMPORTANT]  
>  此功能将在将来版本的 Windows 中删除。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 而是使用 Oracle 提供的 ODBC 驱动程序。  
  
 Oracle 服务器产品及其客户端组件的修补程序是几种 Microsoft 产品和技术正常运行所必需的，包括 Oracle 的 Microsoft ODBC 驱动程序、Oracle 的 Microsoft OLE DB 提供商、互联网信息服务 （IIS）、组件服务（或 Microsoft 事务服务器，如果您使用的是 Windows NT），等等。  
  
> [!NOTE]  
>  以下说明可能不完全准确，因为 Oracle FTP 站点可能会更改。  
  
### <a name="to-download-the-oracle-software-patches"></a>下载 Oracle 软件修补程序  
  
1.  oracle-ftp.oracle.com连接到公共 FTP 站点。 用户 ID 是"匿名的"，密码是您的电子邮件地址。  
  
2.  导航到以下目录：/服务器/wgt_tech/服务器/窗口NT。  
  
3.  要下载与 Windows 95、Windows 98 和 Windows NT/Windows 2000 最相关的修补程序，请导航到版本的 Oracle 的子目录 - 7.3 或 8.0。 两个子目录是 /73 修补程序集和 /80 个修补程序集。  
  
4.  要下载 Oracle 网络技术的修补程序（SQL_Net 或 Net8），请导航到以下目录：/网络。  
  
 从 Web 浏览器访问此 FTP 网站可能不起作用。 如果遇到问题，请尝试使用"传统"FTP 客户端或使用 DOS 命令提示符。  
  
> [!NOTE]  
>  由于 Oracle 修复了当前版本中的错误，然后使用软件修补程序将它们改装为早期版本，因此建议您下载可用的最新修补程序。 对于 Oracle 服务器客户端组件尤其如此。 如果您对安装这些修补程序有疑问，请联系 Oracle 支持。
