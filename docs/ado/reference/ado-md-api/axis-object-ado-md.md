# <a name="axis-object-ado-md"></a>轴对象 (ADO MD)
表示一个位置或筛选器轴的单元集，包含所选的成员的一个或多个维度。  
  
## <a name="remarks"></a>Remarks  
 **轴**可被包含对象[轴](../../../ado/reference/ado-md-api/axes-collection-ado-md.md)集合，或者由[FilterAxis](../../../ado/reference/ado-md-api/filteraxis-property-ado-md.md)属性[单元集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)。  
  
 使用集合和属性的**轴**对象，你可以执行以下操作：  
  
-   标识**轴**与[名称](../../../ado/reference/ado-md-api/name-property-ado-md.md)属性。  
  
-   循环访问每个位置沿**轴**使用[位置](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)集合。  
  
-   上获得的维数**轴**与[DimensionCount](../../../ado/reference/ado-md-api/dimensioncount-property-ado-md.md)属性。  
  
-   获取提供程序特定属性的**轴**用标准 ADO[属性](../../../ado/reference/ado-api/properties-collection-ado.md)集合。
