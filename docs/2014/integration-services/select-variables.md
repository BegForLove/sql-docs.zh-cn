# <a name="select-variables"></a>选择变量
  可以使用 **“选择变量”** 对话框，指定在消息队列任务的发送消息操作中使用的变量。 “可用变量”列表包括消息队列任务或其父容器作用域内的系统变量和用户定义变量  。 该任务使用 **“所选变量”** 列表中的变量。  
  
## <a name="options"></a>选项  
 **可用变量**  
 选择一个或多个变量。  
  
 **“所选变量”**  
 选择一个或多个变量。  
  
 **右箭头**  
 将所选变量移动到“所选变量”列表  。  
  
 **左箭头**  
 将所选变量移回“可用变量”  列表。  
  
 **新建变量**  
 创建新变量。
