# VirtualDirectoryReportServer 属性 (WMI MSReportServer_ConfigurationSetting)
  返回报表服务器应用程序的虚拟目录。 只读。  
  
## 语法  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## 属性值  
 包含虚拟目录的 **string** 。
