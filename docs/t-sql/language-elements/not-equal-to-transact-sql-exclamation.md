# <a name="not-equal-to-transact-sql---exclamation"></a>不等于 (Transact SQL) - 感叹号
[!INCLUDE[tsql-appliesto-ss2008-all-md](../../includes/tsql-appliesto-ss2008-all-md.md)]

  测试某个表达式是否不等于另一个表达式（比较运算符）。 如果任何一个操作数为 NULL，或两个都为 NULL，则返回 NULL。 其功能与 [<>（不等于）](../../t-sql/language-elements/not-equal-to-transact-sql-traditional.md)比较运算符相同。
