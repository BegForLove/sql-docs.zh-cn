# <a name="type-property-key-adox"></a>Type 属性（项）(ADOX)
指示项的类型。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回**长**值，该值可之一[KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md)常量。 默认值是**adKeyPrimary**。  
  
## <a name="remarks"></a>备注  
 此属性是只读[密钥](../../../ado/reference/adox-api/key-object-adox.md)已追加到集合的对象。
