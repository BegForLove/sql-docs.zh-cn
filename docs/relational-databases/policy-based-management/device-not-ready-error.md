# <a name="device-not-ready-error"></a>设备未就绪错误
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  此规则检查系统事件日志中是否存在 EventId 15。 此错误可能是由 SCSI 主机适配器配置问题或相关问题引起的。
