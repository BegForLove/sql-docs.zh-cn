## <a name="syntax"></a>语法  
  
```  
  
object  
.InstanceName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象

 一个指定 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 客户端实例的名称的字符串值。
