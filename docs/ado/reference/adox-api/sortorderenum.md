# <a name="sortorderenum"></a>SortOrderEnum
指定索引列的排序顺序。  
  
|常量|ReplTest1|Description|  
|--------------|-----------|-----------------|  
|**adSortAscending**|1|默认值。 列的排序顺序为升序。|  
|**adSortDescending**|2|列的排序顺序为降序。|
