# <a name="getmaxcharliterallength-method-sqlserverdatabasemetadata"></a>getMaxCharLiteralLength 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库在字符文本中允许的最大字符数。  
  
## <a name="syntax"></a>语法  
  
```  
  
public int getMaxCharLiteralLength()  
```  
  
## <a name="return-value"></a>返回值  
 **Int** ，该值指示最大允许的字符数。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.DatabaseMetaData 接口中的 getMaxCharLiteralLength 方法指定此 getMaxCharLiteralLength 方法。
