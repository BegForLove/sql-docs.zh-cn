# <a name="getstringfunctions-method-sqlserverdatabasemetadata"></a>getStringFunctions 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索的逗号分隔的列表**字符串**与此数据库可用的函数。  
  
## <a name="syntax"></a>语法  
  
```  
  
public java.lang.String getStringFunctions()  
```  
  
## <a name="return-value"></a>返回值  
 A**字符串**包含**字符串**函数。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.DatabaseMetaData 接口中的 getStringFunctions 方法指定此 getStringFunctions 方法。
