# <a name="sample-jdbc-driver-applications"></a>示例 JDBC 驱动程序应用程序
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] 示例应用程序演示了 JDBC 驱动程序的各项功能。 此外，这些示例应用程序还展示了在使用 JDBC 驱动程序和 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 数据库时可以参考的最佳编程实践。  
  
 所有示例应用程序都包含在 *.java 代码文件中，它们可以在您的本地计算机上编译和运行，并且位于以下位置中的不同子文件夹下：  
  
 \<*安装目录*> \sqljdbc_\<*版本*>\\<*语言*> \samples  
  
 此部分的主题说明了如何配置和运行示例应用程序，还包括了对示例应用程序所说明内容的讨论。  
  
## <a name="in-this-section"></a>本节内容  
  
|主题|Description|  
|-----------|-----------------|  
|[连接和检索数据](../../connect/jdbc/connecting-and-retrieving-data.md)|这些示例应用程序说明如何连接 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 数据库。 它们还说明了可以从 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 数据库中检索数据的不同方法。|  
|[使用数据类型](../../connect/jdbc/working-with-data-types-jdbc.md)|这些示例应用程序说明如何使用 JDBC 驱动程序数据类型方法来处理 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 数据库中的数据。|  
|[使用结果集](../../connect/jdbc/working-with-result-sets.md)|这些示例应用程序说明如何使用结果集来处理 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 数据库中包含的数据。|  
|[处理大型数据](../../connect/jdbc/working-with-large-data.md)|这些示例应用程序演示了如何使用自适应缓冲，以便在无需服务器游标开销的情况下从 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 数据库中检索大值数据。|
