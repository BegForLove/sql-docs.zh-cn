# <a name="setdouble-method-sqlserverpreparedstatement"></a>setDouble 方法 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  将指定的参数设置为给定**double**值。  
  
## <a name="syntax"></a>语法  
  
```  
  
public final void setDouble(int n,  
                            double x)  
```  
  
#### <a name="parameters"></a>Parameters  
 *n*  
  
 **Int** ，该值指示参数号。  
  
 *x*  
  
 A **double**值。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.PreparedStatement 接口中的 setDouble 方法指定此 setDouble 方法。
