# <a name="run-sql-server-profiler"></a>运行 SQL Server Profiler
  若要运行 [!INCLUDE[ssSqlProfiler](../includes/sssqlprofiler-md.md)]，请在 **“开始”** 菜单上依次指向 **“所有程序”**、 **Microsoft SQL Server 2014**和 **“性能工具”**，然后单击 **SQL Server Profiler**。
