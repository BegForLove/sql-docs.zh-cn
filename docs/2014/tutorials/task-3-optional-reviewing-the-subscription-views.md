# <a name="task-3-optional-reviewing-the-subscription-views"></a>任务 3（可选）：查看订阅视图
  在本任务中，您将确认使用 SQL Server Management Studio 创建了 SQL 视图。  
  
1.  启动**SQL Server Management Studio**。 单击**启动**菜单上，单击**所有程序**，单击**Microsoft SQL Server 2012**，然后单击**SQL Server Management Studio**。  
  
2.  在**连接到服务器**窗口中，设置**服务器类型**到**数据库引擎**，类型**服务器名称**(或选择 **(本地）**，然后选择相应**身份验证**，然后单击**连接**。  
  
3.  在**对象资源管理器**窗格中，展开**数据库**，展开**MDS**，然后展开**视图**。  
  
4.  确认你看到**mdm。供应商**列表中的视图。  
  
     ![SQL Server Management Studio-mdm。供应商视图](../../2014/tutorials/media/et-reviewingthesubscriptionviews.jpg "SQL Server Management Studio-mdm。供应商视图")
