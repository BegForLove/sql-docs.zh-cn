# <a name="sqlstatistics"></a>SQLStatistics
  可以对静态游标执行**SQLStatistics** 。 尝试对可更新的（键集驱动或动态）执行**SQLStatistics**将返回 SQL_SUCCESS_WITH_INFO 指示游标类型已更改。
