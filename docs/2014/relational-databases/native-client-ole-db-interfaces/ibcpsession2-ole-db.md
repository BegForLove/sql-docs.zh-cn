# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  IBCPSession2 接口是提供一个成员函数，则为每个列调用 IBCPSession::BCPColFmt 的替代方法的 IBCPSession 的扩展。  IBCPSession2 从 IBCPSession 继承，并添加一个新方法： [IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md)。
