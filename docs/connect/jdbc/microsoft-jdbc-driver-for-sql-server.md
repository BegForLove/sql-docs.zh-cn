# <a name="microsoft-jdbc-driver-for-sql-server"></a>Microsoft JDBC Driver for SQL Server

[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

通过在互操作性方面不断投入，Microsoft 提供了适用于 SQL Server 和 Azure SQL 数据库的 Java Database Connectivity (JDBC) 驱动程序。 该驱动程序免费提供，不需要额外付费，并且可从任何 Java 应用程序、应用程序服务器或支持 Java 的小程序连接 Java 数据库。 该驱动程序是 Type 4 JDBC 驱动程序，通过标准 JDBC 应用程序编程接口 (API) 提供数据库连接。

已针对大多数应用程序服务器（如 IBM WebSphere 和 SAP NetWeaver）对 Microsoft SQL Server JDBC 驱动程序进行了测试。
