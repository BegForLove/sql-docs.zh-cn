# <a name="delete-a-power-pivot-data-feed-library"></a>删除 Power Pivot 数据馈送库
  您必须是站点所有者才能删除数据馈送库。 如果库中包含项，则它们将随同库一起被删除。  
  
 您必须知道您要删除的数据馈送库的名称。  
  
### <a name="delete-a-data-feed-library"></a>删除数据馈送库  
  
1.  单击 **“网站操作”**。  
  
2.  单击 **“网站设置”**。  
  
3.  在“网站管理”下单击 **“网站库和列表”**。  
  
4.  单击**自定义"\<库名称 >"**  
  
5.  在“权限和管理”下单击 **“删除此库”**。
