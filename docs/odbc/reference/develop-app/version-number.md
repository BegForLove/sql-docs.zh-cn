# <a name="version-number"></a>版本号
ODBC 有多个版本，每个版本具有不同的功能。 应用程序通过使用SQL_ODBC_VER和SQL_DRIVER_ODBC_VER选项调用**SQLGetInfo**来确定驱动程序管理器和特定驱动程序支持的 ODBC 版本。
