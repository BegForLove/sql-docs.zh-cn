# WindowsServiceIdentityActual 属性 (WMI MSReportServer_ConfigurationSetting)
  用于运行报表服务器 Windows 服务的标识。 只读。  
  
## 语法  
  
```vb  
Public Dim WindowsServiceIdentityActual As String  
```  
  
```csharp  
public string WindowsServiceIdentityActual;  
```  
  
## 属性值  
 包含用于运行报表服务器 Windows 服务的标识的 **String** 对象。
