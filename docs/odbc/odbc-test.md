# <a name="odbc-test"></a>ODBC 测试
Microsoft® ODBC 测试是一个支持 ODBC 的应用程序，可用于测试 ODBC 驱动程序和 ODBC 驱动程序管理器。 ODBC 3.51 包括 ANSI 和启用 Unicode 的 ODBC 测试版本。 相应的文件如下：  
  
-   Odbcte32.exe 和 Gtrtst32.dll，适用于 ANSI 版本。  
  
-   Odbct32w.exe 和 Gtrts32w.dll，适用于 Unicode 版本。  
  
 要使用 ODBC 测试，您必须了解 ODBC API、C 语言和 SQL。 有关 ODBC API 的详细信息，请参阅[ODBC 程序员的参考](../odbc/reference/odbc-programmer-s-reference.md)。  
  
 以前包含在文档此部分中的帮助主题现在包含在 ODBC 测试计划中。 打开 Odbcte32.exe 或 Odbct32w.exe，打开 **"帮助"** 菜单，然后单击"**帮助主题**"。  
  
 请注意，这些应用程序的 64 位版本（用于 64 位 Microsoft Windows 操作系统）的名称与 32 位版本的名称相同，即使它们是单独的文件。 即 64 位版本的 ODBC 测试的 Unicode 版本的名称是 odbct32w.exe。
