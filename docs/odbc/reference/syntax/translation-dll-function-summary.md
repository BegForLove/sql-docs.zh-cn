# <a name="translation-dll-function-summary"></a>转换 DLL 函数摘要
下表描述了转换 DLL 函数。 有关每个函数的语法和语义的详细信息，请参阅[翻译 DLL 函数引用](../../../odbc/reference/syntax/translation-dll-api-reference.md)。  
  
|任务|函数名称|目标|  
|----------|-------------------|-------------|  
|翻译数据|[SQLDataSourceto驱动程序](../../../odbc/reference/syntax/sqldatasourcetodriver-function.md)|将从数据源流向驱动程序的所有数据翻译为驱动程序。|  
||[SQLDriverto数据源](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)|将从驱动程序流向数据源的所有数据翻译为数据源。|
