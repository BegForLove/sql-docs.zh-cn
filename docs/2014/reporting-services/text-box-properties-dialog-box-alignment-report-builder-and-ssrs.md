# <a name="text-box-properties-dialog-box-alignment-report-builder-and-ssrs"></a>“文本框属性”对话框 -&gt;“对齐”（报表生成器和 SSRS）
  在 **“文本框属性”** 对话框中选择 **“对齐”** 可以在选定的单元或数据区域内按一定方式放置值。  
  
## <a name="options"></a>选项  
 **水平**  
 指定在选定单元或数据区域中水平放置值的位置。 默认情况下选择的是 **“默认值”** 。 “默认值”表示的值是 **“常规”**。 有关常规文本对齐计算方式的详细信息，请参阅 [设置文本和占位符的格式（报表生成器和 SSRS）](report-design/formatting-text-and-placeholders-report-builder-and-ssrs.md)。 单击“表达式” (*fx*) 按钮可编辑表达式。  
  
 **垂直**  
 指定在选定单元或数据区域中垂直放置值的位置。 默认情况下选择的是 **“默认值”** 。 “默认值”表示的值是 **“靠上”**。 单击“表达式” (*fx*) 按钮可编辑表达式。  
  
 **左侧**  
 指定单元左边缘与单元中显示值之间的填充量或间距大小。 默认值为 2 磅。 单击“表达式” (*fx*) 按钮可编辑表达式。  
  
 **右侧**  
 指定单元右边缘与单元中显示值之间的填充量或间距大小。 默认值为 2 磅。 单击“表达式” (*fx*) 按钮可编辑表达式。  
  
 **Top**  
 指定单元上边缘与单元中显示值之间的填充量或间距。 默认值为 2 磅。 单击“表达式” (*fx*) 按钮可编辑表达式。  
  
 **底部**  
 指定单元下边缘与单元中显示值之间的填充量或间距。 默认值为 2 磅。 单击“表达式” (*fx*) 按钮可编辑表达式。
