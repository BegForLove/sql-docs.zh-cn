# <a name="datasize-element-assl"></a>DataSize 元素 (ASSL)

## <a name="syntax"></a>语法  
  
```xml  
  
<DataItem>  
   ...  
   <DataSize>...</DataSize>  
   ...  
</DataItem  
```

|数据类型和长度|Integer|  
|默认值|无|  
|基数|0-1：可出现一次且仅出现一次的可选元素。|
