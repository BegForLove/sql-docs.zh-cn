# <a name="clearwarnings-method-sqlserverconnection"></a>clearWarnings 方法 (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  清除所报告的有关此 [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) 对象的所有警告。  
  
## <a name="syntax"></a>语法  
  
```  
  
public void clearWarnings()  
```  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 clearWarnings 方法由 java.sql.Connection 接口中的 clearWarnings 方法指定。
