# <a name="leveldepth-property-ado-md"></a>LevelDepth 属性 (ADO MD)
指示之间的层次结构的根级别数和一个[成员](../../../ado/reference/ado-md-api/member-object-ado-md.md)。  
  
## <a name="return-values"></a>返回值  
 返回**长**整数，并且是只读的。  
  
## <a name="remarks"></a>备注  
 使用**LevelDepth**属性来确定的距离[成员](../../../ado/reference/ado-md-api/member-object-ado-md.md)的根级别的层次结构中的对象。 **LevelDepth**的某一成员在根级别为 0。 这对应于[深度](../../../ado/reference/ado-md-api/depth-property-ado-md.md)的属性[级别](../../../ado/reference/ado-md-api/level-object-ado-md.md)对象。

 [成员对象 (ADO MD)](../../../ado/reference/ado-md-api/member-object-ado-md.md)
