# <a name="mssqlserver12302"></a>MSSQLSERVER_12302
  
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|12302|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|消息正文|“*construct*”不支持更新 PRIMARY KEY 约束中包含的列。|  
  
## <a name="user-action"></a>用户操作  
请勿更新主键约束中包含的列。
