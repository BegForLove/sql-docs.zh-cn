# <a name="toolbar-aggregations-tab-cube-designer"></a>工具栏（多维数据集设计器的“聚合”选项卡）
  可以使用 **“工具栏”** 窗格，执行多维数据集设计器中的 **“聚合”** 选项卡上的常规操作。  
  
## <a name="options"></a>选项  
  
|Option|Description|  
|------------|-----------------|  
|**设计聚合**|单击以显示 **“聚合设计向导”**。<br /><br /> 注意：聚合设计器的标准视图中仅显示此选项。|  
|**基于使用情况的优化**|单击以显示 **“基于使用情况的优化向导”**。<br /><br /> 注意：聚合设计器的标准视图中仅显示此选项。|  
|**分配聚合设计**|单击以显示 **“分配聚合设计”** 对话框。|  
|**标准视图**|单击以显示聚合设计器的 **“标准视图”** ，在该视图中，您可以使用 **“聚合设计向导”** 或 **“基于使用情况的优化向导”** 来创建和修改聚合设计。|  
|**高级的视图**|单击以显示聚合设计器的 **“高级视图”** ，在该视图中，您可以手动创建和修改聚合设计及单个聚合。 此视图应仅允许高级用户使用。<br /><br /> 注意：**聚合设计向导**并**基于使用情况的优化向导**在此视图中不可用。|  
|**新的聚合设计**|单击以显示 **“新建聚合设计”** 对话框。 您可以创建一个空的聚合设计，也可以创建现有的聚合设计的副本。<br /><br /> 注意：聚合设计器的高级视图中仅显示此选项。|  
|**删除**|单击以删除选定的聚合设计。  在聚合设计器的高级视图中，它还可用来从聚合设计中删除聚合。|
