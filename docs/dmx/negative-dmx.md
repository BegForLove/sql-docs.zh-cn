# <a name="--negative-dmx"></a>-（负）(DMX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  执行一元运算，以返回数值表达式的负值。  
  
## <a name="syntax"></a>语法  
  
```  
  
- Numeric_Expression  
```  
  
#### <a name="parameters"></a>Parameters  
 *Numeric_Expression*  
 一个返回数值的有效数据挖掘扩展 (DMX) 表达式。  
  
## <a name="return-value"></a>返回值  
 类型为指定参数的数据类型的值。
