## <a name="icommandexecute"></a>ICommand::Execute  
 如果插入的数据大于列的大小，通常会导致错误。 但是，可能会出现将返回 S_OK、但 dwStatus 将设置为 DBSTATUS_S_TRUNCATED 的情况。 这通常发生在时插入数据使用参数，其中列是不足够大以保存数据，并**icommandwithparameters:: Setparameterinfo**尚未调用。
