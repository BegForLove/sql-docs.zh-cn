# 第 1 课：SQL Server Management Studio 中的基本导航
[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] 是一个功能强大且灵活的工具。 但是，初次使用 Microsoft Visual Studio 的用户有时无法以最快的方式访问所需的功能。 本课介绍各个窗口组成设计界面的方式，帮助您轻松完成工作。 本课结束后，您将了解如何清除不需要的窗口，如何快速访问包含所需信息的窗口，以及如何配置默认界面以支持经常处理的工作。  
  
本课程包含以下主题：

-   [更改环境布局](https://msdn.microsoft.com/library/ms170069.aspx)  
  
-   [显示查询窗口](https://msdn.microsoft.com/library/ms169823.aspx)  
  
-   [显示“对象资源管理器详细信息”页](https://msdn.microsoft.com/library/ms170076.aspx)  
  
-   [选择键盘快捷键方案](https://msdn.microsoft.com/library/ms167416.aspx)  
  
-   [设置启动选项](https://msdn.microsoft.com/library/ms166556.aspx) 
  
-   [还原默认的 SQL Server Management Studio 配置](https://msdn.microsoft.com/library/ms166550.aspx)  
  
-   [摘要：基本导航](https://msdn.microsoft.com/library/ms170043.aspx)
