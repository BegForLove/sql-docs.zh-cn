# <a name="supportsbatchupdates-method-sqlserverdatabasemetadata"></a>supportsBatchUpdates 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库是否支持批更新。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean supportsBatchUpdates()  
```  
  
## <a name="return-value"></a>返回值  
 如果支持,**则为 true** 。 否则为 **false**。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 supportsBatchUpdates 方法由 supportsBatchUpdates 方法在 Java.sql.databasemetadata 接口中指定。
