# <a name="sql-server-xtp-transaction-log"></a>SQL Server XTP 事务日志
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  SQL Server XTP 事务日志性能对象包含与 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]中的内存中 OLTP 事务日志活动相关的计数器。  
  
 下表说明了 **SQL Server XTP 事务日志** 计数器。  
  
|计数器|说明|  
|-------------|-----------------|  
|**写入的日志字节数/秒**|内存中 OLTP 引擎每秒写入到 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 事务日志的字节数（平均值）。|  
|**写入的日志记录数/秒**|内存中 OLTP 引擎每秒写入到 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 事务日志的记录数（平均值）。|
