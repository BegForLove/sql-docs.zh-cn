# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含 Visual FoxPro ODBC 特定于驱动程序的信息。 有关此功能的一般信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)下的相应主题。  
  
 支持： 完整  
  
 ODBC API 符合性：1 级  
  
 将缓冲区与 SQL 语句中的参数标记关联。 Visual FoxPro ODBC 驱动程序支持*fParamType*参数指定的输入参数。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLBind参数](../../odbc/reference/syntax/sqlbindparameter-function.md)。
