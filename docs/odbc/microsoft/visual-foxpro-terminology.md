# <a name="visual-foxpro-terminology"></a>Visual FoxPro 术语
**database**  
 Visual FoxPro 数据库文件有.dbc 扩展，且可以包含一个或多个**表**。  
  
 **数据库表**  
 在 Visual FoxPro 表这是与数据库相关联。 对比度**免费表**。  
  
 **可用表**  
 在 Visual FoxPro，不是与数据库关联的表。  
  
 创建的.dbf 文件 FoxPro 版本 2.x 是一个免费的表，除非它被转换为 Visual FoxPro 表并添加到 Visual FoxPro 数据库。 对比度**数据库表**。  
  
 **可准备对象的 SQL 语句**  
 尚未处理的 SQL 语句**SQLPrepare**函数。 有关使用 Visual FoxPro ODBC 驱动程序的此函数的详细信息，请参阅[SQLPrepare （Visual FoxPro ODBC 驱动程序）](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)。  
  
 **table**  
 在 Visual FoxPro 记录存储在表中。 表的每一行代表一条记录，并且表中的列表示记录的字段。 每个 Visual FoxPro 表都存储在其自己的文件扩展名为.dbf。 Visual FoxPro 表可以与数据库相关联。  
  
 FoxPro 版本 2。*x*表不与数据库相关联。
