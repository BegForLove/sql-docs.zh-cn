# <a name="configurationsetting-property---virtualdirectoryreportmanager"></a>ConfigurationSetting 属性-VirtualDirectoryReportManager
  返回报表管理器应用程序的虚拟目录。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## <a name="property-values"></a>属性值  
 包含虚拟目录的 **string** 。
