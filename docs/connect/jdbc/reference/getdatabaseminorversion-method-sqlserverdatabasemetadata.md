# <a name="getdatabaseminorversion-method-sqlserverdatabasemetadata"></a>getDatabaseMinorVersion 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索基础数据库的次版本号。  
  
## <a name="syntax"></a>语法  
  
```  
  
public int getDatabaseMinorVersion()  
```  
  
## <a name="return-value"></a>返回值  
 指示数据库次版本的 int  。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 getDatabaseMinorVersion 方法由 java.sql.DatabaseMetaData 接口中的 getDatabaseMinorVersion 方法指定。
