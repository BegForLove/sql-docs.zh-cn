# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>PropertyStrValue 属性（SqlServiceAdvancedProperty 类）
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  获取高级属性的字符串值。  
  
## <a name="syntax"></a>语法  
  
```  
  
object.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *对象*  
 A [SqlServiceAdvancedProperty 类](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md)表示高级的属性的对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定高级属性的值的字符串值。
