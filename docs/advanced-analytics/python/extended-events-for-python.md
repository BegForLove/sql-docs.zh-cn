# <a name="extended-events-for-python"></a>Python 的扩展的事件

本部分正处于开发。

> [!NOTE]
> Python 支持是 SQL Server 自 2017 年中的新功能，并在预发行版。 有关详细信息很快查看。

一般情况下，你可以监视任何外部脚本，包括运行 Python，使用为 SQL Server 2016 中的 R 脚本执行提供的相同 framework。 有关详细信息，请参阅[Extended Events for R Services](../r/extended-events-for-sql-server-r-services.md)。
