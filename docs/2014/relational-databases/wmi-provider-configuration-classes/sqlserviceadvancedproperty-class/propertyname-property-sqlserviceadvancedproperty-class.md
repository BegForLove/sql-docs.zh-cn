# <a name="propertyname-property-sqlserviceadvancedproperty-class"></a>PropertyName 属性（SqlServiceAdvancedProperty 类）
  获取或设置高级属性的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*  
 一个表示高级属性的 [SqlServiceAdvancedProperty 类](sqlserviceadvancedproperty-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定高级属性的名称的字符串值。
