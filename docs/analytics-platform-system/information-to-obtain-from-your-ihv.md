# <a name="information-to-obtain-from-your-ihv"></a>若要从你 IHV 获取的信息
独立硬件供应商 (IHV) 为您提供新的 SQL Server PDW 设备，它们将提供有关设备硬件和配置它们已对你的设备执行。 你将需要此信息来管理你的设备。  
  
以下列表显示从你 IHV 通常需要的信息。 在某些情况下，需要额外或其他信息。 请与你 IHV 若要确保所有的相关信息已转移到你的设备传送。  
  
|||  
|-|-|  
|**信息或文档**|**Description**|  
|物料清单 (BOM)|你的帐单的材料列出你的设备中包括的组件。 此信息并需要确认已发送了所有组件。<br /><br />**重要说明：**你的帐单的材料应包括权重，为每个设备节点和为每个完整的机架。 规划如何处理和移动设备组件时，此信息很重要，用于确保你的数据中心可以容纳设备。 如果你 BOM 不包含节点权重，请确保从你的所有节点 IHV 获取此信息。|  
|缆线关系图|缆线图显示了如何连接网络、 电源和每个设备其他电缆机架。 在你的数据中心中安装设备时，需要这些关系图和你需要删除或替换组件任何时间。|  
|设备严苛的需求|你的设备可以安装在你的数据中心中之前，你需要知道你的数据中心是否符合气流和设备，以及大小的电缆长度要求和组件的电源要求。 另请参阅帐单的物料清单 (BOM) 上面有关设备组件权重，后者本身也是必需。|
