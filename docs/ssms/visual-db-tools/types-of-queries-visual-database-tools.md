# <a name="types-of-queries-visual-database-tools"></a>查询类型 (Visual Database Tools)
查询和视图设计器为某些查询类型（例如“选择”、“插入结果”、“插入值”、“更新”和“生成表”）提供了内置的功能，并且还支持 UNION、全文搜索和 CASE 表达式查询。
