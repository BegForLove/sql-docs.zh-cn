# <a name="enter-data-source-credentials-dialog-box"></a>“输入数据源凭据”对话框
  使用**输入数据源凭据**对话框可以提供你[!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]凭据，以便可以从数据源检索数据。  
  
## <a name="options"></a>“常规”  
 **用户名**  
 键入登录数据源时要使用的用户名。  
  
 **密码**  
 键入登录数据源时要使用的密码。
