# <a name="visual-foxpro-terminology"></a>Visual FoxPro 术语
**database**  
 Visual FoxPro，一个数据库文件扩展名为.dbc，和可以包含一个或多**表**。  
  
 **数据库表**  
 在 Visual FoxPro 表这是与数据库关联。 对比度**可用表**。  
  
 **可用的表**  
 在 Visual FoxPro，不是与数据库关联的表。  
  
 创建的.dbf 文件中 FoxPro 版本 2.x 是可用的表，除非它被转换为 Visual FoxPro 表并添加到 Visual FoxPro 数据库。 对比度**数据库表**。  
  
 **preparable SQL 语句**  
 尚未处理的 SQL 语句**SQLPrepare**函数。 有关此函数带 Visual FoxPro ODBC 驱动程序的详细信息，请参阅[SQLPrepare （Visual FoxPro ODBC 驱动程序）](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)。  
  
 **table**  
 Visual FoxPro，在记录存储在一个表。 表的每一行代表一记录，并且表的列表示记录的字段。 每个 Visual FoxPro 表都存储在其自己的文件扩展名.dbf。 Visual FoxPro 表可以与数据库相关联。  
  
 FoxPro 版本 2。*x*表不与数据库相关联。
