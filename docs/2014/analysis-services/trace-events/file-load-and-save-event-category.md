# <a name="file-load-and-save-event-category"></a>“文件加载和保存事件”类别
  “文件加载和保存”事件类别包含下表说明的各个事件类。  
  
|Event Class|事件 ID|Description|  
|-----------------|--------------|-----------------|  
|文件加载开始|90|文件加载开始|  
|文件加载结束|91|文件加载结束|  
|文件保存开始|92|文件保存开始|  
|文件保存结束|93|文件保存结束|  
|PageOut 开始|94|PageOut 开始|  
|PageOut 结束|95|PageOut 结束|  
|PageIn 开始|96|PageIn 开始|  
|PageIn 结束|97|PageIn 结束|  
  
 有关与每个文件加载和保存事件类关联的列的信息，请参阅 [File Load and Save Data Columns](file-load-and-save-data-columns.md)。
