# <a name="logfilesize-element-assl"></a>LogFileSize 元素 (ASSL)
  指定最大日志文件大小 (MB)。  
  
## <a name="syntax"></a>语法  
  
```xml  
  
<Trace>  
  
   <LogFileSize>...</LogFileSize>  
  
</Trace>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|说明|  
|--------------------|-----------------|  
|数据类型和长度|Integer|  
|默认值|**5** （兆字节为单位）|  
|基数|0-1：可出现一次且仅出现一次的可选元素。|

|------------------|-------------|

|子元素|无|
