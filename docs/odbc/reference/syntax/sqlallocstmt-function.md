# <a name="sqlallocstmt-function"></a>SQLAllocStmt 函数
**一致性**  
 版本介绍： ODBC 1.0 标准合规性： 已弃用  
  
 **摘要**  
 在 ODBC 3 中。*x*， ODBC 2.*x*函数**SQLAllocStmt**已被**SQLAllocHandle**替换。 有关详细信息，请参阅[SQLAllocHandle 函数](../../../odbc/reference/syntax/sqlallochandle-function.md)。  
  
> [!NOTE]  
>  有关驱动程序管理器将此功能映射到 ODBC 2 时的详细信息。*x*应用程序使用 ODBC 3。*x*驱动程序，请参阅附录 G 中的[映射已弃用函数](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)：向后兼容性的驱动程序指南。
