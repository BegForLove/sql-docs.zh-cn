# <a name="numeric-functions-visual-foxpro-odbc-driver"></a>数字函数（Visual FoxPro ODBC 驱动程序）
下表描述了 Visual FoxPro ODBC 驱动程序支持的 ODBC 数值函数;当相同函数的 Visual FoxPro 语法与 ODBC 语法不同时，将列出 Visual FoxPro 等效项。  
  
|ODBC 语法|可视化 FoxPro 语法|  
|------------------|---------------------------|  
|ABS *（numeric_exp）*||  
|ACOS *（float_exp）*||  
|阿辛 *（float_exp）*||  
|阿坦 *（float_exp）*||  
|ATAN2 *（float_exp1，float_exp2）*|ATN2 （*float_exp1， float_exp2*）|  
|天花板 *（numeric_exp）*||  
|COS *（float_exp）*||  
|婴儿床 *（float_exp）*||  
|度 *（numeric_exp）*|RTOD *（numeric_exp）*|  
|EXP *（float_exp）*||  
|地板 *（numeric_exp）*||  
|日志 *（float_exp）*||  
|日志*10（float_exp）*||  
|*MOD（integer_exp1、integer_exp2）*||  
|PI *（ ）*||  
|拉迪安*斯 （numeric_exp）*|DTOR *（numeric_exp）*|  
|兰德 *（[integer_exp]）*||  
|圆 *（numeric_exp，integer_exp）*||  
|签名 *（numeric_exp）*||  
|新 *（float_exp）*||  
|SQRT *（float_exp）*||  
|褐*色 （float_exp）*||  
  
 不支持以下数字函数：  
  
 电源 *（numeric_exp、integer_exp）*  
  
 截流 *（numeric_exp，integer_exp）*
