# <a name="reserved-keyword-limitations"></a>保留关键字限制

避免将任何 ODBC 保留的关键字用作 SQL 表或相关对象中的标识符。 如果出现奇数情况，则必须使用保留关键字作为标识符，则必须在标识符周围加上一对*回记*（'）。 *回勾*的另一个名称是*回引号*。

保留关键字限制也适用于保留关键字的任何速记形式。

ODBC 保留关键字的列表可在以下位置查阅：

- [ODBC 保留关键字](https://docs.microsoft.com/sql/odbc/reference/appendixes/reserved-keywords)。

- 在*ODBC 程序员参考指南*中，请参阅[附录 C：SQL 语法](https://docs.microsoft.com/sql/odbc/reference/appendixes/appendix-c-sql-grammar)。
