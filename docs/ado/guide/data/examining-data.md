# <a name="examining-data"></a>检查数据
[获取数据](../../../ado/guide/data/getting-data.md)部分介绍了如何从数据源检索数据，用作一个或多**记录集**对象。 本部分将讨论**记录集**中更多详细信息，包括如何浏览**记录集**和查看其数据。  
  
 **记录集**支持各种[属性和方法](../../../ado/reference/ado-api/recordset-object-properties-methods-and-events.md)的可导航并检查中包含的数据。 这些功能都依赖于提供程序的。 某些属性或方法，可能不支持某些访问接口。
