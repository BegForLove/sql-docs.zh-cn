# <a name="isqlservererrorinfo-ole-db"></a>ISQLServerErrorInfo (OLE DB)
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client OLE DB 提供程序定义**ISQLServerErrorInfo**错误接口。 此接口返回详细信息从[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]错误，包括其严重性和状态。  
  
## <a name="in-this-section"></a>本节内容

|------------|-----------------|  
|[ISQLServerErrorInfo::GetErrorInfo &#40;OLE DB&#41;](../../relational-databases/native-client-ole-db-interfaces/isqlservererrorinfo-geterrorinfo-ole-db.md)|返回一个指向[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]Native Client OLE DB 提供程序包含的 SSERRORINFO 结构[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]错误详细信息。|
