# <a name="lastsibling-mdx"></a>LastSibling (MDX)


  返回指定成员的父成员的最后一个子成员。  
  
## <a name="syntax"></a>语法  
  
```  
  
Member_Expression.LastSibling   
```  
  
## <a name="arguments"></a>参数  
 *Member_Expression*  
 返回成员的有效多维表达式 (MDX)。  
  
### <a name="example"></a>示例  
 下面的示例返回 2002 年 7 月最后一天的默认度量值。  
  
```  
SELECT [Date].[Fiscal].[Date].&[20020717].LastSibling   
   ON 0  
FROM [Adventure Works]  
```
