# <a name="installing-the-software-odbc"></a>安装软件 (ODBC)
> [!IMPORTANT]  
>  将 Windows 的未来版本中删除该功能。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 用于 Oracle 的 ODBC 驱动程序是数据访问组件之一。 它通常会显示其他 ODBC 组件，如 ODBC 数据源管理器中，并应已安装。 该驱动程序还可以找到在"驱动程序和其他下载"www.microsoft.com 在 Microsoft 产品支持服务联机 Web 站点上。  
  
 必须根据自己的文档中安装网络软件。 用于 Oracle 的 ODBC 驱动程序需要任何特殊安装注意事项，只要支持网络软件。  
  
 根据其自己的文档，必须安装 oracle 软件。 用于 Oracle 的 ODBC 驱动程序通常需要任何特殊安装注意事项，只要该驱动程序支持的版本。 但是，若要使产品保持兼容，安装适用于上一次，以确保您拥有最新版本的驱动程序的 Oracle ODBC 驱动程序。 Oracle 维护一个公共的 FTP 站点，其中发，除了别的之外对 Oracle 服务器产品和附带的服务器产品的客户端组件的修补程序。 这些修补程序所需的多个 Microsoft 产品和技术的正确运行。 有关此站点的详细信息，请参阅[Oracle 软件修补程序](../../odbc/microsoft/oracle-software-patches.md)。  
  
> [!CAUTION]  
>  通过 MDAC/Windows DAC 安装 Oracle 软件可能会覆盖当前版本的 MDAC。 如果使用 ODBC 组件出现问题，请重新安装 MDAC。
