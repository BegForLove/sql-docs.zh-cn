# <a name="delete-columns-dialog-box"></a>“删除列”对话框
  使用 **“删除列”** 对话框可以删除表、矩阵或列表数据区域中的列。 对于与组关联的列，您还可以删除组定义。  
  
## <a name="options"></a>“常规”  
 **删除列和相关的组**  
 选择此选项可以删除列和关联的组定义。  
  
 **仅删除列**  
 选择此选项可以仅删除选定的列。
