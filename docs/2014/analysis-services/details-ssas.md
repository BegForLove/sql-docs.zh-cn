# <a name="details-ssas"></a>详细信息 (SSAS)
  **“表导入向导”** 的这一页可用于查看与成功或失败的数据导入操作有关的信息。 若要从 [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]访问该向导，请在 **“模型”** 菜单上，单击 **“从数据源导入”**。
