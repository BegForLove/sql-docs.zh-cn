# <a name="modifying-a-visual-foxpro-data-source"></a>修改 Visual FoxPro 数据源
### <a name="to-modify-a-visual-foxpro-data-source"></a>若要修改 Visual FoxPro 数据源  
  
1.  在运行 Windows 2000 的计算机，打开 Windows 控制面板，并双击管理工具。  
  
2.  双击数据源 (ODBC) 以打开 ODBC 数据源管理器对话框。Visual FoxPro ODBC 驱动程序或任何 ODBC 驱动程序软件已安装之后，此图标才可用。  
  
    > [!NOTE]  
    >  如果正在运行以前版本的 Windows，打开 Windows 控制面板，并双击 32 位 ODBC 或 ODBC 以打开 ODBC 数据源管理器对话框。  
  
3.  在用户 DSN 或系统 DSN 选项卡，选择你想要修改并单击配置数据源的名称。  
  
4.  在[ODBC Visual FoxPro 安装程序对话框中](../../odbc/microsoft/odbc-visual-foxpro-setup-dialog-box.md)、 选择和更改你想要修改，然后单击确定的项。  
  
5.  单击确定以保存更改并关闭 ODBC 数据源管理器对话框。  
  
 所做的更改将从你的应用程序访问数据源的下一步时间生效。
