# <a name="recordtype-property-ado"></a>RecordType 属性 (ADO)
指示[记录](../../../ado/reference/ado-api/record-object-ado.md)对象的类型。  
  
## <a name="return-value"></a>返回值  
 返回[RecordTypeEnum](../../../ado/reference/ado-api/recordtypeenum.md)值。  
  
## <a name="remarks"></a>备注  
 **RecordType**属性是只读的。
