# 如何创建 SQL Server 更改数据库实例
  本过程介绍如何使用 CDC 设计器控制台来创建 CDC 实例。  
  
### 创建 CDC 实例  
  
1.  从 **“开始”** 菜单上，选择 **“CDC 设计器控制台”**。  
  
2.  在左侧的窗格中，展开 **“变更数据捕获”**。  
  
3.  选择您要在其上创建新的 CDC 实例的服务。  
  
4.  从 CDC 设计器控制台右侧的 **“操作”** 窗格中，选择 **“新建 Oracle CDC 实例”**。  
  
     还可右键单击要创建新的 CDC 实例的服务，然后选择“新建 Oracle CDC 实例”。  
  
5.  在新建实例向导中输入创建新实例所需的信息。 有关此向导所需信息的信息，请参阅 [Use the New Instance Wizard](../../integration-services/change-data-capture/use-the-new-instance-wizard.md)。
