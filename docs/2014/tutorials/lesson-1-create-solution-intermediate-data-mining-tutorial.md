# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>第 1 课：创建中级数据挖掘解决方案（数据挖掘中级教程）
  在数据挖掘基础教程中，你创建[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]包含简单数据挖掘解决方案的项目基于新[!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]数据库。  
  
 对于本教程，你将创建一个单独[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]项目通过使用[!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]。 你将创建[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]使用的数据源[!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]，并将多个新的数据源视图添加到该数据源，以支持这些方案和模型类型。  
  
 本课包含以下任务：  
  
-   [创建解决方案和数据源&#40;中间数据挖掘教程&#41;](../../2014/tutorials/creating-a-solution-and-data-source-intermediate-data-mining-tutorial.md)
