# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>书签支持（Visual FoxPro ODBC 驱动程序）
Visual FoxPro ODBC 驱动程序支持简单的书签。 当您调用[SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md)与 SQL_BOOKMARK_PERSISTENCE*信息类型*，返回值是 SQL_BP_SCROLL。  
  
 有关书签的详细信息，请参阅[书签 (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md)。
