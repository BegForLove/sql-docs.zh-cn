# <a name="syssynonyms-transact-sql"></a>sys.synonyms (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-all-md](../../includes/tsql-appliesto-ss2008-all-md.md)]

  每个同义词对象存在对应的一行**sys.objects.type** = SN。  
  
|列名|数据类型|描述|  
|-----------------|---------------|-----------------|  
|**\<从 sys.objects 继承的列 >**||此视图所继承的列的列表，请参阅[sys.objects &#40;TRANSACT-SQL&#41;](../../relational-databases/system-catalog-views/sys-objects-transact-sql.md)。|  
|**base_object_name**|**nvarchar(1035)**|将此同义词的用户重定向到的对象的完全引起来的名称。|
