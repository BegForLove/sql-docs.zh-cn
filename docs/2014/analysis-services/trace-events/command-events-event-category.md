# <a name="command-events-event-category"></a>“命令事件”事件类别
  命令事件事件类别具有下表中说明的事件类。  
  
|Event Class|事件 ID|Description|  
|-----------------|--------------|-----------------|  
|命令开始|15|收集跟踪开始后的所有命令开始事件。|  
|命令结束|16|收集跟踪开始后的所有命令结束事件。|  
  
 有关与每个命令事件事件类相关联的列的信息，请参阅 [Command Events Data Columns](command-events-data-columns.md)。
