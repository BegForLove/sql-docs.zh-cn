# <a name="supportsouterjoins-method-sqlserverdatabasemetadata"></a>supportsOuterJoins 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此数据库是否支持某种形式的外部联接。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean supportsOuterJoins()  
```  
  
## <a name="return-value"></a>返回值  
 **true**如果受支持。 否则为 **false**。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 supportsOuterJoins 方法由 java.sql.DatabaseMetaData 接口中的 supportsOuterJoins 方法指定。
