# <a name="completing-the-wizard-cube-wizard"></a>完成向导（多维数据集向导）
  可以使用 **“完成向导”** 页命名和预览将在完成向导后创建的多维数据集。  
  
## <a name="options"></a>“常规”  
 **多维数据集名称**  
 键入多维数据集的名称。  
  
 **预览**  
 显示将要在完成向导后创建的多维数据集的结构。
