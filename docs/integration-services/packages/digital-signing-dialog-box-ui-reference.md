# “数字签名”对话框 UI 参考
  使用 **“数字签名”** 对话框可以使用数字签名对包进行签名或删除签名。 在 **中，** SSIS **菜单的** “数字签名” **选项中提供了** “数字签名” [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]对话框。  
  
 有关详细信息，请参阅[使用数字签名标识包的源](../../integration-services/packages/identify-the-source-of-packages-with-digital-signatures.md)和[使用数字证书对包签名](../../integration-services/packages/sign-a-package-by-using-a-digital-certificate.md)。  
  
## 选项  
 **签名**  
 单击此项将打开“选择证书”对话框，在其中可选择要使用的证书。  
  
 **删除**  
 单击此项将删除数字签名。
