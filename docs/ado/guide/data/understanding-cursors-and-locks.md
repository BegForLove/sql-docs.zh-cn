# <a name="understanding-cursors-and-locks"></a>了解游标和锁定
务必要了解游标的工作方式，以便为应用程序的数据访问要求选择最佳和最有效的游标类型。 小于最佳的游标配置可以使数据访问操作不厌其烦缓慢。  
  
 ADO**记录集**对象的很多功能由游标的类型和位置以及锁类型决定。
