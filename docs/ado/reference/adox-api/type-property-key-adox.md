# <a name="type-property-key-adox"></a>类型属性 （密钥） (ADOX)
指示键的类型。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回**长**值可以是之一[KeyTypeEnum](../../../ado/reference/adox-api/keytypeenum.md)常量。 默认值是**adKeyPrimary**。  
  
## <a name="remarks"></a>Remarks  
 此属性为只读上[密钥](../../../ado/reference/adox-api/key-object-adox.md)已追加到集合的对象。
