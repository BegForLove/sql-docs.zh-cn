|||  
|-|-|  
|产品名称|SQL Server|

|事件源|MSSQLSERVER|

|符号名称||  
|消息正文|订阅服务器“%s”对发布“%s”中项目“%s”的订阅已通过数据验证。|  
  
## <a name="explanation"></a>解释  
 根据发布服务器上的数据对订阅服务器上的数据进行验证，并且数据匹配；因此验证通过。 有关验证的详细信息，请参阅 [Validate Replicated Data](../../relational-databases/replication/validate-replicated-data.md)。
