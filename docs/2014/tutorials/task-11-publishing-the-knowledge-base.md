# <a name="task-11-publishing-the-knowledge-base"></a>任务 11：发布知识库
  在本任务中，您将发布知识库。 已发布的知识库可用于数据质量项目中的清理或匹配活动。  
  
1.  单击**完成**在窗口底部的按钮。  
  
2.  单击**发布**中**SQL Server Data Quality Services**对话框。  
  
3.  单击“确定”  关闭消息框。
