# <a name="servername-property-sqlserveralias-class"></a>ServerName 属性（SqlServerAlias 类）

## <a name="syntax"></a>语法  
  
```  
  
object  
.ServerName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象
