# <a name="schema-text-file-driver"></a>架构 （文本文件驱动程序）
架构包含有关每个表 （文本文件） 的信息包含在数据源，包括表的格式，要扫描以确定列类型是否表的第一行包含列名称，是否使用 OEM 编写的源代码文件的行数或ANSI 代码页和每个列的名称、 数据类型和宽度。
