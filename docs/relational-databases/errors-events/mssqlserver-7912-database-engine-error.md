|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|7912|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|DBCC2_REPAIR_EXTENT_ALLOCATED|
