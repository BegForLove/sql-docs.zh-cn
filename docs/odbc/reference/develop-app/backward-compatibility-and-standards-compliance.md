# <a name="backward-compatibility-and-standards-compliance"></a>后向兼容性和标准符合性
向后兼容性是较新的 ODBC 组件使用旧 ODBC 组件的能力。 以下各节讨论这些组件如何受到 ODBC *3.x*中的更改的影响。 其中包含的信息主要涉及 ODBC *3.x*应用程序的编写以及 ODBC 驱动程序如何处理向后兼容性问题。 有关向后兼容性问题如何影响 ODBC *3.x*驱动程序编写的具体指南，请参阅附录[G：向后兼容性的驱动程序指南](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md)。
