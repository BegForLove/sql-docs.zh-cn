# <a name="supported-data-types-visual-foxpro-odbc-driver"></a>支持的数据类型（Visual FoxPro ODBC 驱动程序）
通过 ODBC API 和 Microsoft 查询中显示驱动程序支持的数据类型的列表。  
  
## <a name="data-types-in-c-applications"></a>C 语言应用程序中的数据类型  
 您可以获得通过使用 Visual FoxPro ODBC 驱动程序支持的数据类型的列表[SQLGetTypeInfo](../../odbc/microsoft/sqlgettypeinfo-visual-foxpro-odbc-driver.md)中 C 函数或C++应用程序。  
  
## <a name="data-types-in-applications-using-microsoft-query"></a>使用 Microsoft 查询的应用程序中的数据类型  
 如果应用程序使用 Microsoft Query Visual FoxPro 数据源上创建一个新表，将显示 Microsoft Query**新的表定义**对话框。 下**字段说明**，则**类型**框列表[Visual FoxPro 字段数据类型](../../odbc/microsoft/visual-foxpro-field-data-types.md)、 单个字符来表示。
