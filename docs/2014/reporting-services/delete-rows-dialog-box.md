# <a name="delete-rows-dialog-box"></a>“删除行”对话框
  使用 **“删除行”** 对话框可以删除表、矩阵或列表数据区域中的行。 对于与组关联的行，您还可以删除组定义。  
  
## <a name="options"></a>“常规”  
 **删除行和关联的组**  
 选择此选项可删除行和关联的组定义。  
  
 **仅删除行**  
 选择此选项可以仅删除选定的行。
