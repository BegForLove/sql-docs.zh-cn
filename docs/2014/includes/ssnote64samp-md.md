Itanium 不支持 SQL Server Data Tools\-基于计算机。 但是，对 SQL Server Data Tools 是适用于 x64 支持\-基于计算机。 如果已在 Itanium 上部署 SQL Server 示例数据库\-基于的计算机，在 x86 上使用 BI Development Studio\-基于或 x64\-基于计算机来修改和运行这些示例。
