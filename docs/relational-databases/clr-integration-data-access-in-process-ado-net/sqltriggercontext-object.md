# <a name="sqltriggercontext-object"></a>SqlTriggerContext Object
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
**SqlTriggerContext**类提供了有关触发器的上下文信息。 此上下文的信息包括导致触发器激发，修改的列在更新操作中，并且，对于数据定义语言 (DDL) 触发器的 XML 的操作的类型**EventData**结构的描述触发操作。 有关详细信息和如何使用的示例**SqlTriggerContext**类，请参阅[CLR 触发器](http://msdn.microsoft.com/library/302a4e4a-3172-42b6-9cc0-4a971ab49c1c)。  
  
 有关详细信息，请参阅**Microsoft.SqlServer.Server.SqlTriggerContext** .NET Framework SDK 文档中的类参考文档。
