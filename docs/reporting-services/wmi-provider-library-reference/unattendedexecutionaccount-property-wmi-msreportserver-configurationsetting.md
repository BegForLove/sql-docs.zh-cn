# UnattendedExecutionAccount 属性 (WMI MSReportServer_ConfigurationSetting)
  返回以无人参与方式运行报表时报表服务器将模拟的用户帐户。 只读。  
  
## 语法  
  
```vb  
Public Dim UnattendedExecutionAccount As String  
```  
  
```csharp  
public string UnattendedExecutionAccount;  
```  
  
## 属性值  
 表示帐户名的 **String** 对象。
