# MSSQL_REPL-2147200940
    
## 消息详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|-2147200940|  
|事件源|MSSQLServer|  
|符号名称||  
|消息正文|发布服务器上的架构（版本：%2!d!， guid：“%1”）与订阅服务器上的架构（版本：%4!d!， guid：“%3”）不匹配。 在发布服务器已从某个备份中还原时，可能发生此情况。 在此情况下，重新创建初始快照并重新初始化所有订阅。|  
  
## 解释  
 发布服务器上的架构与订阅服务器上的架构不匹配。  
  
## 用户操作  
 重新创建初始快照并重新初始化所有订阅。
