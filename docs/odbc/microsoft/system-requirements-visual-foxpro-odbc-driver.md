# <a name="system-requirements-visual-foxpro-odbc-driver"></a>系统要求（Visual FoxPro ODBC 驱动程序）
安装的系统要求提供了成功安装驱动程序所需的最小操作系统和磁盘空间。 安装驱动程序后，您可以选择要用于访问 Visual FoxPro 数据的特定应用程序软件。  
  
## <a name="installation-requirements"></a>安装要求  
 有关安装要求和其他信息，请参阅[Visual FoxPro ODBC 驱动程序下载网站](https://go.microsoft.com/fwlink/?LinkId=121318)。  
  
## <a name="accessing-visual-foxpro-data"></a>访问可视化 FoxPro 数据  
 访问微软视觉福克斯专业或福克斯Pro 2。*x*数据，您必须具有以下内容：  
  
-   ODBC客户端软件（自动与驱动程序一起安装）  
  
-   微软可视化福克斯Pro ODBC驱动程序  
  
-   以下任何类型的应用软件：  
  
    -   微软办公室应用程序，如微软Excel或微软Word  
  
    -   C 或C++语言 ODBC 应用程序  
  
    -   微软视觉基本ODBC应用程序  
  
-   以下任何类型数据：  
  
    -   可视化 FoxPro[数据库](../../odbc/microsoft/visual-foxpro-terminology.md)或[可用表](../../odbc/microsoft/visual-foxpro-terminology.md)目录  
  
    -   FoxPro 2.0， 2.5， 2.6[表](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Visual FoxPro ODBC 驱动程序支持双字节字符集 （DBCS）。 有关详细信息，请参阅[国际支持](../../odbc/microsoft/international-support-visual-foxpro-odbc-driver.md)。  
  
 该驱动程序不支持 16 位 Microsoft Windows 3.1 应用程序。
