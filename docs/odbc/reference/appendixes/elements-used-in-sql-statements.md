# <a name="elements-used-in-sql-statements"></a>SQL 语句中使用的元素
前面列出的 SQL 语句中使用了以下元素。  
  
## <a name="element"></a>元素  
 *基表标识符*：：**用户定义名称*  
  
 *基表名称*：：**基表标识符*  
  
 *布尔因子*：：*[NOT]*布尔原*  
  
 *布尔 -主*：：* 比较 *- 谓词*&#124; （*搜索条件*）  
  
 *布尔期*：：**布尔因子*[和*布尔术语*]  
  
 *字符字符串文本*：：* "**字符*=..." （*字符*是驱动程序/数据源的字符集中的任何字符。 要在字符字符串文本中包含单个文本引号字符 （''），请使用两个文本引号字符 [''']*  
  
 *列标识符*：：**用户定义的名称*  
  
 *列名*：：* =*表名*.*列标识符*  
  
 *比较运算符*：：* \< < &#124; > &#124; = &#124; >= &#124; = &#124; <>  
  
 *比较谓词*：：**表达式*比较运算符表达式  
  
 *数据类型*：：：**字符字符串类型*（*字符字符串类型*是 SQLGetTypeInfo 返回的结果集中的"DATA_TYPE"列SQL_CHAR或SQL_VARCHAR的任何数据类型。  
  
 *数字*：：*0 &#124; 1 &#124; 2 &#124; 3 &#124; 4 &#124; 5 &#124; 6 &#124; 7 &#124; 8 &#124; 9  
  
 *动态参数*：：* ？

 *因子*：：**+* *-*[&#124;]*主*

 *字母*：：**小写字母&#124;大写字母*

 *小写字母*：：：* &#124; b &#124; c &#124; d &#124; &#124; f &#124; h &#124; h &#124; i &#124; &#124; k &#124; &#124; &#124; &#124; &#124; p &#124; &#124; r &#124; s &#124; &#124; u &#124; v &#124; &#124; x &#124; &#124; y z

 *表标识符*：：**用户定义名称*  
  
 *表名称*：：**表标识符*
