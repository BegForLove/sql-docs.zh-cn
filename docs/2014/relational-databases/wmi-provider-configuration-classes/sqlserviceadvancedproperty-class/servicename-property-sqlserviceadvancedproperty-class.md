# <a name="servicename-property-sqlserviceadvancedproperty-class"></a>ServiceName 属性（SqlServiceAdvancedProperty 类）
  获取与高级属性关联的托管服务的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.ServiceName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示高级属性的 [SqlServiceAdvancedProperty 类](sqlserviceadvancedproperty-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定与高级属性关联的服务名称的字符串值。
