# <a name="outliers-sql-server-data-mining-add-ins"></a>离群值（SQL Server 数据挖掘外接程序）
  ![数据挖掘功能区中的离群值向导](media/dmc-outliers.gif "数据挖掘功能区中的离群值向导")  
  
 *离群值*意味着一个数据值，会产生问题对任何由于以下原因之一：  
  
-   值超出预期的范围。  
  
-   数据输入可能有误。  
  
-   缺少值。  
  
-   数据包含空格或其他 Null 字符串。  
  
-   值是准确的，但与分布范围相距甚远，以致无法显著影响模型。  
  
 Excel 数据挖掘客户端能帮助您检测这种数据，然后更新值或取消值。 例如，可将离群值替换为算术平均值，也可删除可能有错的值所在的行。  
  
## <a name="handling-outliers"></a>处理离群值  
 **删除离群值**向导为你提供了几个工具来相应地处理离群值：  
  
-   首先，可以浏览数据以便更好了解值的分布以及离群值与其他数据之间的关系。  
  
     例如，你可以使用**浏览数据**用于查看和修复这些值任务。 **删除离群值**向导还显示一个图形，一行或条形图，以帮助你了解的所有值的分布。  
  
-   接下来，你可以使用**离群值**向导，以删除或更改离群值。 使用的方法取决于值是离散的还是连续的。  
  
     该向导在条形图中显示离散值，其中的每个条都表示一个特定的值，条的高度指示每个值的事例数。 对于表示远离分布中心的值组或可能错误的值组的条，可通过滑动条形图上的阈值控件来切断这些条。  
  
-   该向导在条形图或折线图上显示连续值。 在折线图中，值在 X 轴上表示，值的计数在 Y 轴上表示。  
  
     你可以控制是删除还是保留在图表的下限和上限端点处的值，通过更改**最小**和**最大**值或滑动条。 更改最小值和最大值设置时，要被取消的数据在图表中以阴影显示。  
  
 选择了要处理的离群值之后，可以设置向导如何处理这些离群值。 您可以删除包含离群值的行，或者指定替换值，例如平均值、Null 或其他选择值。  
  
 最后，向导为您提供多种显示新数据的方式。 您可以将原始数据替换为新值，在表中添加包含新值的新列，或者创建包含更新数据的新工作表。  
  
### <a name="using-the-outlier-wizard"></a>使用离群值向导  
  
1.  在**数据挖掘**功能区上，单击**清理数据**，然后选择**离群值**。  
  
2.  在**选择源数据**对话框中，选择 Excel 数据表或单元格区域，然后单击**下一步**。  
  
    > [!WARNING]  
    >  不能使用**离群值**向导对外部数据，除非你将其复制到 Excel 第一次。  
  
3.  在**Select 列**对话框中，选择**单个**列。  
  
     单击“下一步” 。  
  
4.  在**指定阈值**对话框框中，查看数据的分布。  
  
    -   如果该列含有离散值，则向导显示一个直方图，其中含有每个离散值的数量。  
  
         假设离群值是极少数值，你可以筛选出通过更改**最小**值。  
  
    -   如果列包含数值数据，你可以单击**视图作为离散**按钮或**查看为 Numeric**按钮以查看在条形图或折线图中的值之间切换。  
  
5.  在**指定阈值**对话框框中，选择所需通过键入最小和最大值，或通过拖动滑动条来跟踪的数据的范围。 单击“下一步” 。  
  
6.  在**离群值处理**对话框框中，指定是否希望这些值后，删除或替换，然后单击**下一步**。  
  
7.  在**选择目标**对话框框中，指定想要保存的新数据。  
  
### <a name="related-options"></a>相关的选项  
 该向导提供以下选项：  
  
|**选项**|**注释**|  
|-----------------|-----------------|  
|**选择列**|一次只能处理一列。|  
|**指定处理的阈值**|设置阈值使用**最小**若要排除在更少的行，比阈值的值中找到的值。<br /><br /> 最初中的值**最小**等于的值与最少的行，并且无法进行最小值低于此值。|  
|**离群值处理**|如果决定删除离群值，则可更改当前工作表中的数据，也可在新工作表中创建数据的副本。|
