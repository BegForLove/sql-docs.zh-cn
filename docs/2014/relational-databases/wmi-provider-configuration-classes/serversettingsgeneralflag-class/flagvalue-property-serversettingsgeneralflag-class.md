# <a name="flagvalue-property-serversettingsgeneralflag-class"></a>FlagValue 属性（ServerSettingsGeneralFlag 类）

```  
  
object  
.FlagValue [= value]  
```  
  
## <a name="parts"></a>组成部分  
 *object*
