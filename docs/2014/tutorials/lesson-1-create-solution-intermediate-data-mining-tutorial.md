# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>第 1 课：创建中级数据挖掘解决方案（数据挖掘中级教程）
  在数据挖掘基础教程中，创建[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]项目，其中包含的简单数据挖掘解决方案基于新[!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]数据库。  
  
 本教程中，您将创建一个单独[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]通过使用项目[!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]。 您将创建[!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)]使用数据源的[!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]，并将几个新的数据源视图添加到该数据源，以便支持方案和模型类型。  
  
 本课包含以下任务：  
  
-   [创建解决方案和数据源&#40;数据挖掘中级教程&#41;](../../2014/tutorials/creating-a-solution-and-data-source-intermediate-data-mining-tutorial.md)
