# <a name="define-query-parameters-dialog-box-report-builder"></a>“定义查询参数”对话框（报表生成器）
  当您在设计阶段在查询设计器中运行查询以查看示例数据时，可以使用 **“定义查询参数”** 对话框指定参数值。 查询参数对数据源的数据进行筛选，并帮助限制报表处理的数据量。 这些值不会保存到报表定义中。  
  
## <a name="options"></a>“常规”  
 **参数名称**  
 键入要使用的查询参数的名称。  
  
 **参数值**  
 键入要用于查询参数的值。
