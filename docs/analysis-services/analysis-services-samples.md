## <a name="sample-solutions-and-databases"></a>示例解决方案和数据库  

在不再受正式支持或更新，Adventure Works 会保持于有关学习和测试 Analysis Services 的最非独占和可靠示例数据集之一。 Analysis Services 示例项目和数据库，以及文档、 博客文章和演示文稿中的示例使用 Adventure Works 2014 示例数据。

[下载 Adventure Works 示例项目和 Codeplex 上的数据库](https://msftdbprodsamples.codeplex.com/releases/view/125550)。

SQL Server 2016 引入了示例数据，Wide World Importers，一个新集合。 Wide World Importers 示例数据可用于了解和测试 Analysis Services;但是，提供任何教程、 示例或文档。 Wide World importers 计划数据集和架构不支持的许多功能在 Analysis Services 中。 若要了解详细信息，请参阅[Wide World Importers](https://msdn.microsoft.com/library/mt734199)。
