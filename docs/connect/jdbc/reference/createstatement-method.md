# <a name="createstatement-method-"></a>createStatement 方法 ()
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  创建[SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md)发送到数据库的 SQL 语句的对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
public java.sql.Statement createStatement()  
```  
  
## <a name="return-value"></a>返回值  
 语句对象中。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.Connection 接口中的 createStatement 方法指定此 createStatement 方法。
