# <a name="facet-properties-dialog-box-dependent-conditions-page"></a>“方面属性”对话框，“依赖条件”页
  此对话框用于显示当前引用此基于策略的管理方面的条件的列表。  
  
## <a name="options"></a>选项  
 **名称**  
 列出每个引用当前方面的条件的名称。
