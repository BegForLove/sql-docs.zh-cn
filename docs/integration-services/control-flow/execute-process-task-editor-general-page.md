# <a name="execute-process-task-editor-general-page"></a>执行进程任务编辑器（“常规”页）
  可以使用“执行进程任务编辑器”对话框的“常规”页，对执行进程任务进行命名和说明。  
  
 若要了解此任务，请参阅 [Execute Process Task](../../integration-services/control-flow/execute-process-task.md)。  
  
## <a name="options"></a>选项  
 **名称**  
 为执行进程任务提供唯一的名称。 此名称用作任务图标中的标签。  
  
> [!NOTE]  
>  任务名称在一个包内必须是唯一的。  
  
 **Description**  
 键入对执行进程任务的说明。
