# <a name="depth-property-ado-md"></a>深度属性 (ADO MD)
指示之间级别数[级别](../../../ado/reference/ado-md-api/level-object-ado-md.md)和层次结构的根。  
  
## <a name="return-values"></a>返回值  
 返回一个整数，并且是只读的。  
  
## <a name="remarks"></a>注释  
 A**级别**层次结构的根时有**深度**值为零 (0)。
