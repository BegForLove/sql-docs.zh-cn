# <a name="getlogwriter-method-sqlserverdatasource"></a>getLogWriter 方法 (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  此方法仅供内部使用。 有关日志记录的详细信息，请参阅[Tracing Driver Operation](../../../connect/jdbc/tracing-driver-operation.md)。  
  
## <a name="syntax"></a>语法  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>返回值  
 一个 PrintWriter 对象。  
  
## <a name="remarks"></a>Remarks  
 此 getLogWriter 方法由 javax.sql.DataSource 接口中的 getLogWriter 方法指定。
