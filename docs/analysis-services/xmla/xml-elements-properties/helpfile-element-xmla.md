# <a name="helpfile-element-xmla"></a>HelpFile 元素 (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]
  包含到帮助文件或主题描述父路径或 URL[错误](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)元素。  
  
## <a name="syntax"></a>语法  
  
```xml  
  
<Error>  
   ...  
   <HelpFile>...</HelpFile>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|Description|  
|--------------------|-----------------|  
|数据类型和长度|String|  
|默认值|InclusionThresholdSetting|  
|基数|0-1：可出现一次且仅出现一次的可选元素。|  
  
## <a name="element-relationships"></a>元素关系  
  
|关系|元素|  
|------------------|-------------|  
|父元素|[错误](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)|  
|子元素|InclusionThresholdSetting|
