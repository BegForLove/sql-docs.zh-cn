# 发送邮件任务编辑器（“常规”页）
  可以使用 **“发送邮件任务编辑器”** 对话框的 **“常规”** 页，对发送邮件任务进行命名和说明。  
  
 若要了解此任务，请参阅 [Send Mail Task](../../integration-services/control-flow/send-mail-task.md)。  
  
## 选项  
 **名称**  
 为发送邮件任务提供唯一的名称。 此名称用作任务图标中的标签。  
  
 **注意** 任务名称在包中必须是唯一的。  
  
 **Description**  
 键入发送邮件任务的说明。
