# <a name="options-sql-server-object-explorer---commands"></a>选项（“SQL Server 对象资源管理器”-“命令”）
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
在对象资源管理器中右键单击表或视图时，将显示以下两个用于显示表中的行的选项：  
  
-   **Select Top 1000 Rows**  
  
    在查询编辑器中创建一个选择前 1000 行的脚本。  
  
-   **Edit Top 200 Rows**  
  
    在 Visual Database Tools 中打开填充了 200 行数据的表。  
  
可使用“常规脚本选项”  对话框更改要打开的行数。  
  
## <a name="general-scripting-options"></a>常规脚本选项  
**“选择前 <n> 条审核记录”命令的值**  
指定要返回的审核行的数量。 默认值为 1000 行。  
  
**“编辑前 <n> 行”命令的值**  
指定在编辑表中的行时要返回的行数。 默认值为 200 行。  
  
**“选择前 <n> 行”命令的值**  
指定在创建的查询编辑器脚本中指定的要返回的行数。 默认值为 1000 行。
