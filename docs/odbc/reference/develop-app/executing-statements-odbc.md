# <a name="executing-statements-odbc"></a>执行语句 ODBC
ODBC 应用程序通过执行 SQL 语句来执行几乎所有数据库访问权限。 一般事件顺序是分配语句句柄、 设置任何语句属性、 执行语句，检索任何结果，并释放语句句柄。
