如果 Linux 计算机不能访问之前部分中使用的在线存储库，则可以直接下载包文件。 这些包位于 Microsoft 存储库， [ https://packages.microsoft.com ](https://packages.microsoft.com)。

> [!TIP]
> 如果通过之前部分中的步骤成功安装了包，则无需下载或者手动安装以下包。 这仅适用于脱机的情况下。
