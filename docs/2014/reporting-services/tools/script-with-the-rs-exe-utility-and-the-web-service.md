# <a name="script-with-the-rsexe-utility-and-the-web-service"></a>使用 rs.exe 实用工具和 Web 服务编写脚本
  开发人员和报表服务器管理员可以使用 **rs** 实用工具 (RS.exe) 在报表服务器上执行操作。 使用此实用工具，可以通过使用编写的[!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[vbprvb](../../includes/vbprvb-md.md)]脚本以编程方式管理 Report Server。  
  
 [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]脚本可用于运行任何报表服务器 Web 服务操作。 使用脚本，可以将安全性复制到服务器上的多个报表，添加和删除项，将报表服务器项从一台服务器复制到另一台服务器，等等。 有关脚本环境的详细信息，请参阅 [运行 Reporting Services 脚本文件](run-a-reporting-services-script-file.md)。 脚本文件是用 [!INCLUDE[vbprvb](../../includes/vbprvb-md.md)] .NET 编写的，并采用特定的格式。 有关详细信息，请参阅 [设置 Reporting Services 脚本文件的格式](format-a-reporting-services-script-file.md)。  
  
 有关脚本示例，请参阅下面内容：  
  
 [Reporting Services 用于在报表服务器之间迁移内容的 rs .Exe 脚本的示例](sample-reporting-services-rs-exe-script-to-copy-content-between-report-servers.md)。
