# <a name="browse-for-servers-local-servers"></a>查找服务器（本地服务器）
当连接到 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 组件但不知道 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 的确切名称时，请在“服务器名称”框中单击“浏览更多”，以打开“查找服务器”对话框。  
  
## <a name="options"></a>选项  
**选择要连接的服务器**  
单击树中显示的 [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 实例，指定要连接的服务器。 单击带有 **+** 或 **–** 符号的节点可以显示或隐藏部分树视图。
