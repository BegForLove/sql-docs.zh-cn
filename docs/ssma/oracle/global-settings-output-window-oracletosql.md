# <a name="global-settings-output-window--oracletosql"></a>全局设置 （输出窗口中） (OracleToSQL)
使用**全局设置**对话框中，以指定的 SSMA 设置。  
  
若要访问在输出窗口设置**工具**菜单上，选择**全局设置**，单击**GUI**底部的左窗格中，，然后选择**输出窗口**。  
  
## <a name="options"></a>选项  
**显示错误列表窗格**  
选择**True**如果你想 SSMA 时有消息显示错误列表窗格。  
  
**显示输出窗格**  
选择**True**如果你想 SSMA 以显示输出窗格中，消息时。
