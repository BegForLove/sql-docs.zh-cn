# “会话事件”事件类别
  “会话事件”事件类别具有下表中说明的事件类。  
  
|Event Class|事件 ID|Description|  
|-----------------|--------------|-----------------|  
|Existing Connection|41|收集从开始跟踪之后发生的所有连接事件。|  
|现有会话|42|收集从开始跟踪之后发生的所有会话事件。|  
|会话初始化|43|收集会话初始化事件。|  
  
 有关与每个会话事件事件类关联的列的信息，请参阅 [Session Events Data Columns](../../analysis-services/trace-events/session-events-data-columns.md)。
