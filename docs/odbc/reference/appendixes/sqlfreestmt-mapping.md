# <a name="sqlfreestmt-mapping"></a>SQLFreeStmt 映射
在应用程序调用**SQLFreeStmt**与*选项*参数通过 ODBC 3 SQL_DROP*.x*驱动程序，将会调用  
  
```  
SQLFreeStmt(hstmt, SQL_DROP)   
```  
  
 映射到  
  
```  
SQLFreeHandle(SQL_HANDLE_STMT,Handle)  
```  
  
 与*处理*参数设置中的值为*hstmt*。
