# <a name="proxy-editor-and--add-principal"></a>代理编辑器，并添加主体
  使用此页可以授予服务器主体对 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 代理的代理帐户的访问权限。  
  
## <a name="options"></a>“常规”  
 **主体类型**  
 要显示的主体类型。  
  
 **可用主体**  
 列出所选类型的主体服务器。  
  
 **名称**  
 选择主体的名称以便向其授予访问权限，或者选择“名称”以便向该列表中的所有主体授予访问权限。
