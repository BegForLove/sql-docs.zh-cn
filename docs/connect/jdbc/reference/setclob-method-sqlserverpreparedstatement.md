# <a name="setclob-method-sqlserverpreparedstatement"></a>setClob 方法 (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  将指定的参数设置为 java.sql.Clob 对象。

|名称|Description|  
|----------|-----------------|  
|[setClob 方法&#40;int、 java.io.Reader&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|将指定的参数设置为 java.sql.Clob 对象。|  
|[setClob 方法&#40;int、 java.sql.Clob&#41;](../../../connect/jdbc/reference/setclob-method-int-java-sql-clob.md)|将指定的参数设置为给定的 java.sql.Clob 对象中。|  
|[setClob 方法&#40;int、 java.io.Reader，长&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader-long.md)|将指定的参数设置为一个 java.sql.Clob 对象，它是指定的字符数。|
