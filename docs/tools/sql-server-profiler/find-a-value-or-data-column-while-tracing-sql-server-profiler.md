# <a name="find-a-value-or-data-column-while-tracing-sql-server-profiler"></a>在跟踪时查找值或数据列 (SQL Server Profiler)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]本主题介绍如何为特定值或数据列，在跟踪输出使用运行跟踪时搜索[!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)]。  
  
### <a name="to-find-a-value-or-data-column-while-running-a-trace"></a>在运行跟踪时查找值或数据列  
  
1.  在运行跟踪时，单击跟踪结果中的任何行。  
  
2.  在 **“编辑”** 菜单中，单击 **“查找”**。  
  
3.  在“查找”对话框中的“查找内容”文本框中，输入搜索值，并选择指定数据列。  
  
4.  若要查找下一个匹配值，请单击 **“查找下一个”**。  
  
5.  若要查找上一个匹配值，请单击 **“查找上一个”**。  
  
     搜索从步骤 1 中选择的行开始。
