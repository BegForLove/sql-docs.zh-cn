# <a name="username-mdx"></a>UserName (MDX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  返回当前连接的域名和用户名。  
  
## <a name="syntax"></a>语法  
  
```  
  
UserName [ ( ) ]  
```  
  
## <a name="remarks"></a>注释  
 返回值为以下格式的字符串：  
  
 *域名 \ 用户名*  
  
## <a name="example"></a>示例  
 下例将返回正在执行查询的用户的用户名。  
  
```  
WITH MEMBER Measures.x AS UserName  
SELECT Measures.x ON COLUMNS  
FROM [Adventure Works]  
  
```
