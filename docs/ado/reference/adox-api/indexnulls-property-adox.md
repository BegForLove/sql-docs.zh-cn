# <a name="indexnulls-property-adox"></a>IndexNulls 属性 (ADOX)
指示是否在其索引字段中具有 null 值的记录具有索引条目。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置并返回[AllowNullsEnum](../../../ado/reference/adox-api/allownullsenum.md)值。 默认值是**adIndexNullsDisallow**。  
  
## <a name="remarks"></a>Remarks  
 此属性为只读上[索引](../../../ado/reference/adox-api/index-object-adox.md)已追加到集合的对象。  
  
## <a name="applies-to"></a>适用范围  
 [索引对象 (ADOX)](../../../ado/reference/adox-api/index-object-adox.md)
