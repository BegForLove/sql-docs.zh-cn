# <a name="dmx-statements---data-manipulation"></a>DMX 语句的数据操作

|主题|Description|  
|-----------|-----------------|  
|[删除&AMP;#40;DMX&AMP;#41;](../dmx/delete-dmx.md)|从挖掘模型中清除已定型的内容。|  
|[将插入&AMP;#40;DMX&AMP;#41;](../dmx/insert-into-dmx.md)|为挖掘模型定型。|  
|[选择&AMP;#40;DMX&AMP;#41;](../dmx/select-dmx.md)|浏览挖掘模型。|  
|[&#60;源数据查询&#62;](../dmx/source-data-query.md)|两个查询数据源**INSERT INTO**和**选择**语句。|  
|[更新&AMP;#40;DMX&AMP;#41;](../dmx/update-dmx.md)|更改挖掘模型中的内容。|
