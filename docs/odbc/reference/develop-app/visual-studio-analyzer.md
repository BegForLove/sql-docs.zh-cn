# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Windows 8 （Visual Studio Analyzer 仅包含在较旧版本的 Visual Studio。） 从开始已删除的 Visual Studio Analyzer 的支持。 有关故障排除机制的替代方法，使用 BID 跟踪。  
  
 Microsoft® Visual Studio™ 分析器提供 ODBC 应用程序的高级视图。 开发人员可以评估、 分析和调试结构、 性能和从应用程序的角度来看，而不是从组件或代码的角度来看的交互。 Visual Studio 分析器所收集的信息是交互的形式的事件，表示某种类型的应用程序的两个组件之间。 事件可以由函数调用和函数，如连接、 数据库查询和事务的返回值组成。
