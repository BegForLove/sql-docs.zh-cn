## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|26013|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|SNI_SSL_READY|  
|消息正文|已成功加载用于加密的证书 [Cert Hash(sha1) "%hs"]。|  
  
## <a name="explanation"></a>解释  
这是一条信息性消息。 命名证书已成功加载到 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 中，而且可用于加密。  
  
## <a name="user-action"></a>用户操作
