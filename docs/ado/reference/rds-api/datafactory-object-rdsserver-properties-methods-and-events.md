# <a name="datafactory-object-rdsserver-properties-methods-and-events"></a>DataFactory 对象 (RDSServer) 属性、 方法和事件
> [!IMPORTANT]
>  从 Windows 8 和 Windows Server 2012 开始，不再在 Windows 操作系统中包含 RDS 服务器组件 (请参阅 Windows 8 和[Windows Server 2012 兼容性手册](https://www.microsoft.com/en-us/download/details.aspx?id=27416)有关详细信息)。 将 Windows 的未来版本中删除 RDS 客户端组件。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 使用 RDS 的应用程序应迁移到[WCF 数据服务](http://go.microsoft.com/fwlink/?LinkId=199565)。  
  
## <a name="properties"></a>属性  
 无。  
  
## <a name="methods"></a>方法  
  
|||  
|-|-|  
|[ConvertToString 方法 (RDS)](../../../ado/reference/rds-api/converttostring-method-rds.md)|将转换为 MIME64 字符串的记录集。|  
|[CreateRecordset 方法 (RDS)](../../../ado/reference/rds-api/createrecordset-method-rds.md)|创建并返回一个空的记录集。|  
|[Execute 方法 (RDS)](../../../ado/reference/rds-api/execute-method-rds.md)|执行请求，并创建一个高级的数据行集 （用于用 ADO 2.5 或更高版本）。|  
|[Execute21 方法 (RDS)](../../../ado/reference/rds-api/execute21-method-rds.md)|执行请求，并创建一个高级的数据行集 （用于用 ADO 2.1）。|  
|[Query 方法 (RDS)](../../../ado/reference/rds-api/query-method-rds.md)|执行请求，并创建高级的数据行集。|  
|[SubmitChanges 方法 (RDS)](../../../ado/reference/rds-api/submitchanges-method-rds.md)|给定的记录集包含挂起的更改，此方法将其提交给标识连接字符串中的数据库。|  
|[Synchronize 方法 (RDS)](../../../ado/reference/rds-api/synchronize-method-rds.md)|将给定的记录集与指定的连接字符串 （与 ADO 2.5 或更高版本一起使用） 的数据库同步。|  
|[Synchronize21 方法 (RDS)](../../../ado/reference/rds-api/synchronize21-method-rds.md)|将给定的记录集与指定的连接字符串 （用 ADO 2.1 的使用） 的数据库同步。|  
  
## <a name="events"></a>事件  
 无。
