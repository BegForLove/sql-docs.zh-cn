# <a name="sql-statements"></a>SQL 语句
ODBC 应用程序通过执行 SQL 语句执行几乎所有数据库访问。 这些语句的形式 - 硬编码或构造在运行时，可互操作或数据源特定，等等 - 取决于应用程序的需求。
