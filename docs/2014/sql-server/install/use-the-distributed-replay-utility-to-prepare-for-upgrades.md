# <a name="use-the-distributed-replay-utility-to-prepare-for-upgrades"></a>使用 Distributed Replay 实用工具准备升级
  [!INCLUDE[msCoName](../../includes/msconame-md.md)] Distributed Replay 实用工具可以使用多台计算机重播跟踪数据，并模拟任务关键型工作[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]负荷。 通过在 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 升级前和升级后在测试服务器上执行重播，可以测量性能差异，并查找您的应用程序可能与升级版本不兼容的问题。 有关详细信息，请参阅 [SQL Server Distributed Replay](../../tools/distributed-replay/sql-server-distributed-replay.md)。
