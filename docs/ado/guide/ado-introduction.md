# <a name="ado-overview-and-benefits"></a>ADO 概述和权益
ActiveX 数据对象 (ADO) 是 OLE DB 的高级别、 简单易用的界面。 OLE DB 是多数据存储区的低级别、 高性能界面。 ADO 和 OLE DB 可以使用关系 （表格） 和非关系 (层次结构或流) 数据。

 ADO 提供了一个在客户端或中间层应用程序和低级别的 OLE DB 接口之间的抽象层。 ADO 使用少量的自动化对象到 OLE DB 提供简单且有效的接口。 此接口使 ADO 的更高级别的语言，如 Visual Basic 和 VBScript 中，在开发人员想要访问数据，而无需了解的 COM 和 OLE DB 复杂一个不错的选择。
