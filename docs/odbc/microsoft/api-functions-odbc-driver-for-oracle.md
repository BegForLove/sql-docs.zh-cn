# <a name="api-functions-odbc-driver-for-oracle"></a>API 函数（Oracle ODBC 驱动程序）
> [!IMPORTANT]  
>  此功能将 Windows 的未来版本中删除。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 Microsoft ODBC Driver for Oracle 支持的核心级别、 级别 1 和级别 2 API 函数。 这些函数都列在[ODBC 一致性级别](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)。  
  
-   [核心级别接口](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md)(CLI) 符合性提供了 ISO CLI 规范中定义的特性和打开组 CLI 规范中定义的必需功能。  
  
-   [级别 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md)一致性提供核心级别接口功能还为其他功能，例如事务。  
  
-   [级别 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md)一致性提供第一级功能也为其他功能，例如书签、 动态参数和 ODBC 函数的异步执行。
