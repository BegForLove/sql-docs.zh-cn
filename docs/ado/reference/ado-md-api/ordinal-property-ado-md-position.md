# <a name="ordinal-property-ado-md-position"></a>Ordinal 属性（ADO MD 位置）
唯一标识[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)沿某个轴。  
  
## <a name="return-values"></a>返回值  
 返回**长**整数和是只读的。  
  
## <a name="remarks"></a>备注  
 **序号**的属性[位置](../../../ado/reference/ado-md-api/position-object-ado-md.md)对象的索引对应**位置**内[位置](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)集合。  
  
 快速可使用检索单元格**序号**的**位置**使用的每个轴[项](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md)属性的[单元集](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)对象.
