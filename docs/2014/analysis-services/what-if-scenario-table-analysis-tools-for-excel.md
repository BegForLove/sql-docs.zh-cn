# <a name="what-if-scenario-table-analysis-tools-for-excel"></a>“假设”应用场景（Excel 表分析工具）
  ![表分析工具中的什么如果方案按钮](media/tat-whatif.gif "表分析工具中的什么如果方案按钮")  
  
 **假设**方案工具可分析中现有数据的模式，然后使您能够计算一列中的更改会有不同的列的值的效果。  
  
 例如，您可以检查某件商品在价格上涨时对其总销售额的影响。  
  
 此工具在可以创建的预测数方面比较灵活。 初始分析完成后，可以针对表中的所有数据预测更改，也可以一次输入一个测试值。  
  
## <a name="using-the-what-if-scenario-tool"></a>使用的假设分析方案工具  
  
1.  打开 Excel 数据表。  
  
2.  单击**方案**，然后选择**假设**。  
  
3.  在**方案**框中，选择包含你将更改，并为特定值或百分比值的更改 （不管是增加或减少） 指定更改的值的当前值的列。  
  
4.  在**会发生什么情况**框中，指定你想要评估的影响的列。  
  
5.  （可选） 单击**选择要用于分析列**选择很可能是在进行预测有用的列。 也可取消选择对检测模式几乎没有什么用的列，如行 ID 或名称。  
  
6.  在**指定行或表**框中，选择是否想要评估为当前选定的行仅，或完整的表中的数据集的影响。  
  
7.  如果你选择**在此行上**，该工具会在对话框中显示结果。 在对话框保持打开的情况下，您可以修改选择以测试其他应用场景。  
  
8.  如果你选择**整个表**，该工具在对话框中，显示状态消息，并将两个新列添加到原始数据表。 单击**关闭**工作表中查看完整的结果。  
  
### <a name="requirements"></a>要求  
 此工具使用 Microsoft 逻辑回归算法，这一算法支持数值或离散值的预测。 但是，为达到最佳结果，建议遵循以下最佳实践：  
  
-   选择包含有用信息的列进行分析。  
  
-   如果包含的列太少，则可能很难获得结果。  
  
-   如果你使用**为值**选项，你必须在框中键入一个值，或从列表中选择一个值。  
  
-   如果你使用**百分比**选项，将更改设置为百分比增加或减少。 默认值为 120%，表示在当前值的基础上增加 20%。  
  
> [!NOTE]  
>  如果不设置值，则可能会出现错误。  
  
## <a name="understanding-the-results-of-what-if-analysis"></a>了解假设分析的结果  
 当你创建**假设**方案中，该工具执行三项操作：  
  
-   创建存储有关表中数据关键事实的数据挖掘结构。  
  
-   基于数据创建逻辑回归挖掘模型。  
  
-   创建所指定的每个值的预测查询。  
  
 你可以通过指定的输出所有的预测一次**整个表**。 此工具然后在原始数据表中创建两个新列。  
  
 添加到表中的列包含两种类型的信息：给定更改幅度后得到的预测值以及预测值的置信度。 置信度表示预测准确的概率。  
  
 也可在对话框中逐个输入更改值，然后交互式查看预测。 这是创建相同*单独预测查询*。 输出的预测查询结果中包括以下信息：预测是成功还是失败、预测值以及置信度。 置信度显示为包含点线的条形图。 点线越长，结果的置信度就越高。  
  
 例如，如果你尝试确定提高客户的意愿购买，并增加为 25，客户的年龄上的客户的年龄**假设**工具查询数据挖掘模型，并返回下面的结果：  
  
 **购买自行车的假设分析找到了解决方案。**  
  
 **购买自行车 = 是**  
  
 **置信度： 公平**  
  
 由于此结果基于数据表中的现有行，因此这意味着，对于特定的客户，保持所有其他条件不变，而将该客户的年龄增加到 25 岁，那么该客户将有可能购买自行车。  
  
 将预测输出到原始数据表中更便于确定预测是否有用。
