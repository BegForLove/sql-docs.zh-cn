# <a name="tracing"></a>跟踪
ODBC 驱动程序管理器具有所允许的 ODBC 应用程序，以记录和转录到日志文件进行的函数调用序列的跟踪功能。 将跟踪捕获应用程序和驱动程序管理器中，驱动程序管理器和驱动程序之间以及之间的调用的 DLL 执行跟踪。 此方法跟踪替换由 ODBC 2 执行跟踪*.x* ODBC 2 执行驱动程序管理器和跟踪*.x*通过 ODBC Spy。
