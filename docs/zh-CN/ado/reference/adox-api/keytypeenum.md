# <a name="keytypeenum"></a>KeyTypeEnum
指定的一种[密钥](../../../ado/reference/adox-api/key-object-adox.md)： 主键、 外，或唯一。  
  
|常量|“值”|Description|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|默认值。 密钥是主密钥。|  
|**adKeyForeign**|2|密钥是外键。|  
|**adKeyUnique**|3|该密钥是唯一的。|
