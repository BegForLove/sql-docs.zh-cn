# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>mssql-cli 命令行查询工具, 用于 SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

mssql-cli 是一个交互式命令行工具, 用于查询 SQL Server、在 Windows、macOS 或 Linux 上安装该工具。

## <a name="install-mssql-cli"></a>安装 mssql-cli

有关详细的安装说明, 请参阅[安装指南](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md), 如果你知道 pip, 请运行以下命令进行安装:

```$ pip install mssql-cli```
