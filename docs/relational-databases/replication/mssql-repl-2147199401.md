# MSSQL_REPL-2147199401
    
## 消息详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|-2147199401|  
|事件源|MSSQLServer|  
|符号名称||  
|消息正文|合并代理在检测到基于保持期的元数据清除操作已针对尚未发送到发布服务器的更改删除了订阅服务器上的元数据之后，合并代理失败。 您必须重新初始化订阅（无需上载）。|  
  
## 解释  
 因为合并进程检测到订阅服务器上基于保持期的元数据清除操作已针对尚未发送到发布服务器的更改删除了元数据，所以合并进程失败。  
  
## 用户操作  
 通过指定重新初始化订阅 **@upload_first = 'FALSE'**。
