# <a name="kpiweight-mdx"></a>KPIWeight (MDX)


  返回指定关键绩效指标 (KPI) 的权重。  
  
## <a name="syntax"></a>语法  
  
```  
  
KPIWeight(KPI_Name)  
```  
  
## <a name="arguments"></a>参数  
 *KPI_Name*  
 指定 KPI 的名称的有效字符串表达式。  
  
## <a name="remarks"></a>备注  
 返回的值是 KPI 在父级中所占的比例。
