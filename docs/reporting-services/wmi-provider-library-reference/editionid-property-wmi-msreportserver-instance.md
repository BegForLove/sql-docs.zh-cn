# EditionID 属性 (WMI MSReportServer_Instance)
  返回报表服务器的版本类别。 只读。  
  
## 语法  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## 属性值  
 包含报表服务器版本类别的 **uint** 值。
