# <a name="axis-title-properties-dialog-box-general-report-builder-and-ssrs"></a>“轴标题属性”对话框 -&gt;“常规”（报表生成器和 SSRS）
  在 **“轴标题属性”** 对话框中选择 **“常规”** 可以定义轴标题。 轴标题显示在 X 轴下方或 Y 轴旁边。  
  
## <a name="options"></a>选项  
 **标题**  
 键入一个标题或用来表示轴标题的表达式。  
  
 **标题对齐**  
 选择一种对齐方式，或键入一个表示轴标题相对于轴交点的对齐方式的表达式。  
  
-   **接近**沿两个轴的交集显示轴标题。  
  
-   **中心**在轴的中央显示轴标题。  
  
-   **远**显示离两个轴的交集的轴标题。
