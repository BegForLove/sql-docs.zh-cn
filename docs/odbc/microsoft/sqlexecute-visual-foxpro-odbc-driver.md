# <a name="sqlexecute-visual-foxpro-odbc-driver"></a>SQLExecute （Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含 Visual FoxPro ODBC 驱动程序相关的信息。 有关此函数的常规信息，请参阅下的相应主题[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支持： 完整  
  
 ODBC API 一致性： 核心级别  
  
 执行已准备的 SQL 语句 (已准备的语句[SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md))。 如果语句中存在任何参数，驱动程序将使用的参数标记变量的当前值。  
  
 有关详细信息，请参阅[SQLExecute](../../odbc/reference/syntax/sqlexecute-function.md)中*ODBC 程序员参考*。
