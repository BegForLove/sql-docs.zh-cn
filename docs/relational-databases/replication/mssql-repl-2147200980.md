# MSSQL_REPL-2147200980
    
## 消息详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|-2147200980|  
|事件源|MSSQLServer|  
|符号名称||  
|消息正文|订阅已过期。 将订阅标记为重新初始化，并重新启动合并代理以重新初始化订阅。|  
  
## 解释  
 因为匿名订阅已过期，所以发生此错误。  
  
## 用户操作  
 使用 sp_reinitmergepullsubscription 重新初始化匿名订阅，然后重新运行合并操作。
