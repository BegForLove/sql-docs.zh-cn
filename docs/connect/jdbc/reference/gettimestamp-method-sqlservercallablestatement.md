# <a name="gettimestamp-method-sqlservercallablestatement"></a>getTimestamp 方法 (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索指定参数的值作为 Java 编程语言中的 java.sql.Timestamp 对象。  
  
## <a name="overload-list"></a>重载列表  
  
|Name|Description|  
|----------|-----------------|  
|[getTimestamp (int)](../../../connect/jdbc/reference/gettimestamp-method-int.md)|根据给定的参数索引，检索指定参数的值作为 Java 编程语言中的 java.sql.Timestamp 对象。|  
|[getTimestamp （int、 java.util.Calendar）](../../../connect/jdbc/reference/gettimestamp-method-int-java-util-calendar.md)|作为 Java 编程语言提供的参数索引，使用日历对象中的 java.sql.Timestamp 对象中检索指定参数的值。|  
|[getTimestamp (java.lang.String)](../../../connect/jdbc/reference/gettimestamp-method-java-lang-string.md)|根据给定的参数名称，检索指定参数的值作为 Java 编程语言中的 java.sql.Timestamp 对象。|  
|[getTimestamp （java.lang.String，java.util.Calendar）](../../../connect/jdbc/reference/gettimestamp-method-java-lang-string-java-util-calendar.md)|作为 Java 编程语言提供参数名称，使用日历对象中的 java.sql.Timestamp 对象中检索指定参数的值。|
