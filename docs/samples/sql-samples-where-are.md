## <a name="wideworldimporters-sample-database"></a>WideWorldImporters 示例数据库

WideWorldImporters (WWI) 是演示了如何使用当前的功能、 工具和设计技术的数据库设计的示例数据库。 WideWorldImporters 下载包括 OLTP 和 OLAP 数据库构建运行 SQL Server 2016 及更高版本。 

数据库下载和示例位于[SQL Server 示例](https://github.com/Microsoft/sql-server-samples)GitHub 存储库。


- [WideWorldImporters 自述文件和示例](https://github.com/Microsoft/sql-server-samples/tree/master/samples/databases/wide-world-importers)GitHub 上。
