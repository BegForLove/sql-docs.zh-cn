# <a name="the-sql-server-management-studio-environment"></a>SQL Server Management Studio 环境

[!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull_md.md)] 在 [!INCLUDE[ssNoVersion](../includes/ssnoversion_md.md)] 中为管理和开发查询提供一个功能丰富的环境。
