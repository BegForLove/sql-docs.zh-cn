# <a name="updating-joined-results-unique-table"></a>更新已加入结果： 唯一表
ADO 使您能够紧密控制特定基础表中的组上使用唯一表动态属性的多个基表的联接操作而形成的记录集的修改。 有关使用唯一表的详细信息，请参阅上的唯一表和更新再同步的动态属性的 ADO 程序员参考主题。
