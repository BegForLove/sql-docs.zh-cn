# <a name="calculated-series-properties-dialog-box-general-report-builder-and-ssrs"></a>“计算序列属性”对话框 -&gt;“常规”（报表生成器和 SSRS）
  在“计算序列属性”对话框中选择“公式”可以添加或修改将根据使用现有序列数据作为输入的内置公式计算的序列。  
  
## <a name="options"></a>选项  
 **公式**  
 选择要对图表应用的公式。 选项包括：  
  
-   中间线  
  
-   中线  
  
-   布林线  
  
-   移动平均线  
  
-   指数移动平均线  
  
-   三角移动平均线  
  
-   加权移动平均线  
  
-   平滑移动平均线  
  
-   区间震荡线  
  
-   包络线  
  
-   性能  
  
-   相对强弱指数  
  
-   标准差  
  
-   TRIX  
  
 **设置公式参数**  
 为所选公式指定参数。
