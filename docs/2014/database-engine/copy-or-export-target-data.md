# <a name="copy-or-export-target-data"></a>复制或导出目标数据
  可以复制目标数据，还可以将目标数据导出到 .CSV 文件。  
  
> [!NOTE]  
>  不能复制或导出 **ring_buffer** 目标的目标数据。  
  
## <a name="to-copy-target-data"></a>复制目标数据  
  
-   在目标数据窗口中，选择要复制的行，右键单击所选项，然后选择 **“复制”**。  
  
## <a name="to-export-target-data-to-a-csv-file"></a>将目标数据导出到 .CSV 文件  
  
1.  在目标数据窗口中，选择要导出的行，右键单击所选项，然后选择 **“导出至 CSV”**。  
  
2.  在 **“另存为”** 对话框中，指定文件夹和文件名。  
  
3.  单击 **“确定”** 将目标数据保存为 CSV 文件。
