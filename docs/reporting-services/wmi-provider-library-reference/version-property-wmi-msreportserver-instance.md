# Version 属性 (WMI MSReportServer_Instance)
  以格式 Major.Minor.Build.Revision 返回报表服务器的版本。 只读。  
  
## 语法  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## 属性值  
 包含报表服务器的版本的 **string** 。
