# <a name="product-support"></a>产品支持
针对 ODBC 的产品支持是由 Microsoft 产品支持服务提供的。 由于很多 Microsoft 产品使用 ODBC 作为核心组件并重新分发 ODBC 驱动程序，Microsoft 产品支持服务会将 odbc 视为将 ODBC 作为组件合并的每个产品的一部分。 根据该产品的支持协议提供对 ODBC 的支持。 有关详细信息，请参阅单独的产品文档。
