# <a name="productlabel-element"></a>ProductLabel 元素

## <a name="syntax"></a>语法  
  
```xml  
  
<Server>  
...  
   <ProductLevel>...</ProductLevel>  
...  
</Server>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|说明|  
|--------------------|-----------------|  
|数据类型和长度|字符串|  
|默认值|无|  
|基数|0-1：可出现一次且仅出现一次的可选元素。|
