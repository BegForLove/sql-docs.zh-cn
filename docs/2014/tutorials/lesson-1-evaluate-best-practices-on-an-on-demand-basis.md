# <a name="lesson-1-evaluate-best-practices-on-an-on-demand-basis"></a>课程 1：按需评估最佳实践策略
  通过使用策略评估的按需（或者“即席”）方法，您可以快速评估最佳实践策略，以便确定 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] 的实例是否满足最佳实践指导原则和建议。  
  
 在本课程中，你将执行按需评估的最佳实践策略[!INCLUDE[ssDEnoversion](../includes/ssdenoversion-md.md)]的一个或多个实例上[!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]。  
  
 本课程包含以下主题：  
  
-   [执行按需评估通过使用对象资源管理器](../ssms/object/object-explorer.md)（对于一个实例）  
  
-   [执行按需评估通过使用已注册的服务器](../../2014/tutorials/perform-an-on-demand-evaluation-by-using-registered-servers.md)（适用于一个或多个实例）
