# <a name="path-element-assl"></a>Path 元素 (ASSL)

## <a name="syntax"></a>语法  
  
```xml  
  
<ReportAction>  
   ...  
   <Path>...</Path>  
   ...  
</ReportAction>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|说明|  
|--------------------|-----------------|  
|数据类型和长度|字符串|  
|默认值|无|  
|基数|0-1： 发生一次，并且一次的可选元素|
