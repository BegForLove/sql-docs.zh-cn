# <a name="task-3-verifying-the-data-in-master-data-manager"></a>任务 3：在主数据管理器中验证数据
  在本任务中，您验证**供应商**上创建实体**MDS**使用**主数据管理器 Web 应用程序**。  
  
1.  如果**主数据管理器**已打开，请单击**SQL Server 2012 Master Data Services**顶部导航到主页页面。 否则，请导航到[ http://localhost/mds ](http://localhost/mds)以启动**主数据管理器**。  
  
2.  选择**供应商**有关**模型**，然后单击**资源管理器**。  
  
     ![主数据管理器-资源管理器按钮](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "主数据管理器-资源管理器按钮")  
  
3.  查看存储在 MDS 上的数据。 如果没有看到数据，确认所选**供应商**有关**模型**之前启动的主页上**资源管理器**。 可以将添加到或使用从供应商列表中删除**添加成员**并**删除成员**工具栏上的按钮。
