# <a name="redistributing-the-microsoft-jdbc-driver"></a>再分发 Microsoft JDBC 驱动程序
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

## <a name="redistribute-the-microsoft-jdbc-41-42-60-62-64-70-72-and-74-driver"></a>重新分发 Microsoft JDBC 4.1、4.2、6.0、6.2、6.4、7.0、7.2 和7.4 驱动程序
可重新分发 JDBC 驱动程序4.1、4.2、6.0、6.2、6.4、7.0、7.2 和7.4。 请在许可协议中查看4.1、4.2、6.0、6.2、6.4、7.0、7.2 和7.4 版本驱动程序的 "可分发代码" 条款。
    
## <a name="register-to-redistribute-the-microsoft-jdbc-40-driver"></a>注册以再分发 Microsoft JDBC 4.0 驱动程序  
 Microsoft JDBC 4.0 驱动程序需要注册, 然后再重新分发它。 单击下面的链接，查看 4.0 版驱动程序的许可协议。  如果需要，可以打印并保留一份许可协议留作记录。  
  
 如果选择接受该许可协议，你将定向到注册页，然后再转到下载页。  
  
 [Microsoft SQL Server JDBC 驱动程序 4.0 的再分发许可](https://msdn.microsoft.com/sqlserver/jj589698)  
  
 注意: Microsoft 使用 Microsoft 帐户进行安全身份验证和注册。 通过 Microsoft 帐户注册获得 SQL Server JDBC 驱动程序的再分发权利时，系统将提示你提供一些信息，以作为安全个人资料的一部分。  
  
 如果还没有, 可以获取[Microsoft 帐户](https://signup.live.com/)
