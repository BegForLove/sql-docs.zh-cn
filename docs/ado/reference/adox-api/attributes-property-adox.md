# <a name="attributes-property-adox"></a>Attributes 属性 (ADOX)
描述列的特征。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 设置或返回**长**值。 值指定由表示的表的特征[列](../../../ado/reference/adox-api/column-object-adox.md)对象。 该值可以为的组合[ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md)常量。 默认值为 0 (**0**)，其中既不是**adColFixed**也不**adColNullable**。
