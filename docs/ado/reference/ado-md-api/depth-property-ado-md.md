# <a name="depth-property-ado-md"></a>Depth 属性 (ADO MD)
指示级别之间的数[级别](../../../ado/reference/ado-md-api/level-object-ado-md.md)和层次结构的根。  
  
## <a name="return-values"></a>返回值  
 返回一个整数，并且是只读的。  
  
## <a name="remarks"></a>备注  
 一个**级别**层次结构的根处具有**深度**值为零 (0)。
