# <a name="machine-data-sources"></a>计算机数据源
*计算机数据源*存储在系统中具有用户定义的名称。 与数据源名称相关联是信息的所有驱动程序管理器和驱动程序，需要能够连接到数据源。 对于 Xbase 数据源，这可能是目录的 Xbase 驱动程序，包含 Xbase 文件，并告知驱动程序如何使用这些文件，如单用户模式或只读的一些选项的完整路径的名称。 对于 Oracle 数据源，这可能是 Oracle 驱动程序的名称，Oracle DBMS 所在的服务器的 SQL * Net 连接字符串，标识 SQL\*.net 驱动程序使用，并在服务器上的数据库的系统 ID。
