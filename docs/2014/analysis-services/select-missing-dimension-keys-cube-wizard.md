# <a name="select-missing-dimension-keys-cube-wizard"></a>选择缺少的维度键（多维数据集向导）
  可以使用 **“选择缺少的维度键”** 页选择在新维度没有为其主键表定义键时使用的键列。  
  
## <a name="options"></a>“常规”  
 **维度键**  
 从维度的主键表中选择一个或多个键列，这些键列将用作该维度的键。  
  
 如果您选择多个键列，则将创建一个组合键。
