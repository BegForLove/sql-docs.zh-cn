## <a name="icommandexecute"></a>ICommand::Execute  
 如果插入的数据大于列的大小，通常会导致错误。 但是，可能会出现将返回 S_OK、但 dwStatus 将设置为 DBSTATUS_S_TRUNCATED 的情况  。 当用参数插入数据时, 通常会发生这种情况, 列的大小不足以容纳数据, 并且未调用**ICommandWithParameters:: SetParameterInfo** 。
