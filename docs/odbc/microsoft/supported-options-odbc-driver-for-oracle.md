 用于 Oracle 的 ODBC 驱动程序 SQLGetConnectOption （） 和 SQLSetConnectOption （） 级别 1 函数支持以下选项：  
  
-   SQL_ACCESS_MODE ([SQLGetConnectOption](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md)仅 （))

-   [SQL_TRANSLATE_DLL](../../odbc/microsoft/connect-options.md)  
  
-   [SQL_TRANSLATE_OPTION](../../odbc/microsoft/connect-options.md)
