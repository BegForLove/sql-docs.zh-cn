# <a name="legend-properties-dialog-box-general-report-builder-and-ssrs"></a>“图例属性”对话框 ->“常规”（报表生成器和 SSRS）
  使用 **“图例”** 对话框可以定义图表中图例的外观和布局。 有关详细信息，请参阅[设置图表上的图例的格式&#40;报表生成器和 SSRS&#41;](report-design/chart-legend-formatting-report-builder.md)。  
  
## <a name="options"></a>选项  
 **名称**  
 为相应图例键入一个名称。  
  
 **布局**  
 选择选项以指示图例的布局。 默认图例布局为自动调整表。  
  
-   **长表** 将图例的内容显示在一个表中。 将添加图例条目以便垂直展开图例。  
  
-   **列** 将图例的内容显示在单个列中。  
  
-   **行** 将图例的内容显示在单个行中。  
  
-   **宽表** 将图例的内容显示在一个表中。 将添加图例条目以便水平展开图例。  
  
-   **自动调整表** 将图例的内容显示在一个表中。 将添加图例条目以便水平或垂直展开图例，采用哪种展开方式取决于图表中可用的空间。  
  
 **停靠到图表区**  
 选择此选项可以将图例与图表区关联起来。  
  
 **在图表区之外显示图例**  
 选择此选项可以在图表区外显示图例。 此选项仅在图例停靠到图表区时才适用。
