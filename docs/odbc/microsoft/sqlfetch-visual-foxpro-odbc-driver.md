# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含 Visual FoxPro ODBC 特定于驱动程序的信息。 有关此功能的一般信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)下的相应主题。  
  
 支持： 完整  
  
 ODBC API 一致性：核心级别  
  
 从结果集中检索到以前调用[SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md)指定的位置中的一行。 为调用[SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md)为未绑定列准备驱动程序。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLFetch。](../../odbc/reference/syntax/sqlfetch-function.md)
