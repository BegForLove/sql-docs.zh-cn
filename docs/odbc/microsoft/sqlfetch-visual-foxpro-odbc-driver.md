# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含 Visual FoxPro ODBC 驱动程序特定信息。 有关此函数的常规信息，请参阅下的相应主题[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)。  
  
 支持： 完整  
  
 ODBC API 一致性： 核心级别  
  
 从设置到由以前调用的指定的位置的结果中检索一行[SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md)。 准备驱动程序用于调用[SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md)未绑定列。  
  
 有关详细信息，请参阅[SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md)中*ODBC 程序员参考*。
