# <a name="hostname-property-sqlservice-class"></a>HostName 属性（SqlService 类）
  获取服务的主机名。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.HostName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示服务的 [SqlService 类](sqlservice-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定服务的主机名的字符串值。
