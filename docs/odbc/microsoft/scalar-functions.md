# <a name="scalar-functions"></a>标量函数
Microsoft ODBC 桌面数据库驱动程序支持以下标量函数：  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 有关参数和返回值的标量函数的信息，请参阅的附录 F *ODBC 程序员参考。*
