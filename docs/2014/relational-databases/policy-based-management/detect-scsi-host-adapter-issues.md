# <a name="detect-scsi-host-adapter-issues"></a>检测 SCSI 主机适配器问题
  此规则检查系统事件日志中是否存在 EventId 1066。 此错误是由 SCSI 主机适配器配置问题或设备故障引起的。

-   [Microsoft 知识库文章 311081](https://go.microsoft.com/fwlink/?linkid=117744)
