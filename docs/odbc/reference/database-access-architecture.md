# <a name="database-access-architecture"></a>数据库访问体系结构
ODBC开发中的问题之一是数据库访问体系结构的哪一部分要标准化。 上一节中介绍的 SQL 编程接口 （ 嵌入式 SQL、 SQL 模块和 CLI ） 只是此体系结构的一部分。 事实上，由于 ODBC 主要用于将基于个人计算机的应用程序连接到微型计算机和大型计算机 DBMS，因此也有许多网络组件，其中一些组件可以标准化。
