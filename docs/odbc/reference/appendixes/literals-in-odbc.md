# <a name="literals-in-odbc"></a>ODBC 中的文本
以下部分中的语法用于 ODBC 中的间隔和数字文本。 当从字符串类型执行转换到数字或间隔类型，或从数字或间隔类型到字符串类型时，此语法作为驱动程序编写器的辅助工具提供。 有关详细信息，请参阅附录 D 中的[间隔文本](../../../odbc/reference/appendixes/interval-literals.md)和[数字文本](../../../odbc/reference/appendixes/numeric-literals.md)：数据类型。  
  
 本附录包含以下主题。  
  
-   [间隔文本语法](../../../odbc/reference/appendixes/interval-literal-syntax.md)  
  
-   [数值语法](../../../odbc/reference/appendixes/numeric-literal-syntax.md)
