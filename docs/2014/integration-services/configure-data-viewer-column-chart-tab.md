# <a name="configure-data-viewer-column-chart-tab"></a>配置数据查看器（“柱形图”选项卡）
  可以使用 **“配置数据查看器”** 对话框中的 **“柱形图”** 选项卡，指定要在图表中显示为图条的列。 图条数显示了列中离散数据值的个数，图条的高度显示了每个值的实例数。 该图表只反应一个缓冲区的内容。  
  
## <a name="options"></a>“常规”  
 **列名**  
 选择要在柱形图中进行可视化处理的列。
