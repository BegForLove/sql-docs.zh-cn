# SQL Server，LogPool FreePool 对象
**SQLServer：LogPool FreePool** 性能对象提供日志池内可用池的统计信息的计数器。

下表介绍了 SQL Server **LogPool FreePool** 性能对象。

|**SQL Server LogPool FreePool 计数器**|Description|  
|-------------|-----------------|  
|**可用缓冲区重填数/秒**|每秒为重填分配的缓冲区数。|
|**可用列表长度**|可用列表的长度。|

每个日志池类别都有一个计数器实例。
