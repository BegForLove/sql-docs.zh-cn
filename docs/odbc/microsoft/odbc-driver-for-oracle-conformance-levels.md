# <a name="odbc-driver-for-oracle-conformance-levels"></a>Oracle ODBC 驱动程序一致性级别
> [!IMPORTANT]  
>  此功能将 Windows 的未来版本中删除。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 ODBC 定义的符合性标准，用于驱动程序的两种类型： API 一致性标准和 SQL 语法合规性标准。 API 一致性是指一个驱动程序支持的函数。 SQL 一致性是指该驱动程序支持的 SQL 语法。 每个符合性标准由多个级别组成。  
  
 本部分提供有关适用于 Oracle 的 ODBC 驱动程序的符合性信息。  
  
 本部分包含以下主题。  
  
-   [API 一致性级别 （Oracle ODCB 驱动程序）](../../odbc/microsoft/api-conformance-level-odbc-driver-for-oracle.md)  
  
-   [支持的选项 （Oracle ODCB 驱动程序）](../../odbc/microsoft/supported-options-odbc-driver-for-oracle.md)  
  
-   [SQL 一致性级别 （Oracle ODCB 驱动程序）](../../odbc/microsoft/sql-conformance-levels-odbc-driver-for-oracle.md)
