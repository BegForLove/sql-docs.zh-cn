# <a name="ado-md-code-examples"></a>ADO MD 代码示例
使用下面的代码示例以了解如何使用 ADO MD 对象、 方法和属性。 这些示例是与 SQL Server 7.0 的 Microsoft SQL Server OLAP Services 安装的示例应用程序的子集。  
  
 这些示例使用 MSOLAP OLE DB 提供程序，并运行针对某一 Microsoft SQL Server OLAP Services 本地主机。 但是，这些示例旨在演示基本 ADO MD 编程技术，并且应轻松改写为其他数据源或提供程序。  
  
-   [Visual Basic 中的 ADO MD 代码示例](../../../ado/reference/ado-md-api/ado-md-code-examples-in-visual-basic.md)
