# 升级到 SQL Server 2016
  你可以将 [!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)]、 [!INCLUDE[ssKilimanjaro](../../includes/sskilimanjaro-md.md)]、 [!INCLUDE[ssSQL11](../../includes/sssql11-md.md)]或 [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)] 实例升级到 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]。 在运行 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安装程序以升级到 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]之前，请先查看以下有关升级过程和发行说明的主题。  
  
 有关从 [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]升级的详细信息，请参阅 [是否正在从 SQL Server 2005 进行升级？](../../database-engine/install-windows/are-you-upgrading-from-sql-server-2005.md)。
