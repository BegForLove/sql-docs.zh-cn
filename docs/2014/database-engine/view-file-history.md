# <a name="view-file-history"></a>查看文件历史记录
  你可以使用**历史记录**命令以查看受源代码管理文件的版本历史记录。 此历史记录包括在源代码管理中添加文件或签入文件的日期和时间。  
  
### <a name="to-view-the-history-of-a-file"></a>若要查看文件历史记录  
  
1.  在解决方案资源管理器中，选择该文件。  
  
2.  上**文件**菜单上，指向**源代码管理**单击**历史记录**。  
  
3.  在**文件历史记录**对话框框中，执行任何以下操作：  
  
4.  查看所选的版本。  
  
5.  查看有关所选版本的特定详细信息。  
  
6.  将所选版本的副本检索到您的本地磁盘。  
  
7.  查看所选版本与本地存储版本之间的区别。  
  
8.  将特定文件版本标识为当前版本。  
  
9. 将历史记录报表导出到打印机、文件或 [!INCLUDE[msCoName](../includes/msconame-md.md)] Windows 剪贴板。  
  
 有关如何执行这些任务的信息，请单击**帮助**按钮**文件历史记录**对话框。
