# <a name="icolumnsrowset"></a>IColumnsRowset
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  OLE DB 驱动程序的 SQL Server 将 DBCOLUMN_BASETABLEINSTANCE 列添加到 IColumnsRowset::GetColumnRowset。 此列返回 DBTYPE_I2，以及由 Microsoft 保留供使用。 此列中的信息将在将来的版本中进行修改。
