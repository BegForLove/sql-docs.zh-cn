# <a name="description-property-sqlservice-class"></a>Description 属性（SqlService 类）
  获取所引用服务的说明。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.Description [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示服务的 [SqlService 类](sqlservice-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个包含所引用服务的文本说明的字符串值。
