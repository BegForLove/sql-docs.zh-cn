# <a name="notification-events-event-category"></a>“通知事件”事件类别
  “通知事件”事件类别包含下表说明的事件类。  
  
|Event Class|事件 ID|Description|  
|-----------------|--------------|-----------------|  
|通知|39|收集自启动跟踪后的所有通知事件。|  
|用户定义|40|收集从开始跟踪之后发生的所有用户定义事件。|  
  
 有关与“通知事件”事件类别中各个事件类关联的列的信息，请参阅 [Notification Events Data Columns](../../analysis-services/trace-events/notification-events-data-columns.md)。
