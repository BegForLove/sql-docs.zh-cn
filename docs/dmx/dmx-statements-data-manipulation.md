|主题|描述|  
|-----------|-----------------|  
|[删除&#40;DMX&#41;](../dmx/delete-dmx.md)|从挖掘模型中清除已定型的内容。|  
|[将插入到&#40;DMX&#41;](../dmx/insert-into-dmx.md)|为挖掘模型定型。|  
|[选择&#40;DMX&#41;](../dmx/select-dmx.md)|浏览挖掘模型。|  
|[&#60;源数据查询&#62;](../dmx/source-data-query.md)|两个查询数据源**INSERT INTO**并**选择**语句。|
