 此特定索引通过编译的程序派生 **#import**指令针对 ADO.dll，然后重新格式化\*生成.tlh 文件。 只有方法、 属性和事件有关的信息已保留。 对于每个属性列出的相应声明的替代语法"`__declspec(property...)`"指令。
