# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>设置 Visual FoxPro ODBC 驱动程序
有关下载和安装 Visual FoxPro ODBC 驱动程序的信息，请参阅 MSDN 下载上的[Visual FoxPro ODBC 驱动程序](https://go.microsoft.com/fwlink/?LinkId=121318)。  
  
 您可以使用 Microsoft 可视化 FoxPro ODBC 驱动程序安装程序执行以下操作：  
  
-   添加新组件。  
  
-   卸下已安装的组件。  
  
-   重新安装以还原丢失的文件和设置。  
  
-   删除以前安装的所有组件。  
  
 在系统上安装驱动程序后，安装程序将识别已安装的驱动程序组件，并显示其他对话框，使您能够更改驱动程序的配置。
