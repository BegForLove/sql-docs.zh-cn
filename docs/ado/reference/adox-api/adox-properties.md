# <a name="adox-properties"></a>ADOX 属性

|||  
|-|-|  
|[ActiveConnection](../../../ado/reference/adox-api/activeconnection-property-adox.md)|指示 ADO**连接**对象所属的目录。|  
|[属性](../../../ado/reference/adox-api/attributes-property-adox.md)|描述列特征。|  
|[群集](../../../ado/reference/adox-api/clustered-property-adox.md)|指示是否对索引进行聚集。|  
|[Command](../../../ado/reference/adox-api/command-property-adox.md)|指定 ADO**命令**可用于创建或执行该过程的对象。|  
|[Count](../../../ado/reference/ado-api/count-property-ado.md)|指示集合中的对象数。|  
|[DateCreated](../../../ado/reference/adox-api/datecreated-property-adox.md)|指示该对象的创建的日期。|  
|[DateModified](../../../ado/reference/adox-api/datemodified-property-adox.md)|指示该对象的上次修改的日期。|  
|[DefinedSize](../../../ado/reference/adox-api/definedsize-property-adox.md)|指示列的规定的最大大小。|  
|[DeleteRule](../../../ado/reference/adox-api/deleterule-property-adox.md)|指示删除主键时执行的操作。|  
|[IndexNulls](../../../ado/reference/adox-api/indexnulls-property-adox.md)|指示索引字段中具有 null 值的记录是否具有索引条目。|  
|[项](../../../ado/reference/ado-api/item-property-ado.md)|指示集合中的特定成员按名称或序号。|  
|[名称](../../../ado/reference/adox-api/name-property-adox.md)|指示对象的名称。|  
|[NumericScale](../../../ado/reference/adox-api/numericscale-property-adox.md)|指示列中的数值小数位数。|  
|[ParentCatalog](../../../ado/reference/adox-api/parentcatalog-property-adox.md)|指定的父目录中的表或列，用于提供对特定于提供程序的属性的访问。|  
|[精度](../../../ado/reference/adox-api/precision-property-adox.md)|指示列中的数据值的最大精度。|  
|[PrimaryKey](../../../ado/reference/adox-api/primarykey-property-adox.md)|指示索引是否表示对表的主键。|  
|[RelatedColumn](../../../ado/reference/adox-api/relatedcolumn-property-adox.md)|指示在相关表 （仅限键列） 中的相关列的名称。|  
|[RelatedTable](../../../ado/reference/adox-api/relatedtable-property-adox.md)|指示相关表的名称。|  
|[SortOrder](../../../ado/reference/adox-api/sortorder-property-adox.md)|指示列 （仅限索引列） 的排序顺序。|  
|[类型 （列）](../../../ado/reference/adox-api/type-property-column-adox.md)|指示列的数据类型。|  
|[类型 （键）](../../../ado/reference/adox-api/type-property-key-adox.md)|指示该密钥的数据类型。|  
|[类型 （表）](../../../ado/reference/adox-api/type-property-table-adox.md)|指示表的类型。|  
|[唯一](../../../ado/reference/adox-api/unique-property-adox.md)|指示是否必须是唯一索引键。|  
|[UpdateRule](../../../ado/reference/adox-api/updaterule-property-adox.md)|指示更新主键时执行的操作。|
