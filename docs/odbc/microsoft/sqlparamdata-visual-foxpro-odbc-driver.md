# <a name="sqlparamdata-visual-foxpro-odbc-driver"></a>SQLParamData（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：级别1  
  
 与[SQLPutData](../../odbc/microsoft/sqlputdata-visual-foxpro-odbc-driver.md)结合使用，以在语句执行时指定参数数据。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLParamData](../../odbc/reference/syntax/sqlparamdata-function.md) 。
