# <a name="linked-reports-methods"></a>链接报表方法
  可以使用这些方法来创建和管理链接的项。  
  
|方法|操作|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.CreateLinkedItem%2A>|将新的链接项添加到报表服务器数据库。|  
|<xref:ReportService2010.ReportingService2010.GetItemLink%2A>|返回项的名称，其项定义用于指定的链接项。|  
|<xref:ReportService2010.ReportingService2010.ListDependentItems%2A>|返回链接到指定项的项列表。|  
|<xref:ReportService2010.ReportingService2010.SetItemLink%2A>|返回目录项，它用于现有链接项的项定义。|
