# <a name="columnattributesenum"></a>ColumnAttributesEnum
指定[列](../../../ado/reference/adox-api/column-object-adox.md)的特征。  
  
|返回的常量|值|说明|  
|--------------|-----------|-----------------|  
|**adColFixed**|1|列为固定长度。|  
|**adColNullable**|2|列可能包含 null 值。|
