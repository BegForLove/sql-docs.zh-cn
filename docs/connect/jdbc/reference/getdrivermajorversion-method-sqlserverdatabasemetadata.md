# <a name="getdrivermajorversion-method-sqlserverdatabasemetadata"></a>getDriverMajorVersion 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索此 JDBC 驱动程序的主版本号。  
  
## <a name="syntax"></a>语法  
  
```  
  
public int getDriverMajorVersion()  
```  
  
## <a name="return-value"></a>返回值  
 指示 JDBC 驱动程序主版本的 int  。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>备注  
 此 getDriverMajorVersion 方法是由 java.sql.DatabaseMetaData 接口中的 getDriverMajorVersion 方法指定的。
