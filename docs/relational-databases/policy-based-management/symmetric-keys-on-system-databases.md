# <a name="symmetric-keys-on-system-databases"></a>系统数据库中的对称密钥
  此规则检查 master、msdb、model 和 tempdb 数据库中是否存在用户创建的对称密钥。  
  
## <a name="best-practices-recommendations"></a>最佳做法建议  
 请勿在系统数据库中创建对称密钥。
