# <a name="system-databases-in-parallel-data-warehouse-pdw"></a>并行数据仓库中的系统数据库（PDW）
系统数据库是 SQL Server PDW 用来管理用户、数据库和临时数据的数据库。 SQL Server PDW 包括以下系统数据库： master 和 tempdb。
