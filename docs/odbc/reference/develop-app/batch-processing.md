# <a name="batch-processing"></a>批处理
对批处理 SQL 语句的支持并不普遍，因此可互操作的应用程序应该有条件地使用它们，或者根本不使用它们。 有关详细信息，请参阅[执行批处理](../../../odbc/reference/develop-app/executing-batches.md)。
