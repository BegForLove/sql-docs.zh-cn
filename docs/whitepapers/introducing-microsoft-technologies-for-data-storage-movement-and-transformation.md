# <a name="introducing-microsoft-technologies-for-data-storage-movement-and-transformation"></a>用于存储、移动和转换数据的 Microsoft 技术介绍
**摘要：**本文的目的是介绍用于存储、移动和转换数据的 Microsoft 技术。 采用这些技术，组织可以在云中或本地存储及访问用于分析和报告的异类质量数据，从而实现商业智能。  
  
有关 Microsoft 商业智能技术的概述，请参阅 [Microsoft BI 报告和分析工具介绍](https://msdn.microsoft.com/en-us/library/dn655131.aspx)。
