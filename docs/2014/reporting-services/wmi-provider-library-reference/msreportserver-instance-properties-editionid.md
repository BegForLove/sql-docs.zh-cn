# <a name="editionid-property-wmi-msreportserverinstance"></a>EditionID 属性 (WMI MSReportServer_Instance)
  返回报表服务器的版本类别。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>属性值  
 一个`uint`，其中包含的报表服务器的版本。
