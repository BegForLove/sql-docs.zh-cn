# <a name="ado-fundamentals"></a>ADO 基础知识
ADO 提供开发人员的功能强大、 逻辑对象模型来以编程方式访问、 编辑和更新数据的各种通过 OLE DB 系统接口的数据源。 ADO 的最常见用法是查询表或关系数据库中的表、 检索和显示结果的应用程序，并可能使用户能够进行和将更改保存到数据。 其他任务包括：  
  
-   查询数据库使用 SQL 和显示结果。  
  
-   通过 Internet 访问文件存储中的信息。  
  
-   操作的消息和电子邮件系统中的文件夹。  
  
-   正在从数据库的数据保存到 XML 文件。  
  
-   执行与 XML 中所述的命令和检索 XML 流。  
  
-   将数据保存到二进制或 XML 流。  
  
-   允许用户查看和更改数据库表中的数据。  
  
-   创建和重复使用参数化数据库命令。  
  
-   执行存储的过程。  
  
-   动态创建名为的灵活结构**记录集**，若要将保存、 浏览和操作数据。  
  
-   执行事务的数据库操作。  
  
-   筛选和排序基于运行时条件的数据库信息的本地副本。  
  
-   创建和操作数据库中的层次结构结果。  
  
-   将数据库字段绑定到数据识别组件中。  
  
-   创建远程，断开连接**记录集**。  
  
 ADO 公开各种选项和设置为了提供这种灵活性。 因此，务必采用一种有条不紊的方法来学习如何在应用程序，分解每个目标都分成可管理的部分中使用 ADO。
