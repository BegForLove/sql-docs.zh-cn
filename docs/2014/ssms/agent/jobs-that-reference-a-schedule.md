# <a name="jobs-that-reference-a-schedule"></a>引用计划的作业
  使用此对话框可以查看有关引用特定计划的作业的信息。  
  
## <a name="options"></a>选项  
 **“计划”**  
 显示要查看的计划的名称。  
  
 **所选**  
 只读。  
  
 **名称**  
 使用此计划的作业的名称。  
  
 **已启用**  
 只读。 指示当前是否启用了此作业。  
  
 **类别**  
 作业类别。
