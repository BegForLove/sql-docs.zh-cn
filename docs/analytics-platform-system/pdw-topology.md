# <a name="pdw-topology"></a>PDW 拓扑
显示有关 SQL Server PDW 组件的名称和网络信息。 使用**设备网络配置拓扑**页后，可以更改的网络 IP 地址。  
  
展开此节可配置适用于整个设备的信息的元素。 选择要配置适用于仅设备该区域的信息的并行数据仓库拓扑或 HDInsight 拓扑节。
