# 第 2 课：配置数据库对象的权限
授予用户访问数据库的权限涉及三个步骤。 首先，创建登录名。 使用登录名，用户可以连接到[!INCLUDE[ssDEnoversion](../includes/ssdenoversion-md.md)]。 然后将登录名配置为指定数据库中的用户。 最后，授予该用户访问数据库对象的权限。 本课介绍了这三个步骤，并介绍了如何将视图和存储过程创建为对象。  
  
本课程包含以下主题：

-   [授予访问数据库的权限](../t-sql/granting-access-to-a-database.md)  
  
-   [创建视图和存储过程](../t-sql/creating-views-and-stored-procedures.md)  
  
-   [授予访问数据库对象的权限](../t-sql/granting-access-to-a-database-object.md)  
  
-   [摘要：配置数据库对象的权限](../t-sql/summary-configuring-permissions-on-database-objects.md)
