# <a name="getting-data"></a>获取数据
[ADO 基础知识](../../../ado/guide/data/ado-fundamentals.md)，和[HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md)示例具体而言，引入中创建 ADO 应用程序涉及四个主要操作： 获取数据、 正在检查的数据、 编辑数据和更新数据。 本部分讨论了更多详细信息中的获取数据。  
  
 在基本级别中，有几个 ADO 对象参与获取数据的操作。 你必须首先连接到数据源使用 ADO**连接**对象。 然后将指令传递到数据源使用 ADO**命令**对象。 最后，你最常接收数据 ADO**记录集**对象。  
  
 本部分包含以下主题。  
  
-   [连接到数据源](../../../ado/guide/data/connecting-to-data-sources.md)  
  
-   [准备和执行命令](../../../ado/guide/data/preparing-and-executing-commands.md)  
  
-   [接收结果](../../../ado/guide/data/receiving-results.md)
