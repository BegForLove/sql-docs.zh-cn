# <a name="sqlserverblob-class"></a>SQLServerBlob 类
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

 **包：** com.microsoft.sqlserver.jdbc  
  
 **扩展：** java.lang.Object  
  
 **实现：** java.sql.Blob、 java.io.Serializable  
  
## <a name="syntax"></a>语法  
  
```  
  
public class SQLServerBlob  
```  
  
## <a name="remarks"></a>注释
