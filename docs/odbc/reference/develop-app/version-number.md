# <a name="version-number"></a>版本号
有多个版本的 ODBC，每个都有不同的功能。 应用程序确定驱动程序管理器的 ODBC 版本和特定的驱动程序支持通过调用**SQLGetInfo**使用 SQL_ODBC_VER 和 SQL_DRIVER_ODBC_VER 选项。
