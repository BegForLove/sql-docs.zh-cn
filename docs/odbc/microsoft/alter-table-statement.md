# <a name="alter-table-statement"></a>ALTER TABLE 语句
使用 Microsoft Access、 dBASE 或 Paradoxdriver 时，使用添加和删除表约束定义子句支持 ALTER TABLE 语句。 （ALTER TABLE 语句不支持的 Microsoft Excel 或文本文件驱动程序。）  
  
> [!NOTE]  
>  当您使用 Paradox 驱动程序而无需实现 borland 公司数据库引擎时，不支持 ALTER TABLE 语句;仅读取和追加允许使用的语句。
