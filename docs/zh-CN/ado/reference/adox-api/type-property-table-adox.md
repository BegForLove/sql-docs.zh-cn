# <a name="type-property-table-adox"></a>类型属性 （表） (ADOX)
指示表的类型。  
  
## <a name="return-values"></a>返回值  
 返回**字符串**值，该值指定的表; 例如，"TABLE"、"系统表"或"全局临时"的类型。  
  
## <a name="remarks"></a>注释  
 该属性为只读。
