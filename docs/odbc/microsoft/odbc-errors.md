# <a name="odbc-errors"></a>ODBC 错误
当发生错误时，Microsoft ODBC 桌面数据库驱动程序返回本机错误号、 SQLSTATE （ODBC 错误代码） 和一条错误消息。  
  
 本部分包含以下主题。  
  
-   [本机错误](../../odbc/microsoft/native-error.md)  
  
-   [SQLSTATE](../../odbc/microsoft/sqlstate.md)  
  
-   [错误消息](../../odbc/microsoft/odbc-jet-error-messages.md)  
  
-   [算术错误](../../odbc/microsoft/arithmetic-errors.md)
