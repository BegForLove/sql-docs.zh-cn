# <a name="steps-element-xmla"></a>Steps 元素 (XMLA)
  指定的步骤使用的最大数[DesignAggregations](../xml-elements-commands/designaggregations-element-xmla.md)命令用于设计聚合。  
  
## <a name="syntax"></a>语法  
  
```xml  
  
<DesignAggregations>  
   ...  
   <Steps>...</Steps>  
   ...  
</DesignAggregations>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|Description|  
|--------------------|-----------------|  
|数据类型和长度|Integer|  
|默认值|None|  
|基数|0-1：可出现一次且仅出现一次的可选元素。|  
  
## <a name="element-relationships"></a>元素关系  
  
|关系|元素|  
|------------------|-------------|  
|父元素|[DesignAggregations](../xml-elements-commands/designaggregations-element-xmla.md)|  
|子元素|None|
