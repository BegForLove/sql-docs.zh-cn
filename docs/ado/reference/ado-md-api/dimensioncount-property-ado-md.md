# <a name="dimensioncount-property-ado-md"></a>DimensionCount 属性 (ADO MD)
指示[轴](../../../ado/reference/ado-md-api/axis-object-ado-md.md)上的维度数。  
  
## <a name="return-values"></a>返回值  
 返回一个**长**整型，并且是只读的。
