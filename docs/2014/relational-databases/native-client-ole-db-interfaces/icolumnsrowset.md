# <a name="icolumnsrowset"></a>IColumnsRowset
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client 向 IColumnsRowset::GetColumnRowset 添加了 DBCOLUMN_BASETABLEINSTANCE 列。 此列返回 DBTYPE_I2，以及由 Microsoft 保留供使用。 此列中的信息将在将来的版本中进行修改。
