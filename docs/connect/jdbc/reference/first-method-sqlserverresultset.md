# <a name="first-method-sqlserverresultset"></a>第一种方法 (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  将光标移到的第一行[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
public boolean first()  
```  
  
## <a name="return-value"></a>返回值  
 **true**如果将光标移动到第一行。 否则为 **false**。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 此第一种方法被指定 java.sql.ResultSet 接口中的第一个方法。
