# <a name="select-measures-cube-wizard"></a>选择度量值（多维数据集向导）
  可以使用 **“选择度量值”** 页选择要创建的度量值组和度量值。  
  
## <a name="options"></a>“常规”  
 **度量值**  
 显示可用的度量值组及其度量值。  
  
 对于某个度量值组或度量值，若要指示将要创建它，则请选中其复选框。 默认情况下，所有度量值组和度量值都是选中的。  
  
 若要更改某个度量值组或度量值的名称，请单击其名称，然后输入新名称。
