# <a name="cstring-class"></a>CString 类
由于 Microsoft 中的**CString**类的对象® Visual C++®已签名，并且 ODBC 函数中的字符串参数没有签名，因此将**CString**对象传递给 ODBC 函数而不强制转换它们的应用程序将收到编译器警告。
