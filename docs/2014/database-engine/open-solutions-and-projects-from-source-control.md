# <a name="open-solutions-and-projects-from-source-control"></a>从源代码管理打开的解决方案和项目
  你可以使用[!INCLUDE[msCoName](../includes/msconame-md.md)][!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)]直接从源代码管理中打开的解决方案或项目。 从源代码管理中打开解决方案时，[!INCLUDE[ssManStudio](../includes/ssmanstudio-md.md)] 环境将自动检索该解决方案的最新版本并将其复制到指定的位置。  
  
 如果没有解决方案的本地副本，则必须首先将其从源代码管理中打开，然后才可以对其执行源代码管理。 在检索解决方案的本地副本后，你不需要再次为从源代码管理打开你可以打开本地副本，然后使用标准源代码控制命令，如**获取最新版本**或**签出**，以执行所需的操作。
