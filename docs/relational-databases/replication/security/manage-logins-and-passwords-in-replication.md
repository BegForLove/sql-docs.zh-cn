# 管理复制中的登录名和密码
  在配置复制时，为复制代理指定登录名和密码。 配置复制后，可以更改登录名和密码。 有关详细信息，请参阅 [View and Modify Replication Security Settings](../../../relational-databases/replication/security/view-and-modify-replication-security-settings.md)。 如果更改复制代理使用的帐户的密码，请执行 [sp_changereplicationserverpasswords & #40;Transact SQL & #41;](../../../relational-databases/system-stored-procedures/sp-changereplicationserverpasswords-transact-sql.md)。
