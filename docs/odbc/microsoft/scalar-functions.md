# <a name="scalar-functions"></a>标量函数
Microsoft ODBC 桌面数据库驱动程序支持以下标量函数：  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|USER|  
|DAYOFMONTH|MOD|年|  
|DAYOFWEEK|月||  
  
 有关标量函数的参数和返回值的信息，请参阅*ODBC 程序员参考*的附录 F。
