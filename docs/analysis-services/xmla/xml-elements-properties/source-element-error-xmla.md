# <a name="source-element-error-xmla"></a>Source 元素 (Error) (XMLA)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../../includes/ssas-appliesto-sqlas-aas.md)]
  包含生成父组件的名称[错误](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)元素。  
  
## <a name="syntax"></a>语法  
  
```xml  
  
<Error>  
   ...  
   <Source>...</Source>  
   ...  
</Error>  
```  
  
## <a name="element-characteristics"></a>元素特征  
  
|特征|说明|  
|--------------------|-----------------|  
|数据类型和长度|字符串|  
|默认值|无|  
|基数|0-1：可出现一次且仅出现一次的可选元素。|  
  
## <a name="element-relationships"></a>元素关系  
  
|关系|元素|  
|------------------|-------------|  
|父元素|[错误](../../../analysis-services/xmla/xml-elements-properties/error-element-xmla.md)|  
|子元素|无|
