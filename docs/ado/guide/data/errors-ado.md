# <a name="errors-ado"></a>错误 (ADO)
任何涉及 ADO 对象的操作可以生成一个或多个提供程序错误。 每个错误发生时，一个或多个**错误**对象都将置于**错误**的集合**连接**对象。 有关处理警告和 ADO 应用程序中的错误详细信息，请参阅[的错误处理](../../../ado/guide/data/error-handling.md)。  
  
 可以通过单独的机制引发应用程序错误。 例如，在 Visual Basic **Err**对象将包含应用程序级别的错误。
