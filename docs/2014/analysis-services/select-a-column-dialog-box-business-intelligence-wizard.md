# <a name="select-a-column-dialog-box-business-intelligence-wizard"></a>“选择列”对话框（商业智能向导）
  可以使用 **“选择列”** 对话框从树视图显示的表列表中选择列。 通过商业智能向导可以打开此对话框。  
  
## <a name="options"></a>选项  
 **树视图**  
 单击一个列可以为商业智能向导选择该列。
