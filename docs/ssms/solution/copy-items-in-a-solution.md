# <a name="copy-items-in-a-solution"></a>复制解决方案中的项
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 可以使用解决方案资源管理器或 Windows 资源管理器来复制项。  
  
### <a name="to-copy-items-within-solution-explorer"></a>在解决方案资源管理器中复制项  
  
1.  在解决方案资源管理器中，选择想要复制的项。  
  
2.  在“编辑”菜单上，单击“复制”。  
  
3.  在解决方案资源管理器中，选择目标项目。  
  
4.  在“编辑” 菜单上，单击“粘贴”。  
  
> [!NOTE]  
> 不能在项目之间复制连接。 将具有关联连接的查询复制到另一个项目时，其关联的连接不会复制到目标项目中。 必须在目标项目中手动创建连接。
