# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
指定在文本[流](../../../ado/reference/ado-api/stream-object-ado.md)对象中用作行分隔符的字符。  
  
|返回的常量|值|说明|  
|--------------|-----------|-----------------|  
|**adCR**|13|指示回车符。|  
|**adCRLF**|-1|默认。 指示回车换行符。|  
|**adLF**|10|指示换行。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 等效项  
 这些常量没有 ADO/WFC 等效项。
