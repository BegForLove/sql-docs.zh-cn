# <a name="odbc-component-files"></a>ODBC 组件文件
ODBC 驱动程序文件所需的 ODBC 桌面数据库驱动程序 4.0 的每个组件。  
  
 某些文件所需的多个组件。 如果你想要重新分发任何 ODBC 桌面数据库驱动程序与商业应用程序，必须重新分发这些文件。  
  
 对于 Windows 95 或更高版本，在 \Windows\System 目录中或 \Windows\System32 目录 Windows NT 3.51 或更高版本和 Windows 2000 中安装这些文件。 如果 ODBC 文件以前安装在不同的目录，请确保使用较新的 \Windows\System （或 System32） 中的文件目录。
