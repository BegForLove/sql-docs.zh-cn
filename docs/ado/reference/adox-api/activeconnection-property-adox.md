# <a name="activeconnection-property-adox"></a>ActiveConnection 属性 (ADOX)
指示 ADO[连接](../../../ado/reference/ado-api/connection-object-ado.md)到对象[目录](../../../ado/reference/adox-api/catalog-object-adox.md)所属。  
  
## <a name="settings-and-return-values"></a>设置和返回值  
 集**连接**对象或**字符串**包含连接的定义。 返回活动**连接**对象。  
  
## <a name="remarks"></a>备注  
 默认值为空对象引用。  
  
## <a name="applies-to"></a>适用范围  
 [目录对象 (ADOX)](../../../ado/reference/adox-api/catalog-object-adox.md)
