# <a name="smo-programming-languages"></a>SMO 编程语言
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

  本部分介绍如何在 Visual [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[msCoName](../../includes/msconame-md.md)] C# .net 中对管理对象 (SMO) 进行编程。
  
|主题|描述|  
|-----------|-----------------|  
|[Visual C&#35; .net 中的入门](../../relational-databases/server-management-objects-smo/smo-programming-getting-started-in-visual-csharp-net.md)|提供有关如何在 Visual C# .net 中开始编写 SMO 程序的信息|  
  
 SMO 是一个 .NET 程序集。 若要使用 [!INCLUDE[msCoName](../../includes/msconame-md.md)] .NET Framework 编程，必须选择一种公共语言运行时支持的语言。 在 SMO 引用中, 有适用于 Visual C# .net 等托管语言的语法。
