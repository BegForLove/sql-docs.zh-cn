# <a name="parameter-markers-in-procedure-calls"></a>过程调用中的参数标记
调用接受参数的过程时，可互操作的应用程序应使用参数标记而不是文本参数值。 某些数据源不支持在过程调用中使用文本参数值。 有关参数的详细信息，请参阅[语句参数](../../../odbc/reference/develop-app/statement-parameters.md)。 有关调用过程的详细信息，请参阅本节后面的[过程调用](../../../odbc/reference/develop-app/procedure-calls.md)。
