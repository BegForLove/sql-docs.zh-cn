# <a name="configurationsetting-property---pathname"></a>ConfigurationSetting 属性 - PathName
  指定报表服务器实例的安装路径。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## <a name="property-values"></a>属性值  
 表示安装路径的 **String** 对象。
