# <a name="sqlstatistics"></a>SQLStatistics
  **SQLStatistics**可以执行对静态游标。 尝试执行**SQLStatistics**上更改 SQL_SUCCESS_WITH_INFO，该值指示游标类型可更新 （键集驱动或动态） 返回。
