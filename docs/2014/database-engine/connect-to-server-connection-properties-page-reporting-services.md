# <a name="connect-to-server-connection-properties-page-reporting-services"></a>连接到服务器（“连接属性”页）(Reporting Services)
  当连接到 [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] 或在“已注册的服务器”[!INCLUDE[ssRS](../includes/ssrs.md)]**中注册 ** 时，使用此选项卡可以查看或指定选项。 **连接**和**选项**仅在连接时显示在此对话框中。 注册**并仅**在注册**** [!INCLUDE[ssRS](../includes/ssrs.md)]时显示在此对话框中。  
  
## <a name="options"></a>选项  
 **连接超时值(毫秒)**  
 输入在超时之前等待建立连接的毫秒数。  
  
 **始终使用 SSL**  
 使用安全套接字层进行连接。  
  
 **连接**  
 使用列出的值尝试连接。  
  
 **选项**  
 单击此项可更改对话框并隐藏其他服务器连接选项，例如记住密码。  
  
 **考试**  
 在“已注册的服务器”[!INCLUDE[ssRS](../includes/ssrs.md)]**中注册 ** 时，单击此选项可以测试连接。  
  
 **把**  
 保存“已注册的服务器”**** 中的设置。
