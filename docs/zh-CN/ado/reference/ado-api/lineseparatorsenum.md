# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
指定用作文本中的行分隔符的字符[流](../../../ado/reference/ado-api/stream-object-ado.md)对象。  
  
|常量|“值”|Description|  
|--------------|-----------|-----------------|  
|**adCR**|13|表示回车。|  
|**adCRLF**|-1|默认值。 指示回车换行符。|  
|**adLF**|10|指示换行。|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC 等效项  
 这些常量没有 ADO/WFC 等效项。
