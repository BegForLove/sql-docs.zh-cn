# <a name="marshaloptionsenum"></a>MarshalOptionsEnum
指定应返回到服务器的记录。  
  
|一直|值|说明|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|默认值。 将所有行返回到服务器。|  
|**adMarshalModifiedOnly**|1|仅返回服务器上已修改的行。|
