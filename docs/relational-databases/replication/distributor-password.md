# 分发服务器密码
  如果上 **出版商** 页上的此向导中，启用一个或多个要使用此服务器作为远程分发服务器的发布者，则必须指定在发布服务器和远程分发服务器上使用之间建立的连接复制的密码 **distributor_admin** 登录名。 在新建发布向导或配置分发向导的 **“管理密码”** 页上，必须为每个使用此远程分发服务器的发布服务器输入相同的密码。 分发服务器安全性的更多信息，请参阅 [安全分发服务器](../../relational-databases/replication/security/secure-the-distributor.md)。  
  
## 选项  
 **密码**  
 为发布服务器和远程分发服务器之间的连接输入强密码。  
  
 **确认密码**  
 重新输入密码以确认密码输入正确。
