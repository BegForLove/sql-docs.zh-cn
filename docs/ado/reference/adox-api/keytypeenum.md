# <a name="keytypeenum"></a>KeyTypeEnum
指定[键](../../../ado/reference/adox-api/key-object-adox.md)的类型： primary、foreign 或 unique。  
  
|返回的常量|值|说明|  
|--------------|-----------|-----------------|  
|**adKeyPrimary**|1|默认。 密钥是主键。|  
|**adKeyForeign**|2|该键是外键。|  
|**adKeyUnique**|3|密钥是唯一的。|
