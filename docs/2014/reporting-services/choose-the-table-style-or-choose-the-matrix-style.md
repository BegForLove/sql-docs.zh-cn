# <a name="choose-the-table-style-or-choose-the-matrix-style"></a>选择表样式或选择矩阵样式
  使用报表向导的这些页可为报表选择文本和线条样式。 样式列表基于可配置的模板。 报表设计器提供了下列初始模板：静态图像、 林、 公司、 粗体、 海洋和泛型。  
  
 您还可以添加其他样式模板。
