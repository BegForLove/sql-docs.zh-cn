# <a name="sqlsetpos-visual-foxpro-odbc-driver"></a>SQLSetPos（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含 Visual FoxPro ODBC 特定于驱动程序的信息。 有关此功能的一般信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)下的相应主题。  
  
 支持： 完整  
  
 ODBC API 符合性：2 级  
  
 设置行集中的光标位置。 您可以将游标定位到行集中的特定行后，将**SQLSetPos**与[SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md)一起从未绑定列中检索行。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLSetPos。](../../odbc/reference/syntax/sqlsetpos-function.md)
