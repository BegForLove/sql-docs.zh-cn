# <a name="job-propertiesnew-job-steps-page"></a>作业属性： 新的作业 （步骤页）
  使用此页可以查看和组织 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 代理作业的作业步骤。  
  
## <a name="options"></a>“常规”  
 **作业步骤列表**  
 列出此作业的作业步骤。  
  
 **移动步骤**  
 在列表中将作业上移或下移一个步骤。  
  
 **开始步骤**  
 选择在作业开始时 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 代理开始的步骤。  
  
 **新建**  
 在所选作业步骤下创建新的作业步骤。  
  
 **Insert**  
 在所选作业步骤上方创建新的作业步骤。  
  
 **编辑**  
 更改所选作业步骤。  
  
 **删除**  
 删除所选作业步骤。 删除作业步骤时，将自动删除其输出日志。
