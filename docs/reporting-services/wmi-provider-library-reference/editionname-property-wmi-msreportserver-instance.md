# EditionName 属性 (WMI MSReportServer_Instance)
  返回报表服务器版本类别的可读名称。 只读。  
  
## 语法  
  
```vb  
Public Dim EditionName As String  
```  
  
```csharp  
public string EditionName;  
```  
  
## 属性值  
 一个包含报表服务器版本的可读名称的 **string** 。
