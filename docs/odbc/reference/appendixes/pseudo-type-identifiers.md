# <a name="pseudo-type-identifiers"></a>伪类型标识符
应用程序方便地用于编程，ODBC 定义的伪类型标识符的数目。 这些标识符实际上不对应于实际数据类型，但相反，具体取决于这种情况，解决对现有数据类型。

-   [SQL_ARD_TYPE](../../../odbc/reference/appendixes/sql-ard-type.md)  
  
-   [SQL_C_TCHAR](../../../odbc/reference/appendixes/sql-c-tchar.md)
