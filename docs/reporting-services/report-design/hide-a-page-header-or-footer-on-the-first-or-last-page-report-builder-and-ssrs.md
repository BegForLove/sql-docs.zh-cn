# <a name="hide-a-page-header-or-footer-on-the-first-or-last-page-report-builder-and-ssrs"></a>隐藏第一页或最后一页的页眉或页脚（报表生成器和 SSRS）
  报表可以包含页眉和页脚，它们分别位于每一页的顶部和底部。 添加页眉或页脚后，可以在报表的第一页和最后一页选择性地将其隐藏。  
  
> [!NOTE]  
>  [!INCLUDE[ssRBRDDup](../../includes/ssrbrddup-md.md)]  
  
### <a name="to-hide-a-page-header-on-the-first-or-last-page"></a>隐藏第一页或最后一页的页眉  
  
1.  在“设计”视图中打开报表。  
  
2.  右键单击页面页眉，再单击“页眉属性”。 将打开 **“报表表头属性”** 对话框。  
  
3.  确保未选中 **“显示此报表的页眉”** 。  
  
4.  在 **“打印选项”** 部分，清除每个选项对应的复选框以隐藏报表第一页或最后一页上的显示。  
  
5.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
### <a name="to-hide-a-page-footer-on-the-first-or-last-page"></a>隐藏第一页或最后一页的页脚  
  
1.  在“设计”视图中打开报表。  
  
2.  右键单击页面页脚，再单击“页脚属性”。 将打开 **“报表表尾属性”** 对话框。  
  
3.  确保未选中 **“显示此报表的页脚”** 。  
  
4.  在 **“打印选项”** 部分，清除每个选项对应的复选框以隐藏报表第一页或最后一页上的显示。  
  
5.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]
