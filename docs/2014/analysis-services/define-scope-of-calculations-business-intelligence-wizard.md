# <a name="define-scope-of-calculations-business-intelligence-wizard"></a>定义计算作用域（商业智能向导）
  可以使用 **“定义计算作用域”** 页将新的时间计算与各度量值相关联。  
  
## <a name="options"></a>“常规”  
 **可用度量值**  
 选择时间计算所适用的度量值。  
  
 **全选**  
 选择“可用度量值”中列出的所有度量值。  
  
 **全部清除**  
 清除“可用度量值”中列出的所有度量值。
