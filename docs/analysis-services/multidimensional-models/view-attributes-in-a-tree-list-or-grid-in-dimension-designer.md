# 在维度设计器中查看树、列表或网格中的属性
  此过程介绍如何更改属性在维度设计器中的显示方式。  
  
### 在维度设计器中更改“属性”窗格的格式  
  
1.  在维度设计器中，打开包含要使用的属性的维度，再单击“维度结构”选项卡。  
  
2.  在“属性”窗格中的任意位置单击鼠标，将焦点设置到“属性”窗格。  
  
3.  单击“维度”菜单，或右键单击“属性”窗格。  
  
4.  指向 **“属性显示方式”**，然后单击 **“树”**、 **“网格”**或 **“列表”**。
