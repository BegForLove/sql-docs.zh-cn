# <a name="welcome-to-the-report-wizard"></a>欢迎使用报表向导
  使用报表向导的此页可以查看要执行的任务的摘要信息。  
  
## <a name="options"></a>选项  
 **不再显示此页**  
 选择此选项可以在下次启动向导时跳过此页。
