# <a name="ready-to-install"></a>准备安装
  “准备安装”页提供了 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 安装程序要安装或更改的功能的摘要。  
  
## <a name="options"></a>选项  
 此页上有下列选项：  
  
-   若要继续安装，请单击 **“安装”** 。  
  
-   若要更改安装设置，请单击 **“上一步”** 。  
  
-   若要退出安装程序，请单击 **“取消”** 。
