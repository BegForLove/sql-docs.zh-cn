# 多播转换编辑器
  可以使用 **“多播转换编辑器”** 对话框查看和设置每个转换输出的属性。  
  
 若要了解有关多播转换的详细信息，请参阅 [Multicast Transformation](../../../integration-services/data-flow/transformations/multicast-transformation.md)。  
  
## 选项  
 **输出**  
 在左侧选择输出可以在右侧的表中查看其属性。  
  
 **属性**  
 除了“名称”和“说明”外，所有列出的输出属性都是只读的。
