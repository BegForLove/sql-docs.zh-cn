# <a name="msreportserverinstance-properties---editionid"></a>MSReportServer_Instance 属性-EditionID
  返回报表服务器的版本类别。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>属性值  
 包含报表服务器版本类别的 **uint** 值。
