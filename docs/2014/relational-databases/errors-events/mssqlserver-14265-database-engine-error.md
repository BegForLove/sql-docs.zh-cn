# <a name="mssqlserver14265"></a>MSSQLSERVER_14265
    
## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|14265|  
|事件源|MSSQLSERVER|  
|组件|SQLEngine|  
|符号名称|SQLErrorNum14265|  
|消息正文|MSSQLServer 服务意外终止。 有关可能的原因，请查看 SQL Server 错误日志以及 Windows 系统和应用程序事件日志。|  
  
## <a name="explanation"></a>解释
