# <a name="version-property-ado"></a>版本属性 (ADO)
指示 ADO 版本号。  
  
## <a name="return-value"></a>返回值  
 返回**字符串**值，该值指示版本。  
  
## <a name="remarks"></a>注释  
 使用**版本**属性以返回 ADO 实现的版本号。  
  
 提供程序的版本将用作中的动态属性[属性](../../../ado/reference/ado-api/properties-collection-ado.md)集合。
