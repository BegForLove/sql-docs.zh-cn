# <a name="osql-no-longer-supports-the-ed-and--commands"></a>osql 不再支持 ED 和 !! 命令
  **Osql**实用工具不支持**ED**和 **!!** 命令。  
  
## <a name="corrective-action"></a>纠正措施  
 删除对引用**ED**和 **!!** 命令的所有引用。  
  
 如果你想要使用**ED**和 **!!** 命令，使用**sqlcmd**而不是实用工具**osql**。
