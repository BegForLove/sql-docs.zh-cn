# <a name="configurationsetting-property---databasename"></a>ConfigurationSetting 属性 - DatabaseName
  指定承载报表服务器数据库的 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 实例的名称。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim DatabaseName As String  
```  
  
```csharp  
public string DatabaseName;  
```  
  
## <a name="property-values"></a>属性值  
 **String** 对象，其值表示报表服务器数据库的数据库实例名称。
