# <a name="drop-table-statement-limitations"></a>删除表语句的限制
当使用 Microsoft Excel 5.0、 7.0 或 97 驱动程序时，DROP TABLE 语句将清除工作表，但不会删除工作表的名称。 因为工作表的名称仍存在于工作簿，不能具有相同名称创建另一个工作表。
