# <a name="sqlfreeconnect-mapping"></a>SQLFreeConnect 映射
当应用程序调用**SQLFreeConnect**通过 ODBC 3 *.x*驱动程序，将会调用  
  
```  
SQLFreeConnect(hdbc)   
```  
  
 映射到  
  
```  
SQLFreeHandle(SQL_HANDLE_DBC,Handle)  
```  
  
 与*处理*参数设置为中的值*hdbc*。
