# <a name="errors-and-warnings-event-category"></a>错误和警告事件类别
  错误和警告事件类别具有下表中说明的事件类。  
  
|Event Class|事件 ID|Description|  
|-----------------|--------------|-----------------|  
|错误|17|记录自从启动跟踪以来的所有新错误事件。|  
  
 有关与每个错误和警告事件类关联的列的信息，请参阅 [Errors and Warnings Events Data Columns](errors-and-warnings-events-data-columns.md)。
