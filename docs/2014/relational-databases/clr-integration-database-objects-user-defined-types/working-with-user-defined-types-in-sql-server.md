# <a name="working-with-user-defined-types-in-sql-server"></a>使用 SQL Server 中的用户定义类型
  你可以访问中的用户定义类型 (UDT) 功能[!INCLUDE[msCoName](../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]从[!INCLUDE[tsql](../../includes/tsql-md.md)]语言可以通过使用常规查询语法。 UDT 可以用在数据库对象的定义中，作为 [!INCLUDE[tsql](../../includes/tsql-md.md)] 批、函数和存储过程中的变量，以及作为函数和存储过程中的参数。
