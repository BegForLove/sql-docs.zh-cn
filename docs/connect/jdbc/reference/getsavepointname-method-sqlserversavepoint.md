# <a name="getsavepointname-method-sqlserversavepoint"></a>getSavepointName 方法 (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  获取保存点的名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
public java.lang.String getSavepointName()  
```  
  
## <a name="return-value"></a>返回值  
 包含保存点名称的 String  。  
  
## <a name="exceptions"></a>例外  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>备注  
 此 getSavepointName 方法是由 java.sql.Savepoint 接口中的 getSavepointName 方法指定的。
