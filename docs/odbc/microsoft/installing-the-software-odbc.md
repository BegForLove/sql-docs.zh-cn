# <a name="installing-the-software-odbc"></a>安装软件 (ODBC)
> [!IMPORTANT]  
>  此功能将在将来版本的 Windows 中删除。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 而是使用 Oracle 提供的 ODBC 驱动程序。  
  
 Oracle 的 ODBC 驱动程序是数据访问组件之一。 它附带其他 ODBC 组件，如 ODBC 数据源管理员，并且应该已经安装。 该驱动程序也可以在微软产品支持服务在线网站上找到[www.microsoft.com](https://www.microsoft.com)下的"驱动程序和其他下载"。  
  
 网络软件必须根据自己的文档进行安装。 Oracle 的 ODBC 驱动程序不需要特殊的安装注意事项，只要支持网络软件。  
  
 Oracle 软件必须根据自己的文档进行安装。 Oracle 的 ODBC 驱动程序通常不需要特殊的安装注意事项，只要驱动程序支持该版本。 但是，要保持产品兼容，请最后安装 Oracle 的 ODBC 驱动程序，以确保您拥有最新版本的驱动程序。 Oracle 维护一个公共 FTP 站点，在该网站中，它向 Oracle 服务器产品和服务器产品附带的客户端组件发布修补程序。 这些修补程序是几个 Microsoft 产品和技术正常运行所必需的。 有关此站点的详细信息，请参阅[Oracle 软件修补程序](../../odbc/microsoft/oracle-software-patches.md)。  
  
> [!CAUTION]  
>  通过 MDAC/Windows DAC 安装 Oracle 软件可能会覆盖 MDAC 的当前版本。 如果使用 ODBC 组件出现问题，请重新安装 MDAC。
