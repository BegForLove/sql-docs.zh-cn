# <a name="numeric-literals"></a>数值
当数字数据值存储在字符串中时，将使用数字文本。 对于将数字 SQL 数据转换为SQL_C_CHAR字符串，或将数字 C 数据转换为SQL_CHAR或SQL_VARCHAR字符串，使用数字文本语法指定存储在目标中的内容。 对于将作为SQL_C_CHAR字符串存储的数字转换为数字 SQL 数据，或将作为SQL_CHAR字符串存储的数字转换为数字 C 数据，此语法用于验证存储在源中的内容。  
  
 数字文本应符合附录 C 中["数字文本语法](../../../odbc/reference/appendixes/numeric-literal-syntax.md)"部分中定义的语法：SQL 语法。
