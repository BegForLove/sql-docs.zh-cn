# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>使用 Visual FoxPro 数据在 Microsoft Word 中创建邮件标签
您可以在适用于 Windows 95 或 Windows 98 文档中的 Microsoft Word 中使用 Visual FoxPro 数据。 例如，您可能希望从存储在 Visual FoxPro 表中的客户信息创建邮件标签。  
  
### <a name="to-create-mailing-labels"></a>创建邮件标签  
  
1.  在 Microsoft Word 中，创建新的空白文档。  
  
2.  在"工具"菜单中，选择"邮件合并"。  
  
3.  在"邮件合并帮助程序"中，选择"创建"，然后选择"邮件标签"。  
  
4.  在"主文档"下，选择"活动窗口"。  
  
5.  在"数据源"下，选择"获取数据"，然后选择"打开的数据源"。  
  
6.  在"开源"对话框中，选择 MS 查询。  
  
7.  在"选择数据源"对话框中，选择 Visual FoxPro 数据源，然后单击"使用"。  
  
8.  如果数据源访问的数据库包含表，请从"添加表"对话框中选择一个表。 Microsoft 查询在查询设计器的上半部分显示添加的表。  
  
9. 通过将字段从表拖动到设计器的下半部分，为查询选择字段。  
  
10. 从"文件"菜单中，选择将数据返回到 Microsoft Word。 Microsoft 查询将关闭，您选择的数据可用于邮件合并文档。  
  
11. 在主文档下，选择"设置"。  
  
12. 在"标签选项"对话框中，选择所需的打印机和标签信息，然后单击"确定"。  
  
13. 在"创建标签"对话框中，选择要在邮件标签上打印的字段，然后单击"确定"。  
  
14. 在"邮件合并帮助程序"中，在"将数据与文档合并"下，单击"合并"。  
  
15. 在"合并"对话框中，选择所需的选项，然后单击"合并"。
