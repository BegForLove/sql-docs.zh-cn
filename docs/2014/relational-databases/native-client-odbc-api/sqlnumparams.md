# <a name="sqlnumparams"></a>SQLNumParams
  以开头的数据库引擎中的改进[!INCLUDE[ssSQL11](../../includes/sssql11-md.md)]允许 SQLNumParams 以获取更准确的预期结果的说明。 这些更准确的结果可能与在以前版本的 SQLNumParams 返回的值不同[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]。  
  
 有关详细信息，请参阅[元数据发现](../native-client/features/metadata-discovery.md)。
