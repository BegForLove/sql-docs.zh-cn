# <a name="error-messages-odbc-driver-for-oracle"></a>错误消息（Oracle ODBC 驱动程序）
> [!IMPORTANT]  
>  此功能将 Windows 的未来版本中删除。 请避免在新的开发工作中使用该功能，并着手修改当前还在使用该功能的应用程序。 相反，使用提供的 Oracle 的 ODBC 驱动程序。  
  
 出现错误时，适用于 Oracle 的 Microsoft ODBC 驱动程序将返回 SQLSTATE （ODBC 错误代码） 和一条错误消息。 从驱动程序检测到的错误和 Oracle Server 返回的错误，该驱动程序派生此信息。
