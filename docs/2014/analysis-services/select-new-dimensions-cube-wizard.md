# <a name="select-new-dimensions-cube-wizard"></a>选择新维度（多维数据集向导）
  可以使用 **“选择新维度”** 页选择要创建的新维度。  
  
## <a name="options"></a>“常规”  
 **Dimensions**  
 选择要创建的维度。 默认情况下，会选中所有维度。  
  
 树的根节点表示将要创建的新维度。 用于创建维度的相关表显示在维度节点下。  
  
 若要重命名维度，请单击维度名称，然后输入新名称。 您不能重命名表。
