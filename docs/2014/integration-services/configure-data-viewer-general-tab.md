# <a name="configure-data-viewer-general-tab"></a>配置数据查看器（“常规”选项卡）
  在包执行过程中，数据查看器显示流过连接两个数据流组件的路径的数据。  
  
 使用 **“配置数据查看器”** 对话框中的 **“常规”** 选项卡可以命名数据查看器和指定显示数据所用的图表类型。  
  
## <a name="options"></a>“常规”  
 **名称**  
 为数据查看器提供唯一的名称。  
  
 **类型**  
 列出数据查看器的类型。 单击类型可以显示示例图表。  
  
 **Description**  
 显示示例图表，后跟说明。
