登录名为用户提供了连接到 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]的权限。 用户是可以访问特定数据库的登录名。 使用 GRANT 语句可授予用户读取数据以及访问和更改数据的权限。
