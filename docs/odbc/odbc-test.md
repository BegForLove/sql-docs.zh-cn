# <a name="odbc-test"></a>ODBC 测试
Microsoft® ODBC 测试是 ODBC 启用应用程序可用于测试的 ODBC 驱动程序和 ODBC 驱动程序管理器。 ODBC 3.51 包括 ANSI 和 Unicode 启用版本的 ODBC 测试。 相应的文件如下所示：  
  
-   Odbcte32.exe 和 Gtrtst32.dll，ANSI 版本。  
  
-   Odbct32w.exe 和 Gtrts32w.dll，Unicode 版本。  
  
 若要使用 ODBC 测试，必须了解 ODBC API、 C 语言和 SQL。 有关 ODBC API 的详细信息，请参阅[ODBC 程序员参考](../odbc/reference/odbc-programmer-s-reference.md)。  
  
 现在，已以前包含在文档的此部分的帮助主题都包含在 ODBC 测试程序。 打开 Odbcte32.exe 或 Odbct32w.exe，打开**帮助**菜单，，然后单击**帮助主题**。  
  
 请注意，这些应用程序，用于 64 位 Microsoft Windows 操作系统的 64 位版本具有相同名称的 32 位版本中，即使它们是独立的文件。 ODBC 测试的 64 位版本的 Unicode 版本的名称即 odbct32w.exe。
