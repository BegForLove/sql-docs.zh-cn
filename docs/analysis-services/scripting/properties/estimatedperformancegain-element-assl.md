# <a name="estimatedperformancegain-element-assl"></a>EstimatedPerformanceGain 元素 (ASSL)

## <a name="syntax"></a>语法  
  
```xml  
  
<AggregationDesign>  
      ...  
   <EstimatedPerformanceGain>...</EstimatedPerformanceGain>  
   ...  
</AggregationDesign>  
```
