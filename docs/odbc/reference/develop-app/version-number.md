# <a name="version-number"></a>版本号
有几个版本的 ODBC，每个都有不同的功能。 应用程序确定 ODBC 版本驱动程序管理器和一个特定的驱动程序支持通过调用**SQLGetInfo** SQL_ODBC_VER 和 SQL_DRIVER_ODBC_VER 选项。
