# <a name="sqlgetstmtoption-function"></a>SQLGetStmtOption 函数
**一致性**  
 版本引入了： ODBC 1.0 标准合规性： 不推荐使用  
  
 **摘要**  
 在 ODBC 3。*x*，ODBC 2.0 函数**SQLGetStmtOption**已被取代**SQLGetStmtAttr**。 有关详细信息，请参阅[SQLGetStmtAttr](../../../odbc/reference/syntax/sqlgetstmtattr-function.md)。  
  
> [!NOTE]  
>  有关详细信息有关哪些驱动程序管理器时，将映射此函数可对 ODBC 2。*x*应用程序使用 ODBC 3。*x*驱动程序，请参阅[映射弃用函数](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)为了向后兼容的附录 g： 驱动程序准则中。
