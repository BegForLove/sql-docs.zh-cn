# <a name="capture-ado-error-codes"></a>捕获 ADO 错误代码
除了提供程序错误中返回[错误](../../../ado/reference/ado-api/error-object.md)的对象[错误](../../../ado/reference/ado-api/errors-collection-ado.md)集合，ADO 本身可以返回错误，运行时环境的异常处理机制。 使用错误捕获机制您的编程语言，如**On Error**语句在 Microsoft® Visual Basic 中，或**try catch**中 Microsoft Visual C++®，以捕获 ADO 错误块。

 ADO 错误代码的列表，请参阅[ErrorValueEnum](../../../ado/reference/ado-api/errorvalueenum.md)。
