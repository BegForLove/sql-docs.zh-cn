# <a name="files-to-use-with-the-desktop-database-drivers"></a>要与桌面数据库驱动程序一起使用的文件
ODBC 桌面数据库驱动程序与 Microsoft Jet 数据库引擎、可安装的 ISAM 文件和 Microsoft ODBC 文件一起使用。 并非所有产品都需要重新分发，而且并非所有都是核心组件。  
  
|有关以下方面的信息|查看|  
|---------------------------|---------|  
|每个桌面数据库驱动程序使用的文件|ODBC 驱动程序文件|  
|每个桌面数据库驱动程序使用的微软 Jet 文件|微软喷气文件|  
|提供 Microsoft Jet 和其他 DBMS 之间的数据访问的 IISAM 驱动程序文件|可安装 ISAM 驱动程序文件|
