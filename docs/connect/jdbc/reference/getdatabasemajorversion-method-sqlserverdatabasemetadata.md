# <a name="getdatabasemajorversion-method-sqlserverdatabasemetadata"></a>getDatabaseMajorVersion 方法 (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索基础数据库的主版本号。  
  
## <a name="syntax"></a>语法  
  
```  
  
public int getDatabaseMajorVersion()  
```  
  
## <a name="return-value"></a>返回值  
 指示数据库主版本的 int。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 此 getDatabaseMajorVersion 方法由 java.sql.DatabaseMetaData 接口中的 getDatabaseMajorVersion 方法指定。
