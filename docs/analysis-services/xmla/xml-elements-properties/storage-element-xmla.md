# <a name="storage-element-xmla"></a>Storage 元素 (XMLA)

## <a name="syntax"></a>语法  
  
```xml  
  
<DesignAggregations>  
   ...  
   <Storage>...</Storage>  
   ...  
</DesignAggregations>  
```
