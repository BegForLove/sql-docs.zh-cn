# <a name="detach-database-id-32767"></a>分离数据库 ID 32767
  升级顾问检测到具有数据库 ID 32767 的数据库。 在 [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] 和更高版本中，此数据库 ID 是保留值。 请先分离该数据库，然后再升级。 您可以在升级后再附加该数据库。
