# <a name="getresultsettype-method-sqlserverstatement"></a>getResultSetType 方法 (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  检索结果集类型[SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md)此生成的对象[SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md)对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
public final int getResultSetType()  
```  
  
## <a name="return-value"></a>返回值  
 **Int** ，该值指示结果集类型。  
  
## <a name="exceptions"></a>异常  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>注释  
 由 java.sql.Statement 接口中的 getResultSetType 方法指定此 getResultSetType 方法。
