# MachineAccountIdentity 属性 (WMI)
  获取安装了报表服务器的计算机的计算机帐户标识。  
  
## 语法  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## 属性值  
 一个 **String** 值，其中包含安装了报表服务器的计算机的计算机帐户标识。
