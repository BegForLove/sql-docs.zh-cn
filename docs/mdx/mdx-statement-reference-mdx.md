|主题|Description|  
|-----------|-----------------|  
|[MDX 脚本编写语句 (MDX)](../mdx/mdx-scripting-statements-mdx.md)|包含有关 MDX 脚本编写语句的信息，这种语句用于管理 MDX 脚本中的查询上下文、作用域和流控制。|
