# <a name="machine-data-sources"></a>计算机数据源
*机器数据源*存储在具有用户定义名称的系统上。 与数据源名称关联的是驱动程序管理器和驱动程序连接到数据源所需的所有信息。 对于 Xbase 数据源，这可能是 Xbase 驱动程序的名称、包含 Xbase 文件的目录的完整路径以及告诉驱动程序如何使用这些文件的一些选项，如单用户模式或只读。 对于 Oracle 数据源，这可能是 Oracle 驱动程序的名称、Oracle DBMS 所在的服务器、标识要使用的 SQL Net 驱动程序的 SQL_Net\*连接字符串以及服务器上数据库的系统 ID。
