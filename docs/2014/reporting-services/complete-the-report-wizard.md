# <a name="complete-the-report-wizard"></a>完成报表向导
  使用此页可以对报表进行命名并检查要应用于报表的选项。  
  
## <a name="options"></a>选项  
 **报表名称**  
 键入此报表的名称。 此名称还将显示为报表的标题。 可以在设计视图中更改标题。  
  
 **报表摘要**  
 查看在报表向导的前面步骤中选择的选项。 若要更改某个选项，请单击 **“上一步”** ，直到找到包含要更改的选项的页为止。  
  
 **预览报表**  
 选择此选项可预览所创建的报表。 否则，它将在设计视图中打开。
