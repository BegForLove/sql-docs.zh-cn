**度**  
 引入的版本： ODBC 1.0 标准符合性：已弃用  
  
 **总结**  
 Odbc 2.0 函数**SQLParamOptions**已在 odbc 3 中被替换。*x*调用[SQLSetStmtAttr](../../../odbc/reference/syntax/sqlsetstmtattr-function.md)。
