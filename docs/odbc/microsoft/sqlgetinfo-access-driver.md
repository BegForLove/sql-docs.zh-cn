# <a name="sqlgetinfo-access-driver"></a>SQLGetInfo（Access 驱动程序）
> [!NOTE]  
>  本主题提供特定于访问驱动程序的信息。 有关此功能的一般信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)下的相应主题。  
  
 **SQLGetInfo**支持SQL_FILE_USAGE信息类型。 返回的值是一个 16 位整数，指示驱动程序如何处理数据源中的文件：  
  
-   SQL_FILE_NOT_SUPPORTED - 驱动程序不是单层驱动程序。  
  
-   SQL_FILE_TABLE - 单层驱动程序将数据源中的文件视为表。  
  
-   SQL_FILE_QUALIFIER - 单层驱动程序将数据源中的文件视为限定符。  
  
 ODBC 驱动程序返回SQL_FILE_QUALIFIER因为每个文件都是一个完整的数据库。
