# MSSQL_REPL 2147199371
    
## 消息详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|-2147199371|  
|事件源|MSSQLServer|  
|符号名称||  
|消息正文|发送到 IIS 服务器的请求大于 4 GB，但不支持此大小。 请为“UploadGenerationsPerBatch”参数使用较小的值。|  
  
## 解释  
 当您使用 Web 同步时，所上载的消息的大小不得大于 4 GB。  
  
## 用户操作  
 减小用于 UploadGenerationsPerBatch 参数的值。
