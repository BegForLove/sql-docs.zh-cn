# <a name="enable-dimension-writeback-business-intelligence-wizard"></a>启用维度写回（商业智能向导）
  可以使用 **“启用维度写回”** 页启用或禁用所选维度的写回功能。  
  
## <a name="options"></a>选项  
 **在维度中启用写回**  
 选择此选项可以启用维度的写回功能。
