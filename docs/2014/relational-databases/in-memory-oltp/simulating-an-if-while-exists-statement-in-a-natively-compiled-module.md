# <a name="simulating-an-exists-clause-in-a-natively-compiled-stored-procedure"></a>在本机编译的存储过程中模拟 EXISTS 子句
  本机编译的存储过程不支持 `EXISTS` 子句，但是可以解决这个问题：  
  
```sql  
DECLARE @exists BIT = 0  
SELECT TOP 1 @exists = 1 FROM MyTable WHERE ...  
IF @exists = 1  
```
