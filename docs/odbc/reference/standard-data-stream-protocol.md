# <a name="standard-data-stream-protocol"></a>标准数据流协议
标准数据流协议是一种方法访问异类 Dbms 中的数据。 事实上，标准数据流协议已经存在： ANSI/ISO 远程数据库访问 (RDA) 标准 ([ISO/IEC 9579:2000](http://www.iso.org/iso/catalogue_detail.htm?csnumber=30615)) 和 IBM 分布式关系数据库体系结构 (DRDA) ([ http://www.netstoreusa.com/cbbooks/007/0070351384.shtml](http://www.netstoreusa.com/cbbooks/007/0070351384.shtml)). 虽然这两个系统显示承诺，但既不是广泛目前的实现。
