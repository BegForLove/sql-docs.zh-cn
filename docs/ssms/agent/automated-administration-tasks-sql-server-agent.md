# <a name="automated-administration-tasks-sql-server-agent"></a>自动执行管理任务（SQL Server 代理）
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Microsoft [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 允许用户自动执行管理任务。 若要自动进行管理，您需要先定义可预测的管理任务，然后再指定每个任务发生的条件。 使用自动管理来处理日常事务和事件，使您有时间执行其他管理功能。
