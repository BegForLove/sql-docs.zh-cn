# <a name="sqlstate"></a>SQLSTATE
对于在数据源中发生的错误，ODBC 驱动程序将返回的本机错误映射到相应的 SQLSTATE。 有错误的检测到由驱动程序或驱动程序管理器，ODBC 驱动程序或驱动程序管理器生成相应的 SQLSTATE。
