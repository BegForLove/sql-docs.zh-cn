# <a name="diagnostic-tools"></a>诊断工具
两个设施可以帮助诊断 ODBC 应用中的问题。 两者都可从 ODBC 管理员处获得，并由驱动程序管理器实施。 *跟踪*提供了一种将函数调用序列记录到日志文件的方法。 可视化工作室分析器允许分析有关分布式环境中组件交互的信息。  
  
 本部分包含以下主题。  
  
-   [跟踪](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
