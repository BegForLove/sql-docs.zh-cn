# <a name="deleting-a-visual-foxpro-data-source"></a>删除 Visual FoxPro 数据源
### <a name="to-delete-a-visual-foxpro-data-source"></a>删除可视化 FoxPro 数据源  
  
1.  在运行 Windows 2000 的计算机上，打开 Windows 控制面板并双击管理工具。  
  
2.  双击数据源 （ODBC） 以打开 ODBC 数据源管理员对话框。 此图标在安装 Visual FoxPro ODBC 驱动程序或任何 ODBC 驱动程序软件后可用。  
  
    > [!NOTE]  
    >  如果您正在运行早期版本的 Windows，请打开 Windows 控制面板并双击 32 位 ODBC 或 ODBC 以打开 ODBC 数据源管理员对话框。  
  
3.  在"用户 DSN"或"系统 DSN"选项卡中，选择要删除的数据源的名称。  
  
4.  单击“删除”。  
  
5.  单击"确定"以保存更改并关闭 ODBC 数据源管理员对话框。
