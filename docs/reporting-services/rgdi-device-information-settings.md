# <a name="rgdi-device-information-settings"></a>RGDI 设备信息设置
  下表列出了以 RGDI 格式呈现时的设备信息设置。  
  
|设置|ReplTest1|  
|-------------|-----------|  
|**第**|指定要呈现的报表页。 默认值是 **1**秒。|  
|**RGDIVersion**|此设置不起作用。|
