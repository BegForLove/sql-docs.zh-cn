# <a name="supported-versions-of-foxpro"></a>支持的 FoxPro 版本
您可以使用 Microsoft ®可视化 FoxPro® ODBC 驱动程序访问存储在 FoxPro 表中的数据。 支持以下版本的 FoxPro 数据：  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   可视化 FoxPro（所有版本）  
  
 当您访问存储在 Visual FoxPro 中的数据时，您可以选择连接到包含零个或多个[表](../../odbc/microsoft/visual-foxpro-terminology.md)的[数据库](../../odbc/microsoft/visual-foxpro-terminology.md)或[自由表](../../odbc/microsoft/visual-foxpro-terminology.md)的目录。  
  
 有关如何连接到数据源的详细信息，请参阅[添加 Visual FoxPro 数据源](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md)。  
  
 要下载驱动程序，请参阅[可视化福克斯Pro ODBC驱动程序下载网站](https://go.microsoft.com/fwlink/?LinkId=121318)。
