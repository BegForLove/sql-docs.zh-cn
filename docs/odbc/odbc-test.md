# <a name="odbc-test"></a>ODBC 测试
Microsoft® ODBC 测试是可用于测试 ODBC 驱动程序和 ODBC 驱动程序管理器的支持 ODBC 的应用程序。 ODBC 3.51 包含 ANSI 和 Unicode 支持的版本的 ODBC 测试。 相应的文件如下所示：  
  
-   Odbcte32.exe 和 Gtrtst32.dll，对于 ANSI 版本。  
  
-   Odbct32w.exe 和 Gtrts32w.dll，Unicode 版本。  
  
 若要使用 ODBC 测试，必须了解 ODBC API、 C 语言和 SQL。 有关 ODBC API 的详细信息，请参阅[ODBC 程序员参考](../odbc/reference/odbc-programmer-s-reference.md)。  
  
 ODBC 测试程序中现在包含了以前包含在文档的此部分的帮助主题。 打开 Odbcte32.exe 或 Odbct32w.exe，打开**帮助**菜单，并单击**帮助主题**。  
  
 请注意，这些应用程序，适用于 64 位 Microsoft Windows 操作系统的 64 位版本相同的名称的 32 位版本，即使它们是单独的文件。 ODBC 测试的 64 位版本的 Unicode 版本的名称即 odbct32w.exe。
