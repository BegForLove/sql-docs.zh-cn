# <a name="refresh-from-database-accesstosql"></a>从数据库 (AccessToSQL) 刷新
**从数据库刷新**对话框允许您选择要从 Access 数据库刷新的对象。 在对话框中的行进行了颜色编码基于元数据的状态：  
  
-   如果本地和 Access 数据库中已更改的对象元数据，该行是蓝色的。  
  
-   如果在访问数据库而不是在 SSMA 中已更改的对象元数据，该行是黄色。  
  
-   如果对象元数据已更改本地，但不是在访问数据库中，行是绿色。  
  
-   如果对象是新的 Access 数据库中，该行是粉红色。  
  
可以指定默认值对象中的刷新设置**项目设置**对话框。 有关详细信息，请参阅[项目设置&#40;加载对象&#41; &#40;AccessToSQL&#41;](../../ssma/access/project-settings-loading-objects-accesstosql.md)  
  
访问**从数据库刷新**对话框框中，右键单击任何**数据库**节点中访问元数据资源管理器并单击**从数据库刷新**。
