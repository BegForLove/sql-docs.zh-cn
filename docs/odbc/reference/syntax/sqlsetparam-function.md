# <a name="sqlsetparam-function"></a>SQLSetParam 函数
**一致性**  
 版本介绍： ODBC 1.0 标准合规性： 已弃用  
  
 **摘要**  
 在 ODBC 2.0 中，ODBC 1.0 函数**SQLSetParam**已被**SQLBind参数**替换。 有关详细信息，请参阅[SQLBind参数函数](../../../odbc/reference/syntax/sqlbindparameter-function.md)。  
  
> [!NOTE]  
>  有关驱动程序管理器将此功能映射到 ODBC 2 时的详细信息。*x*应用程序使用 ODBC 3。*x*驱动程序，请参阅[映射已弃用函数](../../../odbc/reference/appendixes/mapping-deprecated-functions.md)。  
  
## <a name="remarks"></a>备注  
 如果应用程序将在 64 位操作系统上运行，请参阅[ODBC 64 位信息](../../../odbc/reference/odbc-64-bit-information.md)。
