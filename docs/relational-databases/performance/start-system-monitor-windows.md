# 启动系统监视器 (Windows)
  使用系统监视器可以监视系统资源的使用率。 使用计数器形式收集和查看服务器资源（如处理器和内存使用）和许多 Microsoft [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 资源（如锁和事务）的实时性能数据。  
  
### 在 Windows 中启动系统监视器  
  
1.  在 **“开始”** 菜单上，指向 **“运行”**，在“运行”对话框中键入 **perfmon** ，然后单击 **“确定”**。
