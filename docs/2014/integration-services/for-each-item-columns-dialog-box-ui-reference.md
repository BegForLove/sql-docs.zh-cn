# <a name="for-each-item-columns-dialog-box-ui-reference"></a>“For Each Item 列”对话框 UI 参考
  可以使用 **“For Each Item 列”** 对话框定义 Foreach Item 枚举器枚举的项中的列。  
  
## <a name="options"></a>选项  
 **列**  
 列出各列。  
  
 **数据类型**  
 选择数据类型。  
  
 **“添加”**  
 添加新列。  
  
 **删除**  
 选择某列，再单击“删除”。
