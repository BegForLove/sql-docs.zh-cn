# <a name="sqlmoreresults-visual-foxpro-odbc-driver"></a>SQLMoreResults（Visual FoxPro ODBC 驱动程序）
> [!NOTE]  
>  本主题包含特定于 Visual FoxPro ODBC 驱动程序的信息。 有关此函数的常规信息，请参阅[ODBC API 参考](../../odbc/reference/syntax/odbc-api-reference.md)中的相应主题。  
  
 支持：完全  
  
 ODBC API 一致性：级别2  
  
 确定对于包含 SELECT、UPDATE、INSERT 或 DELETE 语句的语句句柄（ *hstmt*）是否挂起更多结果，如果是，则为这些结果初始化处理。  
  
 有关详细信息，请参阅*ODBC 程序员参考*中的[SQLMoreResults](../../odbc/reference/syntax/sqlmoreresults-function.md) 。
