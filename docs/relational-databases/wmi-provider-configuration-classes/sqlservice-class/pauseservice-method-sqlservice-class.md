# <a name="pauseservice-method-sqlservice-class"></a>PauseService 方法（SqlService 类）
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]尝试将服务置于暂停状态。  
  
## <a name="syntax"></a>语法  
  
```  
  
object.PauseService()  
```  
  
## <a name="parts"></a>组成部分  
 *对象*  
 一个表示服务的 [SqlService 类](../../../relational-databases/wmi-provider-configuration-classes/sqlservice-class/sqlservice-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个 uint32 值，如果服务已成功停止，则为 0；如果不支持请求，则为 1；其他任何数字表示出现错误。
