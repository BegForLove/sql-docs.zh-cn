# <a name="setbytes-method-sqlserverblob"></a>setBytes 方法 (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  将给定的字节数组写入 BLOB（从给定的位置开始），然后返回写入的字节数。  
  
## <a name="overload-list"></a>重载列表  
  
|Name|Description|  
|----------|-----------------|  
|[setBytes （长度，字节 &#91; &#93;）](../../../connect/jdbc/reference/setbytes-method-long-byte.md)|将给定的字节数组写入 BLOB（从给定的位置开始），然后返回写入的字节数。|  
|[setBytes (长，字节 &#91; &#93;，int，int)](../../../connect/jdbc/reference/setbytes-method-long-byte-int-int.md)|从给定的位置开始根据偏移量和长度，将给定的字节数组的全部或一部分写入 BLOB，然后返回写入的字节数。|
