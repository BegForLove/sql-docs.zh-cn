# <a name="report-parameters-methods"></a>报表参数方法
  可以使用这些方法来设置和检索报表的参数。  
  
|方法|操作|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.GetItemParameters%2A>|返回指定项的参数属性。 此方法还可用于验证指定项的参数值。|  
|<xref:ReportService2010.ReportingService2010.ListParameterStates%2A>|返回支持的参数状态的列表。|  
|<xref:ReportService2010.ReportingService2010.ListParameterTypes%2A>|返回支持的参数类型的列表。|  
|<xref:ReportService2010.ReportingService2010.SetItemParameters%2A>|设置指定项的参数属性。|
