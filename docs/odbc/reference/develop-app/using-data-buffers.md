# <a name="using-data-buffers"></a>使用数据缓冲区
数据缓冲区描述了三个信息片段： 它们的类型、 地址和字节长度。 当函数需要一个这些信息片段，并不已知道它时，它具有与应用程序将其传递的参数。  
  
 本部分包含以下主题。  
  
-   [数据缓冲区类型](../../../odbc/reference/develop-app/data-buffer-type.md)  
  
-   [数据缓冲区地址](../../../odbc/reference/develop-app/data-buffer-address.md)  
  
-   [数据缓冲区长度](../../../odbc/reference/develop-app/data-buffer-length.md)
