# <a name="sqlserverpooledconnection-class"></a>SQLServerPooledConnection 类
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  表示连接池中的物理数据库连接。  
  
 **包：** com.microsoft.sqlserver.jdbc  
  
 **扩展：** java.lang.Object  
  
 **实现：** javax.sql.PooledConnection  
  
## <a name="syntax"></a>语法  
  
```  
  
public final class SQLServerPoolingDataSource  
```  
  
## <a name="remarks"></a>注释  
 SQLServerPooledConnection 类提供连接池管理器来管理连接池的方法。 应用程序一般不直接实例化这些连接。
