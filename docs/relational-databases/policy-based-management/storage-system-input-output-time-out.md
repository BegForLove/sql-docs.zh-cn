# <a name="storage-system-input-output-time-out"></a>存储系统输入输出超时
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  此规则检查系统事件日志中是否存在 EventId 9。 此消息指示存储系统中出现 I/O 超时。
