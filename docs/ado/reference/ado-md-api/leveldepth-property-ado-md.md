# <a name="leveldepth-property-ado-md"></a>LevelDepth 属性 (ADO MD)
指示层次结构的根和[成员](../../../ado/reference/ado-md-api/member-object-ado-md.md)之间的级别数。  
  
## <a name="return-values"></a>返回值  
 返回一个**长**整型，并且是只读的。  
  
## <a name="remarks"></a>备注  
 使用**LevelDepth**属性来确定[成员](../../../ado/reference/ado-md-api/member-object-ado-md.md)对象与层次结构的根级别的距离。 根级别的成员的**LevelDepth**是0。 这对应于[Level](../../../ado/reference/ado-md-api/level-object-ado-md.md)对象的[Depth](../../../ado/reference/ado-md-api/depth-property-ado-md.md)属性。  
  
## <a name="applies-to"></a>应用于  
 [成员对象 (ADO MD)](../../../ado/reference/ado-md-api/member-object-ado-md.md)
