# <a name="pseudo-type-identifiers"></a>伪类型标识符
为应用程序编程方便起见，ODBC 定义了多个伪类型标识符。 这些标识符不实际对应于实际数据类型，但相反，具体取决于这种情况，将解析为现有数据类型。  
  
 本部分包含以下主题。  
  
-   [默认 C 数据类型](../../../odbc/reference/appendixes/default-c-data-types.md)  
  
-   [书签 C 数据类型](../../../odbc/reference/appendixes/bookmark-c-data-type.md)  
  
-   [SQL_ARD_TYPE](../../../odbc/reference/appendixes/sql-ard-type.md)  
  
-   [SQL_C_TCHAR](../../../odbc/reference/appendixes/sql-c-tchar.md)
