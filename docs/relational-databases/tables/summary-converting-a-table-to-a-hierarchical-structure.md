# 摘要：将表转换为层次结构
**hierarchyid** 数据类型可以采用精简格式存储层次结构关系。 正确的索引可以通过层次结构级别、层次结构子树或传统单独查找方式有效地返回数据。  
  
在本课中，您使用了两种分层方法。 下一课将说明如何使用更多此类方法查询和操作分层数据。
