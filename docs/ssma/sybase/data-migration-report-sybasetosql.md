# <a name="data-migration-report-sybasetosql"></a>数据迁移报告 (SybaseToSQL)
**数据迁移报告**对话框中将显示在您迁移到的数据后[!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]。  
  
## <a name="options"></a>选项  
**状态**  
显示从迁移数据源到目标数据库的状态。  
  
**从**  
源表。  
  
**若要**  
目标表中。  
  
**总行数**  
源表中的数据的行数。  
  
**已成功迁移的行数**  
数据的行数已成功迁移到目标表。  
  
**比率**  
已成功迁移的行的百分比。  
  
**详细信息**  
如果任何数据迁移失败，请单击要在报表中显示所选行的迁移详细信息。 SSMA 将显示失败的原因。  
  
**保存报表**  
保存到报表。CSV （逗号分隔值） 文件，可以使用 Microsoft Excel 进行检查。
