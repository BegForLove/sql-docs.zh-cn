# <a name="task-2-adding-domain-values-manually"></a>任务 2：手动添加域值
  在此任务中，你添加的值**国家/地区**域手动。 请参阅[Change Domain Values](http://msdn.microsoft.com/library/hh510408.aspx)有关此页上的字段的更多详细信息的主题。  
  
1.  单击**国家/地区**中的域**域列表**。  
  
2.  在右窗格中，切换到**域值**选项卡。  
  
3.  单击**添加新域值**在右窗格中的工具栏上的按钮。  
  
     ![添加新域值的工具栏按钮](../../2014/tutorials/media/et-addingdomainvaluesmanually-01.jpg "添加新域值的工具栏按钮")  
  
4.  类型**美国**为**值**字段，然后按**ENTER**。 你可以看到，默认情况下，**类型**设置为**更正**（绿色复选标记）。 类型可以设置为**错误**（红叉） 或**无效**（橙色三角形），并将其正确的值可以输入在**更正为**字段。  
  
     ![域值的美国境内](../../2014/tutorials/media/et-addingdomainvaluesmanually-02.jpg "域值的美国")
