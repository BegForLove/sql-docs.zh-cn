## <a name="details"></a>详细信息  
  
|||  
|-|-|  
|产品名称|SQL Server|  
|事件 ID|266|  
|事件源|SQL Server 本地数据库运行时 12.0|  
|组件|本地数据库运行时 API|  
|消息正文|在本地数据库实例启动期间出错: 无法启动 SQL Server 进程。|  
  
## <a name="explanation"></a>解释
