# <a name="data-type-change-required-dialog-box-visual-database-tools"></a>“需要更改数据类型”对话框 (Visual Database Tools)
  当您更改参与关系的列的数据类型、长度、小数位数、精度或排序规则时，将显示此对话框。  
  
## <a name="options"></a>“常规”  
 **是**  
 更改列表中显示的相关列的数据类型属性，以保留现有的关系。  
  
 **“否”**  
 取消对数据类型的更改，并将刚更改的列还原为以前的数据类型。
